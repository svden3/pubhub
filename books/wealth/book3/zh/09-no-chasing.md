# 第九章：不要追涨杀跌

> "Sarah 说：你总是在高点买，在低点卖。这叫追涨杀跌。"

---

## 故事

Mike 的投资历史：

| 年份 | 操作 | 结果 |
|------|------|------|
| 2000 | 高点买网络股 | 亏了 |
| 2007 | 高点买曼哈顿房子 | 亏了 40% |
| 2021 | 儿子高点买 Zoom | 学费没了 |

Mike 的模式：

1. 看到什么涨，就去买
2. 看到什么跌，就卖掉
3. 重复

**这叫追涨杀跌。这是亏钱的最可靠方法。**

---

## 为什么追涨杀跌会亏钱

人类的本能：

- 看到涨 → 觉得会继续涨 → 买
- 看到跌 → 觉得会继续跌 → 卖

但市场的规律是：

- 涨太多 → 会跌
- 跌太多 → 会涨

**你的本能和市场规律是反的。**

---

## 规则

### 规则 1：看到热门的，不买

当你听到所有人都在讨论一只股票：

- 2000年：Pets.com
- 2020年：Zoom
- 2021年：GameStop
- 2021年：Dogecoin
- 2024年：Nvidia（要小心）

**等到你听说的时候，通常已经晚了。**

### 规则 2：看到暴跌的，不卖

当新闻头条全是恐慌：

- 2008年：金融危机
- 2020年：新冠暴跌
- 2022年：科技股暴跌

**这通常是机会，不是逃跑的时候。**

### 规则 3：用系统代替情绪

不要根据感觉做决定。用规则：

```
买入规则：
- 每月定投 $X
- 不管涨跌
- 自动执行

卖出规则：
- 只在需要用钱时卖
- 只在投资逻辑变化时卖
- 不因为涨跌而卖
```

## 自我检测：你是不是在追涨杀跌？

问自己这些问题：

| 问题 | 如果是，你可能在追涨杀跌 |
|------|-------------------------|
| 你是在看到涨之后才买的吗？ | ✓ |
| 你是在看到跌之后想卖的吗？ | ✓ |
| 你是因为朋友在买才买的吗？ | ✓ |
| 你是因为新闻说要崩盘才卖的吗？ | ✓ |
| 你是在涨了之后后悔没早买吗？ | ✓ |
| 你是在跌了之后后悔没早卖吗？ | ✓ |

**如果有 2 个以上是"是"，你需要改变策略。**

---

## 怎么改

### 方法 1：定投

每月固定金额，不管涨跌。

**把决策权从情绪手中拿走。**

### 方法 2：不看新闻

财经新闻的目的是吸引眼球，不是帮你赚钱。

**删掉财经 App。**

### 方法 3：写投资日记

每次买卖之前，写下：

- 我为什么买/卖？
- 我的情绪是什么？
- 这是系统决策还是冲动？

**强迫自己思考。**
