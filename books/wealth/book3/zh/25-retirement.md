## 退休的三个数字

### 数字一：你需要多少钱？

**简单公式**：
```
退休储蓄 = 年支出 × 25
```

这叫 **4% 法则**——每年从退休储蓄中提取 4%，理论上可以用 30 年以上。

**举例**：
- 年支出 $60,000 → 需要 $1,500,000
- 年支出 $80,000 → 需要 $2,000,000
- 年支出 $100,000 → 需要 $2,500,000

妈妈说：
> "儿子，这只是理论。实际上，你不会花一样多的钱。60 岁和 80 岁，开支完全不同。"

---

### 数字二：你还有多少时间？

**复利的魔法**：
```
25 岁开始 vs 35 岁开始

假设每月存 $500，年回报 7%：

25 岁开始 → 65 岁有 $1,200,000
35 岁开始 → 65 岁有 $567,000
```

**10 年的差距 = 63 万美元**

Sarah 说：
> "复利是时间的函数。开始得越早，终点越远。"

---

### 数字三：你的收入来源有几个？

**理想的退休收入来源**：
```
1. 社保（Social Security）  —— 基础保障
2. 退休账户（401k/IRA）    —— 税务优惠
3. 投资收入（股息/利息）    —— 被动收入
4. 兼职/顾问                —— 如果你想
```

**Mike 的教训**：
> "我所有的钱都在 Enron 股票里。退休金、401k、个人账户。
> 我以为我是百万富翁。
> 三个月后，我一无所有。"

**规则**：退休收入至少要有 2-3 个独立来源。

## 退休账户：硅谷人的税务武器

### 401(k)：公司帮你存

```
2024 年限额：$23,000（50 岁以上 +$7,500）

优点：
✓ 税前存入，当年省税
✓ 公司 Match（免费的钱！）
✓ 自动扣款，无需纪律

缺点：
✗ 59.5 岁前取出要罚款
✗ 取出时按收入纳税
✗ 投资选择有限
```

**妈妈的命令**：
> "儿子，401k 至少存到公司 Match 的上限。不存就是白送钱给政府。"

---

### Roth IRA：年轻人的最佳选择

```
2024 年限额：$7,000（50 岁以上 +$1,000）

优点：
✓ 税后存入，增长免税
✓ 取出时完全免税
✓ 退休后灵活度高
✓ 没有强制取出要求

缺点：
✗ 收入太高不能存（$161k 单身 / $240k 夫妻）
✗ 限额较低
```

**Sarah 的建议**：
> "年轻时收入低，税率低。用 Roth IRA。
> 老了收入高，税率高。用 Traditional 401k。
> 这叫税务套利。"

---

### Backdoor Roth：高收入者的后门

如果收入超过 Roth IRA 限额：

```
步骤：
1. 存入 Traditional IRA（非抵税）
2. 立即转换成 Roth IRA
3. 缴少量转换税
4. 享受终身免税增长
```

**注意**：要咨询税务专家，有 Pro Rata 规则。

## 退休的四个阶段

### 第一阶段：积累期（25-45 岁）

**目标**：存得越多越好

```
策略：
- 最大化 401k + Roth IRA
- 投资主要放在股票
- 承受波动，追求增长
- 不要碰退休账户
```

---

### 第二阶段：过渡期（45-55 岁）

**目标**：降低风险，保护本金

```
策略：
- 逐步从股票转向债券
- 开始规划退休后的生活
- 计算你的"数字"
- 考虑提前退休的可能
```

Sarah 在 45 岁时：
> "我开始认真算了。股票从 80% 降到 60%。债券从 10% 升到 30%。现金保持 10%。"

---

### 第三阶段：退休初期（55-70 岁）

**目标**：活跃、自由、有选择

```
策略：
- 健康是最大的投资
- 可以做自己想做的事
- 支出可能比工作时更高（旅行、爱好）
- 保持一定的股票配置（对抗通胀）
```

---

### 第四阶段：退休后期（70 岁以后）

**目标**：简化、安心、传承

```
策略：
- 支出通常下降
- 医疗费用上升
- 考虑遗产规划
- 简化投资组合
```

## 加州房产：退休的隐形炸弹

2022 年，Mike 的邻居 David 要退休了。

"我 1995 年买的房子，$300,000。现在值 $2,500,000。"

"恭喜啊！"

"**恭喜什么？我想卖掉搬去德州，但联邦资本利得税要交 $440,000。**"

这就是加州退休的第一课：**你的房子可能是金手铐**。

---

### Prop 13：加州最伟大的法案

**1978 年的规则**：
```
Prop 13 核心：
- 房产税基于购买价，不是市场价
- 每年最多涨 2%
- 房子不卖，税基不变
```

**David 的例子**：
```
1995 年买入：$300,000
2024 年市值：$2,500,000

按 Prop 13：每年房产税 ≈ $5,000（基于调整后的购买价 ~$420,000）
按市价算：每年房产税 ≈ $30,000

每年省：$25,000
```

妈妈说：
> "儿子，在加州买房，就像买了终身打折卡。但你一卖，折扣就没了。"

---

### 1031 Exchange：延税的艺术

**联邦税法第 1031 条**：
```
规则：
- 卖投资房产，买同类房产
- 资本利得税可以延迟
- 不是免税，是递延

时间要求：
- 45 天内确定新房产
- 180 天内完成交易

限制：
- 只适用于投资房产
- 不适用于自住房（有其他免税额）
- 必须"同类交换"（Like-Kind）
```

**David 的想法**：
> "我能不能用 1031 把 Palo Alto 的房子换成德州的房子？"

**答案**：可以。但你会失去 Prop 13 的保护。而且德州虽然没有州税，但房产税是加州的 3 倍。

---

### Prop 19 陷阱：2020 年的糖衣毒药

**表面的"好处"**：
```
Prop 19（2020 年生效）：

"好处"：
- 55 岁以上可以带税基搬家（全州范围）
- 残疾人、灾民也可以
- 最多可用 3 次

听起来不错？等等...
```

**真正的陷阱**：
```
Prop 19 的代价：

旧规则（Prop 58）：
- 父母房产传给子女，税基不变
- 子女继承 $300,000 的税基
- 继续享受低房产税

新规则（Prop 19）：
- 继承后如果不自住，重新评估市值
- 子女继承 $2,500,000 市值的房子
- 房产税从 $5,000 跳到 $30,000
- 除非子女搬进去住
```

**Mike 的分析**：
> "Prop 19 就是政府的把戏。
> 表面上给老人搬家的便利。
> 实际上取消了子女继承的税基保护。
> 加州政府每年多收几十亿房产税。"

□ 计算卖房 vs 持有的税务成本

□ 想清楚退休后要做什么
□ 买对不难，拿住才难
□ 时间比钱贵
