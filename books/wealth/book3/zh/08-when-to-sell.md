# 第八章：什么时候该走

> "Sarah 说：后悔是投资者最大的敌人。做了决定就不要回头看。"

---

## 故事

2004 年，Sarah 离开我们的小公司，去了 Google。

她给了我一个机会："Jim，你要不要一起来？"

我没去。

Google 上市后，Sarah 的期权值了几百万。

那天晚上，我给 Sarah 打电话。

"我后悔了。"

"**后悔是投资者最大的敌人。做了决定就不要回头看。**"

"这是谁说的？"

"我爸。他炒股亏了很多钱之后说的。"

---

## 规则

### 规则 1：该卖的三种情况

| 情况 | 例子 |
|------|------|
| **需要用钱** | 买房、退休、紧急情况 |
| **投资逻辑变了** | 公司基本面恶化、行业消失 |
| **有更好的选择** | 机会成本更高 |

### 规则 2：不该卖的情况

| 情况 | 为什么不该卖 |
|------|-------------|
| 股价跌了 | 可能只是情绪波动 |
| 新闻说要崩盘 | 新闻通常滞后 |
| 涨了很多想落袋为安 | 你会错过更多上涨 |
| 朋友都在卖 | 群体不一定对 |

### 规则 3：做了决定不要回头看

卖了之后：
- 不要看它后来涨了多少
- 不要计算"如果我没卖会怎样"
- 不要后悔

**钱已经不是你的了。向前看。**

---

## 卖出检查清单

在卖出之前，问自己：

1. **我为什么买入？** 那个理由还成立吗？

2. **这是情绪还是理性？** 是因为恐惧/贪婪，还是真的有逻辑？

3. **卖出后钱去哪里？** 有更好的地方放吗？

4. **税的影响？** 短期持有税更高（美国：短期 = 收入税，长期 = 资本利得税 15-20%）

5. **我会后悔吗？** 如果它之后涨了 50%，我能接受吗？

---

## 指数基金卖出时机

如果你买的是指数基金，卖出时机很简单：

```
年轻时：不卖
中年时：不卖
退休时：每年卖 4%（4% rule）
```

**就这么简单。**

---

## 一句话

> **卖出的理由只有三个：需要用钱、逻辑变了、有更好选择。其他都是借口。**

---

## 检查清单

- [ ] 我有明确的卖出理由（三个之一）
- [ ] 我不是因为情绪在卖
- [ ] 我考虑了税的影响
- [ ] 我做好了不后悔的准备
