# 附录 A：投资决策检查清单

> 每次投资前，过一遍这个清单。

---

## 投资前检查

### 1. 基础条件 ✓

- [ ] 我有 6 个月以上的紧急备用金
- [ ] 我没有高息债务（信用卡等）
- [ ] 这是"闲钱"，5 年内不需要用
- [ ] 我设置了自动定投

### 2. 理解检查 ✓

- [ ] 我能用三句话解释这个投资怎么赚钱
- [ ] 我知道如果错了会亏多少
- [ ] 我不是因为"别人都在买"而买
- [ ] 我没有被"复杂"的术语吓住

### 3. 情绪检查 ✓

- [ ] 我不是在恐慌中卖出
- [ ] 我不是在狂热中买入
- [ ] 这是计划内的决策，不是冲动
- [ ] 我没有 FOMO（错过恐惧）

### 4. 分散检查 ✓

- [ ] 这笔投资不超过总资产的 10%
- [ ] 我没有买自己公司的股票（或很少）
- [ ] 我的投资组合是分散的
- [ ] 我有股票、债券、现金的配置

## 卖出检查

在点击"卖出"之前：

| 问题 | 如果是"是"，可能不该卖 |
|------|------------------------|
| 我是因为价格跌了而卖？ | ⚠️ |
| 我是因为新闻吓到了而卖？ | ⚠️ |
| 我是因为想落袋为安而卖？ | ⚠️ |
| 我是因为朋友都在卖而卖？ | ⚠️ |

**合理的卖出理由：**
- [ ] 我需要用这笔钱
- [ ] 投资逻辑发生了根本变化
- [ ] 有更好的投资机会

## 危机时检查

当市场大跌（> 20%）时：

1. **不要打开账户看数字**
2. **不要看财经新闻**
3. **问自己这些问题：**

| 问题 | 答案 |
|------|------|
| 这是暂时的恐慌还是真的出问题了？ | _________________ |
| 我持有的公司还在赚钱吗？ | _________________ |
| 我有现金可以加仓吗？ | _________________ |
| 我能承受再跌 30% 吗？ | _________________ |

**如果公司基本面没变，不要卖。**
