## 五大核心教導 | Five Key Lessons

### 1. Kill Your Darlings | 殺死你的心頭好
> 不要對股票有情感依附

Nicholas Darvas 建議投資者不要對任何股票產生情感依附。即使你喜歡一家公司或過去曾成功持有它，如果它停止表現良好，你必須願意賣出。情感依附會蒙蔽你的判斷並導致糟糕的投資決策。始終專注於股票的表現和潛力，而非你對它的個人感受。

**情感依附的危害**:
- 拒絕承認錯誤
- 持有虧損太久
- 錯過更好的機會
- 基於過去而非未來

**實踐要點**:
- 定期檢視持股是否仍符合標準
- 設定止損點並嚴格執行
- 不要讓過去的成功影響當前判斷

---

### 2. Buy on Strong Positive Trends and Rising Volumes | 在強勁上漲趨勢和成交量增加時買入
> 動量是你的朋友

Darvas 強調購買顯示強勁價格上漲趨勢同時交易量增加的股票。這表明股票正在獲得動力和投資者興趣。通過關注這些信號，你可以識別可能繼續上漲的股票，給你更好的獲利機會。

**Darvas 箱形理論**:
```
股價
  ↑
  │    ┌────────┐  ← 新箱形頂部
  │    │        │
  │    │  買入  │  突破箱形時買入
  │    │   ↑   │
  │ ┌──┴────────┴──┐
  │ │              │ ← 前一個箱形
  │ └──────────────┘
  │
  └────────────────→ 時間
```

**買入信號**:
- 股價創新高
- 成交量顯著增加
- 突破前一個交易區間

**實踐要點**:
- 只買強勢股
- 關注成交量確認
- 等待突破再進場

---

### 3. Cut Your Losses Short, Let Your Winners Run | 快速止損，讓贏家奔跑
> 小虧大賺的藝術

Darvas 相信最小化損失並最大化收益。如果你買的股票開始下跌，快速賣出以防止進一步損失。相反，如果股票表現良好並繼續上漲，持有它以充分利用其上升動力。這種策略幫助你保護資本並更有效地增長投資。

**損益管理規則**:
| 情況 | 行動 |
|------|------|
| 股價跌破止損點 | 立即賣出 |
| 股價在箱形內 | 持有觀察 |
| 股價突破新高 | 繼續持有 |
| 股價創新箱形 | 調高止損點 |

**實踐要點**:
- 設定嚴格的止損點（如-8%）
- 不要設定獲利了結點
- 讓市場告訴你何時賣出

---

### 4. Create Your Own Stock Logbook | 建立你自己的股票日誌
> 記錄是學習的基礎

Darvas 建議保持詳細的股票投資日誌。記錄你買入每支股票的理由、你的觀察，以及你交易的結果。這本日誌幫助你從經驗中學習、識別模式，並隨時間改進你的投資策略。它是自律和持續學習的工具。

**日誌應記錄的內容**:
- 買入/賣出日期和價格
- 買入/賣出理由
- 當時的市場環境
- 事後反思和教訓

**實踐要點**:
- 每筆交易都要記錄
- 定期回顧和分析
- 從錯誤中提煉規則

---

### 5. Be the Lone Wolf — Stay Detached | 做一匹孤狼 — 保持超然
> 獨立於市場噪音

Darvas 建議投資者保持獨立，避免跟隨人群。根據流行意見或趨勢做投資決策可能導致糟糕的結果。相反，做自己的研究，相信你的分析，保持超脫於市場炒作和恐慌。這種孤狼心態幫助你保持客觀並做出更理性的投資選擇。

**孤狼策略**:
- 不看財經新聞（減少噪音）
- 不聽經紀人建議
- 只看價格和成交量
- 遠離市場情緒

**實踐要點**:
- 減少市場噪音的干擾
- 建立自己的系統並堅持
- 不要讓他人影響你的決策

## Darvas 箱形理論詳解 | Darvas Box Theory

**核心概念**:
1. 股票在箱形區間內交易
2. 突破箱形頂部是買入信號
3. 跌破箱形底部是賣出信號
4. 隨著股價上漲，調高止損點

**實際應用**:
```
            ┌─────────────┐
            │  新箱形     │ ← 股價在新區間
       ┌────┼─────────────┤
       │    │  突破買入 ↗ │
  ┌────┼────┴─────────────┘
  │    │  前箱形
  │    │  止損設在底部 ↓
  └────┴──────────────────
```
