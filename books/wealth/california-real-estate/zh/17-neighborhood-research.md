## 一、学区分析

### School District Analysis

### 数据来源

| 来源 | 信息 |
|-----|-----|
| **GreatSchools.org** | 综合评分 1-10 |
| **Niche.com** | 详细分析、排名 |
| **California School Dashboard** | 官方数据 |
| **学区官网** | 边界、政策 |

### 评估要点

| 指标 | 好学区标准 |
|-----|----------|
| GreatSchools 评分 | > 8/10 |
| 测试成绩 | 超过州平均 |
| 师生比 | < 25:1 |
| 大学录取率 | > 80% |
| 家长满意度 | 高评价 |

### 学区边界陷阱

```
注意：
- 同一街道可能分属不同学区
- 地图可能过时
- 必须与学区确认

验证方法：
1. 学区官网查地址
2. 致电学区确认
3. 检查最近成交的学区信息
```

### 学区对房价的影响

| 学区评分 | 价格溢价 |
|---------|---------|
| 10/10 | +20-30% |
| 8-9/10 | +10-15% |
| 6-7/10 | 基准 |
| < 6/10 | -10-20% |

## 二、交通分析

### Transportation Analysis

### 评估指标

| 交通方式 | 评估标准 |
|---------|---------|
| **Caltrain** | 步行 < 15分钟 |
| **BART** | 步行 < 15分钟 |
| **公交** | 步行 < 5分钟 |
| **高速公路** | 车程 < 5分钟 |
| **机场** | 车程 < 30分钟 |

### 通勤时间测试

```
关键问题：
- 到主要科技公司多久？
- 高峰期实际通勤时间？
- 有多种交通选择？

测试方法：
1. Google Maps 实时导航
2. 不同时段测试
3. 询问当地居民
```

### 交通对房价的影响

| 因素 | 影响 |
|-----|-----|
| 步行到 Caltrain | +5-10% |
| 靠近高速入口 | +3-5% |
| 远离公共交通 | -5-10% |
| 交通噪音 | -5-15% |

## 四、安全分析

### Safety Analysis

### 数据来源

| 来源 | 信息 |
|-----|-----|
| **CrimeMapping.com** | 实时犯罪数据 |
| **SpotCrime** | 犯罪趋势 |
| **City-Data.com** | 综合统计 |
| **本地警局** | 官方数据 |

### 评估要点

| 指标 | 好区标准 |
|-----|---------|
| 暴力犯罪率 | 低于州平均 |
| 财产犯罪率 | 低于州平均 |
| 趋势 | 下降或稳定 |
| 主观感受 | 安全舒适 |

### 实地考察

```
安全性实地评估：

白天：
- 街道整洁度
- 房屋维护状况
- 居民活动
- 商业状况

夜晚：
- 照明情况
- 街道活动
- 停车状况
- 噪音水平
```

### Amenities Analysis

### 评估清单

| 类别 | 评估项 |
|-----|-------|
| **购物** | 超市、商场距离 |
| **餐饮** | 餐厅质量和多样性 |

### Walk Score

| 分数 | 含义 |
|-----|-----|
| 90-100 | 步行者天堂 |
| 70-89 | 非常适合步行 |
| 50-69 | 有些适合步行 |
| 25-49 | 依赖汽车 |
| 0-24 | 几乎完全依赖汽车 |

## 区域评分表

### Neighborhood Scorecard

| 维度 | 权重 | 评分 (1-5) | 加权分 |
|-----|-----|-----------|-------|
| 学区 | 25% | | |
| 交通 | 20% | | |
| 就业 | 20% | | |
| 安全 | 15% | | |
| 便利 | 10% | | |
| 趋势 | 10% | | |
| **总分** | 100% | | |

**决策**：
- 总分 > 4.0：优质区域
- 总分 3.5-4.0：良好区域
- 总分 3.0-3.5：一般区域
- 总分 < 3.0：谨慎或避免

## 本章总结

### Chapter Summary

区域研究是投资的基础：

| 维度 | 核心问题 |
|-----|---------|
| **学区** | 孩子能上好学校吗？ |
| **交通** | 通勤方便吗？ |
| **就业** | 有好工作吗？ |
| **安全** | 安全吗？ |
| **便利** | 生活方便吗？ |
| **趋势** | 在变好还是变差？ |

**核心原则**：

> "夫地形者，兵之助也。"

了解区域，才能做出正确的投资决策。
