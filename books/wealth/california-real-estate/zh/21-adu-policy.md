## ADU 政策演进

### Evolution of ADU Policy

### 时间线

| 年份 | 法案 | 主要变化 |
|-----|-----|---------|
| **2017** | SB 1069, AB 2299 | 开始放松限制 |
| **2020** | AB 68, AB 881 | 重大松绑 |
| **2021** | SB 9 | 允许 lot split |
| **2022** | AB 2221 | 进一步简化 |
| **2024** | 持续更新 | 各城市细化 |

### 2020年重大变化

```
之前的限制：
- 只允许在单户区域
- 严格的 setback 要求
- 停车要求
- 业主自住要求
- 审批时间长

2020年后：
- 所有住宅区域都可以
- Setback 放宽到 4 英尺
- 取消额外停车要求
- 取消业主自住要求（州法）
- 审批时间限制 60 天
```

## ADU 类型详解

### Types of ADUs

### 三种主要类型

| 类型 | 说明 | 优势 | 劣势 |
|-----|-----|-----|-----|
| **Detached ADU** | 独立新建 | 隐私好，租金高 | 成本最高 |
| **Attached ADU** | 与主屋相连 | 成本适中 | 需改动主屋 |
| **Conversion ADU** | 改建车库/空间 | 成本最低 | 面积受限 |

### 加上 JADU

```
JADU (Junior ADU)：
- 面积：500 sqft 以内
- 位置：必须在主屋内部
- 厨房：可以是效率厨房
- 入口：可以共享
- 业主：州法要求业主自住

优势：
- 许可更简单
- 成本更低
- 可以和 ADU 同时建

限制：
- 面积小
- 租金相对低
- 业主自住要求
```

### 最大配置

```
单户地块最大可能：
主屋（1单元）
  +
ADU（最大 1,200 sqft）
  +
JADU（最大 500 sqft）
  =
3个单元

多户地块：
现有单元
  +
最多2个 Detached ADU（800 sqft each）
  +
多个 Conversion ADU（每个现有单元一个）
```

## 州法核心规定

### State Law Key Provisions

### 面积规定

| 类型 | 最小 | 最大 |
|-----|-----|-----|
| **Studio/1BR ADU** | 150 sqft | 850 sqft |
| **2+BR ADU** | 150 sqft | 1,000 sqft |
| **Detached（任何）** | 150 sqft | 1,200 sqft |
| **JADU** | 150 sqft | 500 sqft |

### Setback 规定

```
州法最低要求：

后院/侧院 Setback：
- 最少 4 英尺
- 比传统规定（如 10-15 英尺）大幅减少

高度：
- 最高 16 英尺（1层）
- 某些情况可以 2 层

特殊情况：
- 改建现有结构不需要满足 setback
- 只要不扩大非法不合规部分
```

### 停车规定

```
州法：
- 不要求额外停车位

例外（可能要求停车）：
- 距离公交站 0.5 英里以外
- 不在历史街区
- 不是 Car Share 区域

实际情况：
大部分 Bay Area 都在公交覆盖范围内
= 不需要额外停车
```

### 审批时限

| 申请类型 | 时限 |
|---------|-----|
| **完整申请** | 60 天 |
| **不完整→补充** | 30 天回复 |
| **超时未回复** | 自动批准 |

## 地方政策差异

### Local Policy Variations

### 友好城市 vs 不友好城市

| 城市 | ADU 友好度 | 审批时间 | 特殊规定 |
|-----|----------|---------|---------|
| **Mountain View** | ⭐⭐⭐⭐⭐ | 60天 | 鼓励 ADU |
| **Sunnyvale** | ⭐⭐⭐⭐ | 60-90天 | 较友好 |
| **Palo Alto** | ⭐⭐⭐ | 90-120天 | 设计审查 |
| **San Jose** | ⭐⭐⭐⭐ | 60-90天 | 大规模推广 |
| **Fremont** | ⭐⭐⭐⭐ | 60-90天 | 较友好 |
| **Cupertino** | ⭐⭐⭐ | 90天+ | 设计要求高 |

### 地方可能的额外限制

```
州法允许地方增加的限制（有限）：

1. 设计审查
   - 外观匹配主屋
   - 材料要求

2. 租赁限制
   - 某些城市禁止短租
   - 最短租期要求（如 30 天）

3. 业主自住要求
   - 某些城市要求业主住在主屋或 ADU
   - 但正在逐渐取消

4. 影响费
   - 州法限制 750 sqft 以下不能收
   - 750 sqft 以上可按比例收
```

## ADU 财务分析

### ADU Financial Analysis

### 成本估算

| 项目 | Conversion | Attached | Detached |
|-----|-----------|----------|----------|
| **每 sqft 成本** | $200-350 | $350-450 | $400-550 |
| **500 sqft 总成本** | $100-175K | $175-225K | $200-275K |
| **800 sqft 总成本** | $160-280K | $280-360K | $320-440K |
| **设计+许可** | $15-25K | $25-35K | $30-45K |

### 成本明细

```
典型 650 sqft Detached ADU 成本：

硬成本：
地基：$25,000
框架：$35,000
屋顶：$18,000
MEP（水电暖）：$45,000
内装：$55,000
外装：$22,000
硬成本小计：$200,000

软成本：
设计：$15,000
许可：$12,000
检查：$3,000
项目管理：$10,000
Contingency (10%)：$24,000
软成本小计：$64,000

总成本：$264,000

```

### 收益分析

```
案例：Mountain View ADU

投资：
ADU 建造成本：$280,000
融资成本（HELOC 1年）：$14,000
总投资：$294,000

收益：
ADU 月租金：$2,800
年租金：$33,600
运营费（5%）：$1,680
净收入：$31,920

回报率：
现金回报：$31,920 / $294,000 = 10.9%
物业增值：$350,000（保守估计）
总回报：($31,920 + $350,000) / $294,000 = 130%

回本期：$294,000 / $31,920 = 9.2 年
```

## ADU 审批流程

### ADU Permit Process

### 标准流程

```
Step 1: 前期研究（2-4周）
├── 查 Zoning
├── 查 Setback/Coverage
├── 查地方 ADU 规定
└── 确定可行性

Step 2: 设计（4-8周）
├── 建筑设计
├── 结构工程
├── MEP 工程
└── Title 24 能源合规

Step 3: 提交申请（1周）
├── 完整图纸
├── 申请表
├── 费用缴纳
└── 等待受理

Step 4: 审批（60-120天）
├── 计划审查
├── 修改回复
├── 最终批准
└── 获得许可

Step 5: 施工（3-6个月）
├── 地基检查
├── 框架检查
├── MEP 检查
├── 最终检查
└── 获得 CO

总时间：8-14个月
```

### 加速技巧

| 技巧 | 效果 |
|-----|-----|
| **预制 ADU** | 减少施工时间 |
| **标准设计** | 审批更快 |
| **经验丰富设计师** | 减少来回修改 |
| **完整文件** | 避免被退回 |
| **熟悉当地规定** | 一次通过 |

## 预制 ADU

### Prefab ADU Options

### 预制 vs 现场建造

| 因素 | 预制 ADU | 现场建造 |
|-----|---------|---------|
| **成本** | 可能略高 | 更可控 |
| **时间** | 施工快（2-4周） | 3-6个月 |
| **质量** | 工厂标准化 | 取决于 GC |
| **定制** | 有限 | 完全定制 |
| **审批** | 可能更简单 | 标准流程 |

### 预制 ADU 供应商

```
Bay Area 常见供应商：

Abodu
- 价格：$200K-350K（不含地基、接驳）
- 特点：高端设计，快速安装
- 适合：预算充足，追求设计

Villa
- 价格：$150K-250K
- 特点：多种尺寸选择
- 适合：平衡性价比

Prefab Logic
- 价格：$180K-300K
- 特点：本地制造
- 适合：定制需求

注意：
报价通常不含：
- 地基（$25-50K）
- Utility 接驳（$20-50K）
- 许可费（$10-20K）
- 景观（$10-20K）
```

## ADU 租控与税务

### ADU Rent Control & Tax

### 租控豁免

```
州法 AB 1482 对 ADU 的规定：

豁免条件（满足任一）：
1. 15年内新建
2. 业主住在主屋或 ADU

实际影响：
- 大多数新建 ADU 自动豁免 15 年
- 建议业主自住其一以保持豁免

地方租控：
- 查看当地 ADU 特别规定
- 某些城市可能有额外限制
```

### 税务影响

| 项目 | 影响 |
|-----|-----|
| **房产税** | ADU 会触发 supplemental tax |
| **所得税** | ADU 租金收入需报税 |
| **折旧** | 可以折旧 ADU 成本 |
| **1031** | ADU 部分可能需要特殊处理 |

### Supplemental Tax 计算

```
案例：

ADU 完工前：
主屋评估值：$800,000
年税：$8,000

ADU 完工后：
重新评估：主屋 $800,000 + ADU $280,000
新评估值：$1,080,000
新年税：$10,800

增加：$2,800/年

注意：
- 仅 ADU 部分重估
- 主屋保持 Prop 13 税基
```

## 常见问题与解决

### Common Issues & Solutions

### 问题一：公用设施接驳

```
问题：
- 水电气接驳成本高
- 需要挖掘
- 可能需要升级主线

解决方案：
- 提前与 Utility 公司沟通
- 预算 $20-50K 用于接驳
- 考虑位置规划减少管线长度
```

### 问题二：邻居反对

```
问题：
- 邻居可能投诉
- 影响审批或施工

解决方案：
- 提前与邻居沟通
- 设计考虑隐私（窗户朝向）
- 施工期间保持良好关系
- 选择专业安静的施工团队
```

### 问题三：成本超支

```
问题：
- 实际成本超过预算
- 隐藏费用多

解决方案：
- 预留 15-20% Contingency
- 详细的施工合同
- 固定价格合同
- 经验丰富的 GC
- 定期检查进度
```

## 本章总结

### Chapter Summary

ADU 是加州投资者的重要工具：

| 要点 | 内容 |
|-----|-----|
| **政策** | 2020年后大幅松绑 |
| **类型** | Detached、Attached、Conversion、JADU |
| **成本** | $150K-450K 取决于类型和面积 |
| **回报** | 10%+ 现金回报，显著增值 |
| **时间** | 8-14个月从设计到完工 |

**核心策略**：

| 策略 | 适用 |
|-----|-----|
| **Conversion** | 成本敏感，有现有空间 |
| **Detached** | 追求最高租金，有空间 |
| **Prefab** | 追求速度，预算充足 |
| **JADU + ADU** | 最大化单元数 |

**核心原则**：

> "善用兵者，役不再籍，粮不三载。"

ADU 是"就地取材"——利用现有土地创造新价值，比购买新物业更高效。
