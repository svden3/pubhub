## 什么是 1031 交换？

### What is a 1031 Exchange?

**IRC Section 1031** 允许投资者在出售投资物业时，将资本利得税延迟到未来的交易中。

**基本原理**：

```
普通出售：
卖出物业A → 缴纳资本利得税 → 剩余资金买物业B

1031交换：
卖出物业A → 延迟税款 → 全部资金买物业B
```

**税务影响 (加州)**：

| 税种 | 税率 | 说明 |
|-----|-----|-----|
| 联邦长期资本利得 | 0-20% | 取决于收入 |
| 联邦净投资收入税 | 3.8% | 高收入纳税人 |
| 加州资本利得 | 9.3-13.3% | 作为普通收入 |
| 折旧回收 (Section 1250) | 25% | 联邦 |
| **总计** | **30-45%** | |

**例子**：

```
物业A购买价：$500,000
物业A卖出价：$1,500,000
资本利得：$1,000,000
折旧回收：$200,000

不做1031的税负：
- 联邦资本利得 (23.8%): $190,400
- 加州 (13.3%): $133,000
- 折旧回收 (25%): $50,000
- 总税负: ~$373,400

做1031交换：
- 当前税负: $0
- 延迟: $373,400
```

## 1031 交换的类型

### Types of 1031 Exchanges

| 类型 | 说明 | 复杂度 |
|-----|-----|-------|
| **Simultaneous** | 同一天完成交割 | 低 |
| **Delayed (最常见)** | 先卖后买，180天内完成 | 中 |
| **Reverse** | 先买后卖 | 高 |
| **Improvement** | 对新物业进行改善 | 高 |

**本章聚焦**：Delayed Exchange（最实用）

## 1031 交换的规则

### 1031 Exchange Rules

### 关键规则

| 规则 | 要求 | 违反后果 |
|-----|-----|---------|
| **Like-Kind** | 投资/商业物业换投资/商业物业 | 不符合1031 |
| **45天规则** | 卖出后45天内确定替代物业 | 交换失败 |
| **180天规则** | 卖出后180天内完成交割 | 交换失败 |
| **Qualified Intermediary** | 必须使用第三方QI | 交换失败 |
| **Equal or Greater Value** | 替代物业价值 ≥ 卖出物业 | 部分应税 |

### "Like-Kind" 的定义

**符合条件**：

| 卖出 | 买入 | 符合？ |
|-----|-----|-------|
| 出租房 | 出租房 | ✓ |
| 出租房 | 商业物业 | ✓ |
| 商业物业 | 土地 | ✓ |
| 出租房 | 多个出租房 | ✓ |

**不符合条件**：

| 卖出 | 买入 | 原因 |
|-----|-----|-----|
| 自住房 | 任何 | 自住不符合 |
| 出租房 | 股票 | 不是不动产 |
| 出租房 | 美国以外物业 | 必须美国境内 |
| 翻修物业(Dealer) | 任何 | Dealer不符合 |

### 45天规则详解

**三种识别方法**：

| 方法 | 规则 | 适用情况 |
|-----|-----|---------|
| **三物业规则** | 最多识别3个物业，无价值限制 | 最常用 |
| **200%规则** | 可识别更多，但总价 ≤ 卖出价200% | 多选项 |
| **95%规则** | 任意数量，但必须买95%+ | 专业投资者 |

**实务建议**：
- 使用三物业规则最简单
- 识别3个，最终只需买1个
- 识别必须书面，交给QI

### Boot（应税部分）

**什么会产生Boot？**

| 情况 | Boot金额 | 应税 |
|-----|---------|-----|
| 收到现金 | 收到金额 | ✓ |
| 新物业价值 < 旧物业 | 差额 | ✓ |
| 新贷款 < 旧贷款 | 差额 | ✓ |
| 非like-kind财产 | 价值 | ✓ |

**避免Boot**：
- 新物业价值 ≥ 旧物业价值
- 新贷款 ≥ 旧贷款
- 不取出任何现金

## 1031 交换流程

### 1031 Exchange Process

### 时间线

```
Day 0: 卖出物业交割 (Close of Relinquished Property)
         ↓
    资金转入QI托管账户
         ↓
Day 1-45: 寻找替代物业
         ↓
Day 45: 识别截止日 (Identification Deadline)
         ↓
Day 46-180: 完成购买交割
         ↓
Day 180: 交换截止日 (Exchange Deadline)
```

### Step-by-Step

**Step 1：准备阶段**

| 任务 | 时机 | 负责方 |
|-----|-----|-------|
| 确认物业符合1031条件 | 出售前 | 投资者+CPA |
| 选择Qualified Intermediary | 出售前 | 投资者 |
| 通知QI交易详情 | 合同签署时 | 投资者 |
| 修改买卖合同加入1031条款 | 合同阶段 | 律师 |

**Step 2：出售旧物业**

| 任务 | 说明 |
|-----|-----|
| 签署交换协议 | 与QI签署 |
| 分配协议 | 将合同权益分配给QI |
| 交割指示 | Title公司将资金直接转给QI |
| **关键**：**不能触碰资金** | |

**Step 3：45天识别期**

| 任务 | 要求 |
|-----|-----|
| 寻找替代物业 | 积极看房 |
| 书面识别 | 地址+法律描述 |
| 送达QI | 签名+日期 |
| **截止日严格** | 45天 = 45天，无例外 |

**Step 4：购买新物业**

| 任务 | 说明 |
|-----|-----|
| 合同谈判 | 正常购买流程 |
| 加入1031条款 | 合同中注明 |
| 资金来源 | QI直接支付 |
| 交割 | 180天内完成 |

## 选择 Qualified Intermediary

### Selecting a QI

**QI的作用**：
- 持有交换资金
- 确保符合IRS规则
- 准备所有文书
- 协调交割

**选择标准**：

| 标准 | 检查内容 |
|-----|---------|
| **经验** | 至少5年，100+交易 |
| **保险** | Errors & Omissions保险 |
| **资金保护** | Fidelity bond, 隔离账户 |
| **费用** | 透明，无隐藏费用 |
| **评价** | 业内声誉 |

**典型费用**：

| 服务 | 费用范围 |
|-----|---------|
| 基本交换服务 | $600-$1,500 |
| 反向交换 | $2,500-$5,000+ |
| 改善交换 | $2,500-$5,000+ |

**推荐QI公司**：
- IPX 1031
- Exeter 1031
- First American Exchange
- Old Republic Exchange

## 加州特殊考虑

### California-Specific Considerations

### Clawback规则

加州有**追回条款 (Clawback)**：

| 规则 | 说明 |
|-----|-----|
| 卖出加州物业 | 加州税延迟 |
| 买入外州物业 | 加州仍有权追税 |
| 未来卖出外州物业 | 加州可追回原延迟税 |

**报告要求**：
- FTB 3840 (每年申报)
- 记录1031交换详情
- 持续报告至物业最终出售

**策略**：
- 可以1031出加州，但税务追踪复杂
- 考虑留在加州简化合规
- 咨询加州税务专家

### 实体结构注意

| 情况 | 处理 |
|-----|-----|
| 个人名义持有 | 最简单 |
| Single-member LLC | Pass-through，与个人相同 |
| Multi-member LLC | 复杂，需要drop-and-swap |
| S-Corp/C-Corp | 通常不适合1031 |

**Drop-and-Swap**：
- 先将LLC权益分配给成员
- 成员再以个人名义做1031
- 需要提前规划

## 高级策略

### Advanced Strategies

### 策略1：步步高升 (Trade Up)

```
第一套：$500K → 第二套：$800K → 第三套：$1.2M → ...

每次1031交换，复利增长，延迟所有税款
```

**优势**：
- 全部资金用于增值
- 复利效应最大化
- 可能最终免税（见策略3）

### 策略2：分散投资

```
卖出：$2M物业
买入：4个$500K物业

分散风险，增加现金流
```

**1031规则允许**：
- 一换多 (1 to many)
- 多换一 (many to 1)
- 多换多 (many to many)

### 策略3：终极退出 — Step-Up Basis

```
策略：持有到去世

去世时：继承人获得Step-Up Basis
结果：之前所有递延税款 → 归零
```

**例子**：

```
原始购买：$500K
历次1031后现值：$3M
累计递延税款：$500K+

去世时Step-Up：$3M
继承人税基：$3M
之前递延：$0
```

**注意**：
- 需要长期规划
- 考虑遗产税（$12.92M免税额，2023）
- 咨询遗产律师

### 策略4：Delaware Statutory Trust (DST)

**什么是DST？**
- 被动投资工具
- 符合1031条件
- 专业管理的物业组合

**适用情况**：
- 年龄大，不想管理物业
- 找不到合适替代物业
- 想分散到多个物业

**优缺点**：

| 优点 | 缺点 |
|-----|-----|
| 被动收入 | 流动性低 |
| 专业管理 | 控制权少 |
| 1031符合 | 费用较高 |
| 分散风险 | 需要认证投资者 |

## 常见错误与避免

### Common Mistakes to Avoid

| 错误 | 后果 | 避免方法 |
|-----|-----|---------|
| **触碰资金** | 交换失败 | 资金只能QI持有 |
| **45天超期** | 交换失败 | 严格记录日期 |
| **180天超期** | 交换失败 | 提前锁定物业 |
| **不合格物业** | 应税 | 确认投资用途 |
| **Boot产生** | 部分应税 | 价值和贷款匹配 |
| **不报告** | 罚款 | 每年FTB 3840 |

## 1031 交换检查表

### 1031 Exchange Checklist

**出售前**：
- ☐ 确认物业符合1031（持有>1年，投资用途）
- ☐ 评估潜在税负
- ☐ 选择并联系QI
- ☐ 通知agent和律师1031意图
- ☐ 修改买卖合同加入1031条款

**出售时**：
- ☐ 签署交换协议（与QI）
- ☐ 签署分配协议
- ☐ 确认资金直接转给QI
- ☐ 记录Day 0日期
- ☐ 开始寻找替代物业

**45天内**：
- ☐ 积极看房
- ☐ Day 30前有候选名单
- ☐ 准备书面识别文件
- ☐ Day 45前送达QI
- ☐ 保留签收证明

**180天内**：
- ☐ 锁定购买合同
- ☐ 合同中加入1031条款
- ☐ 确认资金来源为QI
- ☐ 完成尽调
- ☐ Day 180前交割

**交割后**：
- ☐ 获取交割文件
- ☐ 更新保险
- ☐ 保留所有1031文件
- ☐ 税务年度申报Form 8824
- ☐ 加州申报FTB 3840

| 要点 | 内容 |

> "上兵伐谋，其次伐交，其次伐兵，其下攻城。"

最高明的税务策略不是硬抗税务，而是合法延税，让资金持续为你工作。
