Prop 19 改变了加州房产继承的游戏规则。看似简单的条款，却有深远的影响。本章深入分析这些变化及应对策略。

---

## Prop 19 是什么

### What is Proposition 19?

### 背景

| 事项 | 内容 |
|-----|-----|
| **通过时间** | 2020年11月 |
| **生效时间** | 2021年2月16日 |
| **前身** | Prop 58/193 (1986) |
| **目的** | 增加地方税收、帮助老年人 |

### 两大变化

```
变化一：老年人/残疾人/灾民可转移税基
  - 可以带着低税基搬家
  - 全州范围内（之前仅限同县）
  - 可转移3次（之前1次）

变化二：限制继承时的税基转移
  - 之前：子女可继承父母的低税基
  - 现在：仅限主居且子女必须自住
```

---

## 继承规则的变化

### Changes to Inheritance Rules

### 之前 (Prop 58/193)

```
父母 → 子女 继承：

主居：
- 无限制地保留父母税基
- 子女可以自住或出租

投资物业：
- 最多 $1M 评估价值可转移
- 子女可继承低税基

祖父母 → 孙子女：
- 父母已故的情况下可转移
```

### 现在 (Prop 19)

```
父母 → 子女 继承：

主居：
- 子女必须在继承后1年内自住
- 必须申请 Homeowner's Exemption
- 如果 FMV > 税基 + $1M，差额需重新评估
- 如果不自住 → 全部重新评估

投资物业：
- 100% 重新评估
- 无例外

祖父母 → 孙子女：
- 同样规则
```

### 数学影响

**案例：继承主居**

```
父母购买价（1980年）：$100,000
父母税基（2024年）：$220,000（每年 +2%）
市场价值（2024年）：$2,500,000

之前 (Prop 58)：
子女税基：$220,000
年税：$2,200

现在 (Prop 19) - 子女自住：
子女税基：$220,000 + ($2,500,000 - $220,000 - $1,000,000)
        = $220,000 + $1,280,000
        = $1,500,000
年税：$15,000

现在 (Prop 19) - 子女不自住：
子女税基：$2,500,000
年税：$25,000
```

**案例：继承投资物业**

```
父母购买价：$200,000
父母税基：$440,000
市场价值：$2,000,000

之前：
子女税基：$440,000（可保留部分）
年税：$4,400

现在：
子女税基：$2,000,000
年税：$20,000

差异：$15,600/年
```

## 对投资的影响

### Impact on Investment

### 影响一：增加市场供应

```
逻辑链：
继承税基不再保留
  ↓
持有成本大增
  ↓
更多继承人选择出售
  ↓
市场供应增加
  ↓
可能的价格压力
```

### 影响二：投资策略调整

| 之前策略 | 现在策略 |
|---------|---------|
| 持有到去世，传给子女 | 需要重新评估 |
| 依赖低税基 | 考虑 1031 或出售 |
| 被动持有 | 更积极管理 |

### 影响三：机会出现

```
机会来源：
- 继承人被迫出售
- Probate 出售
- 不想负担高税的继承人
- 缺乏流动性的继承人

投资者策略：
- 关注 Probate 销售
- 监控继承后出售
- 准备快速出价
```

## 规划策略

### Planning Strategies

### 策略一：生前转移

```
问题：Prop 19 生效后转移仍然重估

可能的例外：
- 配偶之间转移（不触发）
- 某些家庭信托安排

注意：
- 需要咨询税务律师
- 规则复杂且可能变化
```

### 策略二：LLC 结构

```
传统想法：
用 LLC 持有 → 转移 LLC 份额 → 避免重估

现实：
- 大多数情况仍会触发重估
- Prop 19 扩大了"所有权变更"定义
- 需要非常谨慎的结构设计

建议：
- 咨询专业律师
- 不要自己尝试
```

### 策略三：出售后 1031

```
逻辑：
如果要被重估，不如现在卖出做 1031
  ↓
1. 延迟资本利得税
2. 换到更好的物业
3. 重新开始 Prop 13 锁定

适用情况：
- 投资物业（反正要重估）
- 有好的替代物业
- 长期投资视角
```

### 策略四：考虑出售

```
分析：
持有成本上升 → 可能出售更划算

计算：
继承后年税：$20,000
之前年税：$4,000
增加成本：$16,000/年

10年增加：$160,000

问题：
这笔成本是否值得继续持有？
还是应该出售、重新配置？
```

## 主居规划

### Primary Residence Planning

### 子女自住策略

```
条件：
1. 继承后1年内申请 Homeowner's Exemption
2. 实际自住（不是投资）
3. 符合其他要求

好处：
- 保留父母税基（部分）
- 减少税负增加

限制：
- 必须自住
- 不能出租
- 如果超过 $1M 差额仍需调整
```

### 多子女继承

```
问题：
- 多个子女共同继承
- 只有一个能自住

解决方案：
- 一人买断其他人份额
- 共同持有（复杂）
- 出售后分配

需要：
- 提前规划
- 遗嘱/信托安排
- 家庭沟通
```

## 老年人策略

### Senior Strategies

### Prop 19 对老年人的好处

```
新权利：
- 可以带着低税基搬家
- 全州范围
- 最多3次

条件：
- 55岁以上
- 或残疾人
- 或灾民

限制：
- 新房价值有上限
- 需要在2年内完成
```

### 应用案例

```
案例：
老年人在 Palo Alto 有房，想搬到便宜地区

之前：
- 搬家 = 失去低税基
- 阻止搬家

现在：
- 可以带税基搬家
- 更多选择

对市场影响：
- 可能释放更多 Peninsula 房源
- 老年人更愿意 downsize
```

## 本章总结

### Chapter Summary

Prop 19 深刻改变了继承规则：

| 变化 | 影响 |
|-----|-----|
| **投资物业** | 100% 重估，无法继承税基 |
| **主居** | 仅自住可部分保留 |
| **老年人** | 可转移税基搬家 |

**策略应对**：

| 情况 | 策略 |
|-----|-----|
| 持有投资物业 | 考虑 1031 或出售 |
| 计划传承 | 咨询专业律师 |
| 老年人 | 利用新的转移权利 |
| 投资者 | 关注继承出售机会 |

**核心原则**：

> "兵者，诡道也。"

规则已变，策略必须调整。不了解新规则的人将付出代价。
