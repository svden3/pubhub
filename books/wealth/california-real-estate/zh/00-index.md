### 1. 加州地产五事 (Five Factors)

| 因素 | 内涵 | 评估重点 |

| **地** | 区位与护城河 | 学区, 交通, 供应限制 |
| **将** | 团队与能力 | GC, Agent, 合伙人 |
| **法** | 结构与纪律 | 实体, 税务, 融资 |

### 2. 安全边际公式

```
ARV (After Repair Value)
- 购买价格
- 硬成本 (施工)
- 软成本 (设计, 许可)
- 持有成本 (利息, 税)
- 销售成本 (佣金, 过户)
- 意外准备 (10-15%)
─────────────────────────
= 净利润

安全边际 = (ARV - 总成本) / ARV
目标：> 20%
```

### 3. 决策矩阵

| 评分 | 决策 |
|-----|------|
| 总分 > 4.0 | 可以投资 |
| 总分 3.5-4.0 | 需要更多分析 |
| 总分 < 3.5 | 不投资 |
| "地" < 3.5 | 一票否决 |
