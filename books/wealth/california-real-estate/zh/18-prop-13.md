## Chapter 18: Proposition 13 Deep Dive

---

> "道者，令民与上同意也，故可以与之死，可以与之生，而不畏危。"
>
> "The Moral Law causes the people to be in complete accord with their ruler."
>
> — 《孙子兵法·始计篇》

**Proposition 13 (1978)** 是加州地产的根本之"道"——它定义了加州地产投资的基本规则。理解 Prop 13，才能理解加州地产的定价逻辑和投资策略。

---

## Prop 13 是什么

### What is Proposition 13?

### 历史背景

| 年份 | 事件 |
|-----|-----|
| 1970s | 加州房价快速上涨，房产税随之飙升 |
| 1978 | Howard Jarvis 推动公投 |
| 1978年6月 | 以 65% 支持率通过 |
| 之后 | 成为加州宪法一部分 |

### 核心条款

| 条款 | 内容 | 影响 |
|-----|-----|-----|
| **1% 税率** | 房产税不超过评估价的 1% | 限制税负 |
| **税基锁定** | 税基 = 购买价格 | 长期持有者优势 |
| **2% 上限** | 每年评估价最多涨 2% | 增长受限 |
| **重新评估** | 产权转移时重新评估 | 买入成本高 |

## Prop 13 的数学

### The Math of Prop 13

### 长期持有的税务优势

```
案例：1985年购买 vs 2024年购买

1985年买家：
购买价：$200,000
2024年税基 = $200,000 × (1.02)^39 = $433,000
2024年房产税 = $433,000 × 1% = $4,330/年

2024年买家：
购买价：$2,000,000
2024年税基 = $2,000,000
2024年房产税 = $2,000,000 × 1% = $20,000/年

差异：4.6 倍！
```

### 税负随时间变化

| 持有年数 | 税基增长 | 假设市值 | 有效税率 |
|---------|---------|---------|---------|
| 0 | 1.00x | $2M | 1.00% |
| 10 | 1.22x | $2.5M | 0.49% |
| 20 | 1.49x | $3.2M | 0.47% |
| 30 | 1.81x | $4M | 0.45% |
| 40 | 2.21x | $5M | 0.44% |

**结论**：持有越久，有效税率越低。

## Prop 13 对投资策略的影响

### Impact on Investment Strategy

### 影响一：鼓励长期持有

| 策略 | Prop 13 影响 |
|-----|-------------|
| 长期持有 | ✓✓ 税负锁定，越持有越划算 |
| 频繁交易 | ✗ 每次买卖重置税基 |

### 影响二：交易成本考量

```
卖出 $2M 物业 → 买入 $2M 物业

新税负：$20,000/年
旧税负（假设持有20年）：$6,000/年
增加税负：$14,000/年

问题：新物业需要产生足够额外收益才值得换
```

### 影响三：1031 交换更有价值

**1031 交换**可以延迟资本利得税，但**不能保留 Prop 13 税基**。

```
物业 A → 1031 → 物业 B

资本利得税：延迟 ✓
Prop 13 税基：重置 ✗

所以 1031 后税负会上升
```

### 影响四：继承规划重要性

**Prop 19 之前**：
- 子女可继承父母税基
- 代代传承，税负极低

**Prop 19 之后**：
- 仅限主居
- 投资物业税基重置
- 需要提前规划

## Prop 13 与估值

### Prop 13 and Valuation

### 对卖家的影响

**低税基 = 隐形成本**

卖家在出售时失去低税基优势：

```
案例：卖家持有30年

当前税负：$5,000/年
买家税负：$20,000/年
每年"损失"：$15,000

资本化价值 = $15,000 / 5% = $300,000

问题：这 $300K 是否反映在售价中？
```

### 对买家的影响

**买家需要考虑"真实"持有成本**：

```
Peninsula 物业比较：

物业 A（新买）：
价格：$2,000,000
税负：$20,000/年

物业 B（假设能买到低税基）：
价格：$2,100,000（溢价 5%）
税负：$8,000/年（假设转移税基）

10年总税负差：$120,000
净节省：$120,000 - $100,000 = $20,000 ✓
```

## Prop 13 例外情况

### Exceptions to Prop 13

### 何时重新评估

| 情况 | 是否重新评估 |
|-----|-------------|
| 出售 | ✓ 重新评估 |
| 赠与 | ✓ 重新评估（大部分情况） |
| 继承 | △ 看 Prop 19 规则 |
| 新建/加建 | 仅新建部分评估 |
| 再融资 | ✗ 不重新评估 |
| 加名字（配偶） | ✗ 不重新评估 |
| LLC 转移 | △ 看具体情况 |

### 新建/加建的评估规则

**ADU 加建案例**：

```
主屋税基：$500,000
ADU 成本：$400,000

新总税基 = $500,000 + $400,000 = $900,000
年税增加 = $400,000 × 1% = $4,000
```

**注意**：ADU 增加税负，但通常远低于租金收入。

## Prop 13 投资策略

### Investment Strategies with Prop 13

### 策略一：Buy and Hold Forever

```
核心逻辑：
买入 → 持有 → 租金覆盖成本 → 税基锁定 → 长期增值

优势：
- 税负随时间相对下降
- 无交易成本
- 简单

适合：
- 长期投资者
- 不需要流动性者
- 有继承计划者
```

### 策略二：ADU 增值

```
核心逻辑：
现有物业 → 加建 ADU → 租金增加 → 仅 ADU 部分新增税负

数学：
ADU 税增加：$4,000/年
ADU 租金：$30,000/年
净收益：$26,000/年

优势：
- 不触发主屋税基重置
- 杠杆现有资产
```

### 策略三：1031 连续升级

```
核心逻辑：
物业 A → 1031 → 物业 B → 1031 → 物业 C → ...

权衡：
+ 延迟资本利得税
+ 扩大投资规模
- 每次税基重置

适合：
- 追求规模增长
- 长期持有最终物业
```

### 策略四：Step-Up Basis 终极策略

```
核心逻辑：
持有至去世 → 继承人获得 Step-Up Basis

效果：
- 所有递延资本利得消失
- Prop 13 税基重置（但按去世时市值）
- 继承人可选择卖出或继续持有

适合：
- 超长期持有
- 有明确继承计划
```

## Prop 13 常见误区

### Common Misconceptions

| 误区 | 真相 |
|-----|-----|
| "Prop 13 限制房价上涨" | ✗ 只限制税负，不限制价格 |
| "加名字会触发重新评估" | △ 配偶不会，其他情况看具体 |
| "LLC 持有可避免重估" | △ 大部分情况不能 |
| "1031 保留税基" | ✗ 1031 只延迟资本利得税 |
| "Prop 13 即将被废除" | △ 政治上极难，但要关注 |

## Prop 13 检查表

### Prop 13 Checklist

**买入时**：
- ☐ 了解当前物业税基
- ☐ 计算买入后新税负
- ☐ 将税负纳入现金流分析

**持有时**：
- ☐ 跟踪年度 2% 增长
- ☐ ADU 加建的税务影响
- ☐ 考虑长期持有优势

**出售/转移时**：
- ☐ 评估放弃低税基的成本
- ☐ 考虑 1031 交换
- ☐ 规划继承策略
