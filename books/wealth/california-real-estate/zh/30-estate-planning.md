房产是遗产中最复杂的资产之一。没有规划的房产传承可能面临高额税务和法律纠纷。本章教你如何提前规划。

## 遗产规划基础

### Estate Planning Basics

### 为什么重要？

```
没有规划的后果：

1. Probate（遗产认证）
   - 公开程序
   - 耗时 1-3 年
   - 费用 4-7% 资产价值

2. 税务
   - 联邦遗产税（>$13.61M/人）
   - 加州没有遗产税但有其他考虑
   - Prop 19 限制税基继承

3. 家庭纠纷
   - 没有明确指示
   - 多继承人争议
   - 分割困难
```

### 核心工具

| 工具 | 功能 | 适用 |
|-----|-----|-----|
| **遗嘱 Will** | 指定分配 | 基础，但需 Probate |
| **Living Trust** | 避免 Probate | 房产必备 |
| **POA** | 生前授权 | 失能时管理 |
| **DPOA for Healthcare** | 医疗决策 | 医疗紧急 |
| **LLC** | 资产保护 | 投资物业 |
| **Family LP** | 传承+保护 | 多物业 |

## Living Trust

### 可撤销生前信托

### 基本结构

```
Living Trust 结构：

[你（设立人 Grantor）]
      ↓ 转入资产
[信托 Trust]
      ↓ 管理
[你（受托人 Trustee）] → 生前自己管理
      ↓ 去世后
[继任受托人 Successor Trustee] → 配偶/子女
      ↓ 分配
[受益人 Beneficiaries] → 配偶、子女等
```

### 优势

| 优势 | 说明 |
|-----|-----|
| **避免 Probate** | 直接转移，无需法院 |
| **隐私** | 不公开记录 |
| **控制** | 详细的分配指示 |
| **失能保护** | 继任受托人可接管 |
| **灵活** | 随时可修改 |

### 设立步骤

```
Step 1: 咨询律师
- 讨论目标
- 选择结构
- 费用：$2,000-5,000

Step 2: 起草信托文件
- 信托协议
- Pour-Over Will（配套遗嘱）
- POA 文件

Step 3: Funding（注资）
- 将资产转入信托
- 房产需要 Deed 转移
- 银行账户改名

Step 4: 更新受益人
- 退休账户
- 保险单
- 其他账户

Step 5: 保管文件
- 原件安全保管
- 副本给受托人
```

## 房产转入信托

### Transferring Property to Trust

### 流程

```
将房产转入 Living Trust：

Step 1: 准备 Grant Deed
- 从"你个人"到"你作为 XX Trust 的受托人"

Step 2: 公证签名
- Notary Public

Step 3: 记录 Deed
- County Recorder's Office
- 记录费约 $50-100

Step 4: 通知相关方
- 贷款公司（可能需要）
- 保险公司（更新被保险人）
- HOA（如适用）

重要：
- 这不触发 Prop 13 重估
- 不触发 Due on Sale（通常）
- 房贷可以保留
```

### 常见问题

| 问题 | 答案 |
|-----|-----|
| **贷款会被要求全额偿还吗？** | 通常不会（Garn-St. Germain Act 保护） |
| **会触发重估吗？** | 不会（同一人/配偶的信托） |
| **保险需要更新吗？** | 是，通知保险公司 |
| **可以再融资吗？** | 可以，可能需要暂时转出 |

## Step-Up in Basis

### 税基提升

### 重要概念

```
Step-Up in Basis：
继承时资产基础调整为市场价值

案例：
父母购买价：$200,000
父母去世时市价：$1,500,000
子女继承基础：$1,500,000

如果子女立即卖出：
售价 $1,500,000 - 基础 $1,500,000 = $0 资本利得
免税！

如果父母生前赠与：
子女基础 = 父母原基础 $200,000
子女卖出利得：$1,500,000 - $200,000 = $1,300,000
税务巨大！

结论：
继承 >> 赠与（税务角度）
```

### Prop 19 影响

```
Prop 19（2021年2月生效）限制了税基继承：

之前（Prop 58/193）：
子女继承 = 保留父母低税基
无论是否自住

现在（Prop 19）：
主居：
- 子女必须 1 年内自住
- 申请 Homeowner's Exemption
- 超过 $1M 差额要重估

投资物业：
- 100% 重估
- 无例外

策略：
- 如果子女要自住 → 仍可部分保留
- 如果是投资 → 考虑其他规划
```

## 多继承人规划

### Planning for Multiple Heirs

### 挑战

```
多子女继承房产的问题：

1. 分割困难
   - 房产不能物理分割
   - 共同持有复杂

2. 意见分歧
   - 一个想卖，一个想留
   - 管理决策分歧

3. 财务不平衡
   - 一个需要钱，一个不需要
   - 贡献不同

4. 远程管理
   - 有人在外地
   - 管理责任不均
```

### 解决方案

| 方案 | 说明 | 适用 |
|-----|-----|-----|
| **指定单一继承人** | 一人继承房产，其他人得其他资产 | 有足够其他资产 |
| **出售后分配** | 信托要求出售，分配现金 | 继承人都想要现金 |
| **买断条款** | 允许一人按公平价买断 | 一人想保留 |
| **分别物业** | 不同物业给不同人 | 有多套物业 |

### 信托语言示例

```
分配条款选项：

选项 A：单一继承人
"物业 123 Main St 归子女 A 所有；
 等值资产归子女 B"

选项 B：出售分配
"受托人应在我去世后 12 个月内
 出售物业并平均分配给受益人"

选项 C：买断权
"子女 A 有优先购买权，
 按独立评估价买断其他受益人份额，
 90 天内行使"
```

## LLC 与遗产规划

### LLC in Estate Planning

### 整合策略

```
结合 LLC 和 Trust：

结构：
[Living Trust]
    ↓ (持有)
[LLC Membership Interest]
    ↓ (LLC 持有)
[物业]

优势：
1. LLC 提供资产保护
2. Trust 避免 Probate
3. 可以转让 LLC 份额（不是物业）
4. 便于部分赠与

转让方式：
- 转让 LLC 份额给信托
- 不是转让物业
- 可能更简单
```

### 渐进式转让

```
使用 LLC 份额进行渐进式转让：

目的：利用年度赠与免税额

2024年年度免税额：$18,000/人/年

策略：
每年赠与一部分 LLC 份额
价值 ≤ $18,000/受赠人

案例：
物业价值：$1,000,000 in LLC
你和配偶给两个孩子
每年可赠与：$18,000 × 2（父母）× 2（子女）= $72,000
= 7.2% LLC 份额/年

14年后：
100% 份额转让完成
零赠与税

注意：
- 需要评估折扣（Valuation Discount）
- 需要专业评估
- 考虑控制权问题
```

## 联邦遗产税

### Federal Estate Tax

### 2024年规则

```
联邦遗产税免税额：$13.61M/人

超过部分：40% 税率

夫妻：
可"携带"未使用免税额
= 最高 $27.22M 免税

2025年后：
目前免税额是《减税与就业法》临时提高
2026年可能降回 ~$7M/人

规划意义：
- 大多数人目前不需担心
- 但高净值应提前规划
- 免税额可能变化
```

### 减少遗产的策略

| 策略 | 说明 |
|-----|-----|
| **年度赠与** | $18,000/人/年免税 |
| **教育/医疗** | 直接支付无限制 |
| **慈善** | 降低遗产 |
| **GRAT** | 冻结资产增值 |
| **ILIT** | 人寿保险信托 |
| **Family LP** | 估值折扣 |

## 实际案例

### Case Studies

### 案例一：简单传承

```
情况：
- 一套自住房 $1.5M
- 一个孩子
- 希望避免 Probate

方案：
1. 建立 Living Trust
2. 转入房产
3. 孩子作为受益人

结果：
- 去世后孩子直接继承
- 无 Probate
- Step-Up Basis（$1.5M）
- 孩子可自住保留税基（Prop 19）

成本：$3,000 律师费
```

### 案例二：多物业多继承人

```
情况：
- 自住房 $2M
- 投资房 A $1.2M
- 投资房 B $1.2M
- 三个孩子

方案：
1. Living Trust + 3 个 LLC
2. 每套物业一个 LLC
3. 信托持有所有 LLC 份额
4. 分配指示：
   - 自住房 → 孩子 A（想住）
   - 投资房 A → 孩子 B
   - 投资房 B → 孩子 C
5. 任何差额用现金平衡

注意：
- 投资房会重估（Prop 19）
- 孩子 A 需自住保留税基
```

## 本章总结

### Chapter Summary

遗产规划是长期投资的关键：

| 要点 | 内容 |
|-----|-----|
| **Living Trust** | 避免 Probate，必须有 |
| **Step-Up Basis** | 继承比赠与更有利 |

**核心工具**：

| 工具 | 功能 |
|-----|-----|
| **Living Trust** | 避免 Probate |
| **LLC** | 资产保护 |
| **年度赠与** | 渐进转让 |

**核心原则**：

> "胜兵先胜而后求战。"

遗产规划就是"先胜"——在需要之前就做好安排。等到需要时再规划往往太晚。
