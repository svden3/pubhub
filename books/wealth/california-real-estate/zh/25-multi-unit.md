## 多单元类型

### Types of Multi-Unit Properties

### 按规模分类

| 类型 | 单元数 | 特点 | 融资 |
|-----|-------|-----|-----|
| **Duplex** | 2 | 住宅贷款 | Fannie/Freddie |
| **Triplex** | 3 | 住宅贷款 | Fannie/Freddie |
| **Fourplex** | 4 | 住宅贷款 | Fannie/Freddie |
| **小型多户** | 5-20 | 商业贷款 | 本地银行 |
| **中型公寓** | 21-50 | 商业贷款 | 机构贷款 |
| **大型公寓** | 50+ | 商业贷款 | CMBS, 机构 |

### 1-4 单元优势

```
为什么从 2-4 单元开始？

融资优势：
- 可用住宅贷款（3-5% 首付可能）
- 利率比商业低
- 30年固定利率
- Fannie/Freddie 支持

管理优势：
- 规模可控
- 可以自管
- 学习曲线平缓

现金流优势：
- 多个租金来源
- 一个空置不是灾难
- 通常比单户现金流好
```

## Duplex/Triplex/Fourplex 投资

### Small Multi-Unit Investment

### 收益分析

```
案例：Fremont Fourplex

购买价：$2,400,000
首付 (25%)：$600,000
贷款：$1,800,000 @ 6.5%

租金收入：
单元 A：$2,800/月
单元 B：$2,600/月
单元 C：$2,600/月
单元 D：$2,500/月
总租金：$10,500/月 = $126,000/年

费用：
房产税：$24,000
保险：$6,000
维修 (5%)：$6,300
空置 (5%)：$6,300
管理 (8%)：$10,080
总费用：$52,680

NOI：$126,000 - $52,680 = $73,320
Cap Rate：$73,320 / $2,400,000 = 3.1%

贷款还款：$136,600/年
现金流：$73,320 - $136,600 = -$63,280/年

现金流（月）：-$5,273
```

### House Hacking 策略

```
House Hacking with Fourplex：

策略：
买 Fourplex，自住一个单元，出租三个

财务分析：
购买价：$2,400,000
首付 (5% FHA)：$120,000
贷款：$2,280,000 @ 6.5%
月供：$14,410

租金收入（3单元）：
$2,600 × 3 = $7,800/月

净住房成本：
$14,410 - $7,800 = $6,610/月

对比：
同区域单户租金：$4,500/月
自己住 Fourplex 成本：$6,610/月
但你在积累权益

5年后：
- 权益增长：~$300,000
- 租金增长：额外 $500+/月
- 可以搬出，全部出租
```

## 5+ 单元商业物业

### Commercial Multi-Family

### 商业贷款区别

| 因素 | 住宅贷款（1-4） | 商业贷款（5+） |
|-----|---------------|---------------|
| **首付** | 3-25% | 25-35% |
| **利率** | 住宅利率 | 商业利率（通常更高） |
| **期限** | 30年固定 | 5-10年球型 |
| **审批** | 基于借款人 | 基于物业 NOI |
| **估值** | Comparable Sales | Income Approach |

### NOI 驱动估值

```
商业物业估值公式：
价值 = NOI / Cap Rate

案例：10 单元公寓

租金收入：$20,000/月 = $240,000/年
费用比例：45%
NOI：$240,000 × 55% = $132,000

市场 Cap Rate：5%
物业价值：$132,000 / 5% = $2,640,000

提高价值的方法：
1. 增加租金 → NOI 上升
2. 降低费用 → NOI 上升
3. 等待 Cap Rate 下降

NOI 增加 $10,000：
新价值：$142,000 / 5% = $2,840,000
增值：$200,000
```

## 多单元开发策略

### Development Strategies

### 策略一：单户→多单元

```
利用 SB9 和 ADU 政策：

原始：单户住宅
  ↓
Step 1: 加建 ADU
  = 2 单元
  ↓
Step 2: SB9 分割（如适用）
  + 在新地块建 Duplex
  = 4 单元

或者：
原始：单户
  ↓
加建 ADU + JADU
  = 3 单元
```

### 策略二：收购小多户

```
目标：2-4 单元物业

寻找标准：
- 低于市价（motivated seller）
- 租金低于市场
- 有增值空间（ADU 可能）
- 好位置

增值策略：
1. 提高租金到市场水平
2. 降低空置率
3. 减少运营成本
4. 加建 ADU
5. 提升物业状况
```

### 策略三：土地开发

```
新建多单元：

流程：
1. 购买土地或 teardown
2. 规划多单元开发
3. 获得许可
4. 施工
5. 出租或出售

挑战：
- 资金需求大
- 时间长（2-3年）
- 风险高
- 需要专业团队

适合：
- 有经验的开发者
- 资金充足
- 能承受风险
```

## 多单元管理

### Multi-Unit Management

### 自管 vs PM

| 因素 | 自管 | 物业管理公司 |
|-----|-----|-------------|
| **成本** | 免费（时间成本） | 8-10% 租金 |
| **时间** | 需要大量时间 | 省时间 |
| **专业性** | 学习曲线 | 专业团队 |
| **适合** | 1-4 单元，本地 | 5+ 单元，远程 |

### PM 选择标准

```
选择物业管理公司：

必要条件：
- 有执照（California DRE）
- 有保险
- 管理同类物业经验
- 本地presence

评估指标：
- 收费结构透明
- 空置率记录
- 租户筛选流程
- 维修处理效率
- 财务报告质量

面试问题：
- 目前管理多少单元？
- 平均空置率？
- 如何筛选租户？
- 维修如何处理？
- 多久提供财务报告？
- 解约条款？
```

### 运营效率

```
多单元运营效率提升：

1. 标准化
   - 统一租约
   - 统一装修标准
   - 统一流程

2. 规模效应
   - 批量维修采购
   - 共享维护人员
   - 保险折扣

3. 租户管理
   - 在线收租
   - 在线报修
   - 自动提醒

4. 财务管理
   - 物业软件
   - 分单元核算
   - 预算管理
```

## 多单元融资

### Multi-Unit Financing

### 1-4 单元融资选项

| 产品 | 首付 | 特点 | 适合 |
|-----|-----|-----|-----|
| **FHA** | 3.5% | 必须自住 | House Hacking |
| **Conventional** | 15-25% | 可投资 | 投资者 |
| **VA** | 0% | 退伍军人 | 符合条件者 |
| **Portfolio** | 20-25% | 灵活标准 | 复杂情况 |

### 5+ 单元融资选项

| 产品 | 首付 | 特点 |
|-----|-----|-----|
| **本地银行** | 25-30% | 关系型，灵活 |
| **信用社** | 25-30% | 利率可能好 |
| **Fannie/Freddie** | 25-30% | 5-50 单元，好利率 |
| **CMBS** | 25-35% | 大型物业 |
| **Private** | 30%+ | 快速，灵活，贵 |

### DSCR 要求

```
Debt Service Coverage Ratio（债务覆盖率）

计算：DSCR = NOI / 年债务还款

商业贷款要求：通常 DSCR > 1.20-1.25

案例：
NOI：$132,000
要求 DSCR：1.25
最大债务还款：$132,000 / 1.25 = $105,600/年

利率 6%，30年：
最大贷款：约 $1,450,000

如果购买价 $2,000,000：
需要首付：$550,000 (27.5%)
```

## 风险管理

### Risk Management

### 主要风险

| 风险 | 描述 | 缓解 |
|-----|-----|-----|
| **空置** | 多单元空置影响大 | 位置、定价、管理 |
| **租控** | 加州租控严格 | 选择豁免物业 |
| **维修** | 老旧物业维修多 | 检查、储备金 |
| **管理** | 管理复杂度增加 | 专业 PM |
| **利率** | 商业贷款会调整 | 固定期限选择 |

### 储备金要求

```
多单元储备金建议：

运营储备：
- 3-6 个月运营费用
- 用于空置、意外维修

资本储备：
- 每单元 $3,000-5,000/年
- 用于大项维修（屋顶、电器、地毯）

案例：10 单元公寓
运营费：$108,000/年
运营储备：$54,000（6个月）
资本储备：$40,000/年

总储备目标：$100,000+
```

## 退出策略

### Exit Strategies

### 选项比较

| 策略 | 优势 | 劣势 |
|-----|-----|-----|
| **长期持有** | 现金流、增值 | 流动性低 |
| **1031 交换** | 延税升级 | 找替代物业 |
| **出售** | 变现 | 资本利得税 |
| **Condo 转换** | 最大化价值 | 复杂、限制多 |
| **再融资** | 取出权益、继续持有 | 增加债务 |

### Condo 转换

```
Condo Conversion（公寓转为 Condo 出售）：

优势：
- 每单元单独出售
- 通常比整体出售价值高
- 买家可获得住宅贷款

挑战：
- 加州限制很多
- 需要租户同意或搬迁
- 审批复杂
- 成本高

适用情况：
- 高端区域
- 每单元价值高
- 租控不严格
- 有转换经验
```

## 本章总结

### Chapter Summary

多单元投资是规模化的关键：

| 要点 | 内容 |
|-----|-----|
| **类型选择** | 1-4 单元住宅贷款优势大 |
| **融资** | 住宅 vs 商业贷款区别 |
| **管理** | 5+ 单元考虑专业 PM |
| **增值** | NOI 驱动商业物业价值 |

**策略路径**：

| 阶段 | 策略 |
|-----|-----|
| **入门** | House Hacking 2-4 单元 |
| **积累** | 收购增值小多户 |
| **规模化** | 进入 5+ 单元商业物业 |

**核心原则**：

> "全国为上，破国次之。"

保全资本比激进扩张更重要。从小规模开始，积累经验，逐步扩大——这是多单元投资的正道。
