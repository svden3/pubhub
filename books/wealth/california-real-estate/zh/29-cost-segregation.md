## 成本分离基础

### Cost Segregation Basics

### 什么是成本分离？

```
标准折旧：
住宅物业 = 27.5 年直线折旧
商业物业 = 39 年直线折旧

成本分离：
将物业成本分解成不同类别
每个类别有不同折旧年限

类别包括：
- 5年财产（地毯、电器）
- 7年财产（办公家具）
- 15年财产（停车场、景观）
- 27.5/39年财产（建筑结构）
```

### 折旧年限

| 类别 | 年限 | 示例 |
|-----|-----|-----|
| **5年** | 5 | 地毯、电器、窗帘、某些电气 |
| **7年** | 7 | 办公家具、固定装置 |
| **15年** | 15 | 景观、停车场、某些外部改善 |
| **土地改良** | 15 | 人行道、围栏 |
| **建筑** | 27.5/39 | 结构、屋顶、墙体 |

## 税务效益

### Tax Benefits

### 加速折旧效果

```
案例：$2,000,000 物业（不含土地）

传统折旧（27.5年）：
年折旧：$2,000,000 / 27.5 = $72,727

成本分离后：
5年财产 (15%)：$300,000 / 5 = $60,000/年
7年财产 (5%)：$100,000 / 7 = $14,286/年
15年财产 (10%)：$200,000 / 15 = $13,333/年
建筑 (70%)：$1,400,000 / 27.5 = $50,909/年

第一年总折旧：$138,528
对比传统：$72,727

增加折旧：$65,801
税务节省（40%税率）：$26,320
```

### Bonus Depreciation（额外折旧）

```
2024年 Bonus Depreciation：60%

可以对 5、7、15 年财产立即折旧 60%

案例继续：
5年财产 $300,000 × 60% = $180,000（第一年）
7年财产 $100,000 × 60% = $60,000（第一年）
15年财产 $200,000 × 60% = $120,000（第一年）

第一年额外折旧：$360,000
加上建筑折旧：$50,909
第一年总折旧：$410,909

税务节省（40%）：$164,364

注意：
Bonus Depreciation 逐年递减：
2023：80%
2024：60%
2025：40%
2026：20%
2027+：0%
```

## 成本分离研究

### Cost Segregation Study

### 什么是 CSS？

```
Cost Segregation Study (CSS)：
专业工程师对物业进行分析
将物业成本分配到不同折旧类别

研究内容：
1. 现场检查
2. 建造成本分析
3. 工程评估
4. 税务分类
5. 详细报告
```

### 何时做 CSS

| 情况 | 建议 |
|-----|-----|
| **新购买/新建** | 最佳时机 |
| **重大翻新后** | 可以做 |
| **已持有物业** | 可以追溯（Look-back） |
| **小物业（<$500K）** | 可能不值得 |

### CSS 成本

| 物业类型 | 典型成本 |
|---------|---------|
| **住宅（单户）** | $3,000-5,000 |
| **多户（2-4单元）** | $4,000-7,000 |
| **公寓（5-20单元）** | $6,000-12,000 |
| **商业物业** | $8,000-20,000+ |

### ROI 分析

```
是否值得做 CSS？

简单测试：
物业价值（不含土地）× 5% = 潜在首年额外折旧
额外折旧 × 税率 = 首年税务节省
税务节省 > CSS 成本的 3-5 倍？

案例：
物业价值：$1,000,000
潜在额外折旧：$50,000
税务节省（40%）：$20,000
CSS 成本：$5,000
ROI：4x ✓

经验法则：
物业价值 > $750,000（不含土地）通常值得
```

## Look-Back Study

### 追溯研究

### 如何追溯

```
已经持有物业几年了？
可以用 Look-Back Study 追溯

方法：
1. 做 CSS
2. 计算之前年份"应该"有的折旧
3. 在当前年份一次性调整（Form 3115）
4. 无需修改之前的税表

案例：
2020年购买物业 $1,500,000
2020-2023 用传统折旧
2024年做 CSS

2020-2023 实际折旧：$218,182（$54,545/年）
2020-2023 应有折旧：$450,000（CSS 方法）
差额：$231,818

2024年 Form 3115：
一次性减少应税收入 $231,818
税务节省（40%）：$92,727
```

### 注意事项

```
Look-Back 考虑：

1. 时机
   - 收入高的年份做更有价值
   - 预计卖出前不建议

2. 未来出售影响
   - 更多折旧 = 更多 Depreciation Recapture
   - 如果做 1031 可以延迟

3. 州税影响
   - 加州不完全跟随联邦规则
   - 可能有差异
```

## 折旧回收

### Depreciation Recapture

### 出售时的影响

```
折旧回收 (Depreciation Recapture)：
出售时，之前的折旧需要"回收"
按 25% 税率（联邦）征税

案例：
购买价：$1,000,000
折旧总额：$300,000
出售价：$1,500,000

计算：
调整后基础：$1,000,000 - $300,000 = $700,000
总利润：$1,500,000 - $700,000 = $800,000

利润分解：
折旧回收：$300,000 × 25% = $75,000
资本利得：$500,000 × 20% = $100,000
总税：$175,000

注意：
加速折旧不会增加总税
只是改变税的时机
（现在少付，将来付 + 时间价值）
```

### 1031 Exchange 解决方案

```
如何避免折旧回收：

使用 1031 Exchange：
- 延迟所有税（包括折旧回收）
- 换到新物业继续折旧
- 可以再做 CSS

最佳策略：
购买 → CSS → 加速折旧 → 1031 Exchange → 新物业 CSS → 重复
= 持续延迟税务 + 时间价值
```

## 实际操作

### Implementation

### 选择 CSS 供应商

```
评估标准：

1. 专业资质
   - 工程师团队
   - 税务专家
   - CPA 合作

2. 经验
   - 同类物业经验
   - 案例研究

3. 报告质量
   - 详细文档
   - IRS 审计支持

4. 价格
   - 固定价格 vs 按节省比例
   - 包括什么服务

供应商类型：
- 专业 CSS 公司
- CPA 事务所（带工程团队）
- 房产专业服务公司
```

## 加州特别考虑

### California Considerations

### 加州 vs 联邦差异

```
加州不完全跟随联邦折旧规则：

1. Bonus Depreciation
   - 加州不允许
   - 需要加回调整

2. Section 179
   - 加州有自己的限制
   - 低于联邦

3. 折旧方法
   - 加州用自己的折旧表
   - 可能有差异

影响：
联邦税务节省：$100,000
加州可能不同：$70,000-80,000
仍然有价值，但需分开计算
```

## 与其他策略结合

### Combining Strategies

### 综合策略

```
最大化税务效益：

1. 购买物业
2. 做 Cost Segregation Study
3. 利用 Bonus Depreciation（如果还有）
4. 创造 Paper Loss 抵消其他收入
5. 持有几年
6. 1031 Exchange 到更大物业
7. 新物业再做 CSS
8. 重复

结果：
- 持续的税务延迟
- 时间价值收益
- 规模增长
```

```
如果是 Real Estate Professional：

条件：
- 750+ 小时/年在房产活动
- 超过一半工作时间在房产

效果：
高收入者 + REP + CSS = 显著税务节省
```

## 本章总结

### Chapter Summary

成本分离是强大的税务工具：

| 要点 | 内容 |
|-----|-----|
| **原理** | 将物业分解为不同折旧类别 |
| **效果** | 加速折旧，延迟税务 |
| **时机** | 购买时或追溯 |
| **成本** | $3,000-20,000 |
| **门槛** | 物业价值 > $750K 通常值得 |

**最佳实践**：

| 实践 | 说明 |
|-----|-----|
| **购买时做** | 最大化效益 |
| **结合 1031** | 延迟折旧回收 |
| **高收入年份** | 税务节省更大 |
| **追溯可行** | Look-back study |

**核心原则**：

> "故善用兵者，屈人之兵而非战也。"

成本分离是"不战而胜"——不改变实际支出，却能显著减少当前税负。聪明的投资者善用这些工具。
