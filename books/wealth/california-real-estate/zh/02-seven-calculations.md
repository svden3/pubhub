# 第二章：七计 — 项目评估法

## Chapter 2: Seven Calculations — Project Evaluation Method

---

> "主孰有道？将孰有能？天地孰得？法令孰行？兵众孰强？士卒孰练？赏罚孰明？吾以此知胜负矣。"
>
> "Which of the two sovereigns is imbued with the Moral Law? Which general is more capable? With whom lie the advantages of Heaven and Earth? On which side is discipline more rigorously enforced?"
>
> — 《孙子兵法·始计篇》

---

孙子用"七计"来比较敌我双方，预测战争胜负。在地产投资中，我们也需要一套量化的评估框架。

---

## 地产投资的七计

### The Seven Calculations for Real Estate

| 七计 | 原意 | 地产含义 | 评估指标 |
|-----|-----|---------|---------|
| **主孰有道** | 君主是否得民心 | 政策环境 | 城市发展规划、ADU友好度 |
| **将孰有能** | 将帅能力 | 团队能力 | GC经验、Agent专业度 |
| **天地孰得** | 天时地利 | 周期与区位 | 利率周期、学区质量 |
| **法令孰行** | 纪律执行 | 投资纪律 | 安全边际、止损规则 |
| **兵众孰强** | 军队实力 | 资金实力 | 可投资资金、融资能力 |
| **士卒孰练** | 士兵训练 | 投资经验 | 过往项目、市场认知 |
| **赏罚孰明** | 激励机制 | 收益风险比 | Cap Rate、现金流 |

## 一、政策环境 (主孰有道)

### 评估指标

| 指标 | 评估方法 | 数据来源 |
|-----|---------|---------|
| **城市总体规划** | 未来发展方向 | General Plan |
| **ADU 审批效率** | 平均审批时间 | 城市规划部门 |
| **开发费用** | Impact fees, permit fees | 城市费率表 |
| **租控政策** | AB 1482 + 本地条例 | 城市法规 |
| **房东友好度** | 驱逐流程、租户保护 | 本地律师 |

### 评分标准

| 分数 | 政策环境 |
|-----|---------|
| 5 | 非常支持开发，费用低，无本地租控 |
| 4 | 支持开发，费用合理，仅 AB 1482 |
| 3 | 中性，正常审批流程 |
| 2 | 限制较多，审批慢 |
| 1 | 强租控，高费用，审批困难 |

## 二、团队能力 (将孰有能)

### 核心团队评估

| 角色 | 评估维度 | 关键问题 |
|-----|---------|---------|
| **Agent** | 专业度、网络 | 有多少 off-market 机会？ |
| **GC** | 可靠性、成本 | 过往 ADU 项目表现？ |
| **Lender** | 灵活性、速度 | 能否快速预审？ |
| **CPA** | 地产专业度 | 了解 1031 吗？ |
| **律师** | 响应速度 | 能处理租约纠纷吗？ |

### 评分标准

| 分数 | 团队状态 |
|-----|---------|
| 5 | 所有角色到位，合作多年 |
| 4 | 核心角色到位，合作过 |
| 3 | 有推荐人选，需要建立关系 |
| 2 | 需要从头寻找 |
| 1 | 完全没有资源 |

## 七、收益风险比 (赏罚孰明)

### 收益指标

| 指标 | 计算 | 目标 |
|-----|-----|-----|
| **Cap Rate** | NOI / 价格 | > 4% |
| **Cash-on-Cash** | 年现金流 / 投入 | > 6% |
| **IRR** | 内部收益率 | > 12% |
| **Equity Multiple** | 总回报 / 投入 | > 1.5x |

### 风险指标

| 指标 | 计算 | 警戒线 |
|-----|-----|-------|
| **LTV** | 贷款 / 价值 | < 75% |
| **DSCR** | NOI / 还贷 | > 1.25 |
| **Break-even** | 空置率 | < 20% |
| **Payback** | 回本期 | < 7年 |

### 评分标准

| 分数 | 收益风险比 |
|-----|----------|
| 5 | 高收益 + 低风险 |
| 4 | 中等收益 + 低风险 |
| 3 | 中等收益 + 中等风险 |
| 2 | 低收益 或 高风险 |
| 1 | 低收益 + 高风险 |

## 案例：七计实战分析

### Case Study: Seven Calculations in Action

**物业信息**：
- 地址：Mountain View, CA
- 类型：单户住宅 + ADU 潜力
- 要价：$2,200,000
- 租金潜力：主屋 $4,500 + ADU $2,800

**七计评估**：

| 计 | 评估 | 分数 |
|---|-----|-----|
| 政策 | ADU 友好，审批 60 天 | 4 |
| 团队 | 有熟悉 GC，Agent 一般 | 3 |
| 周期区位 | 利率高但学区 9/10 | 4 |
| 纪律 | 安全边际 18%，略低 | 3 |
| 资金 | 25% 首付 + 6个月储备 | 4 |
| 经验 | 第二个 ADU 项目 | 3 |
| 收益 | Cap 4.5%, CoC 7% | 4 |

**加权总分**：3.65

**决策**：**黄灯** — 需要：
1. 谈判降价以提高安全边际
2. 更详细的 ADU 成本估算
3. 确认 GC 可用性

## 本章总结

### Chapter Summary

"七计"提供了一套**系统化**的项目评估框架：

| 计 | 核心问题 |
|---|---------|
| 一 | 政策支持吗？ |
| 二 | 团队靠谱吗？ |
| 三 | 时机和位置对吗？ |
| 四 | 我遵守纪律吗？ |
| 五 | 资金够吗？ |
| 六 | 我有经验吗？ |
| 七 | 收益配得上风险吗？ |

**核心原则**：

> "多算胜，少算不胜，而况于无算乎？"

每笔投资前，都要做"七计"评估。
