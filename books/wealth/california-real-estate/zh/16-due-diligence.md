"知彼"在地产投资中就是**尽职调查**。本章提供一个完整的尽调流程，确保你在成交前充分了解物业。

---

## 尽调时间线

### Due Diligence Timeline

```
Day 0: Offer 被接受
       ↓
Day 1-7: 紧急检查
       ├── 安排 Home Inspection
       ├── 安排 Pest Inspection
       └── 获取 Preliminary Title
       ↓
Day 7-14: 深度检查
       ├── 审查检查报告
       ├── 要求修复或信用
       ├── 专项检查（如需）
       └── 财务验证
       ↓
Day 14-17: 最终决策
       ├── 所有问题解决？
       ├── 移除/保留 Contingency
       └── 决定继续或退出
       ↓
Day 17+: 完成交易
       └── 等待 Close of Escrow
```

## 第一阶段：物理检查

### Phase 1: Physical Inspection

### 1.1 Home Inspection

**检查范围**：

| 系统 | 检查内容 |
|-----|---------|
| **结构** | 地基、框架、墙体 |
| **屋顶** | 材料、状况、漏水 |
| **电气** | 电箱、布线、接地 |
| **水暖** | 管道、热水器、漏水 |
| **HVAC** | 空调、暖气、通风 |
| **外部** | 排水、walkway、deck |
| **内部** | 墙面、地板、门窗 |
| **安全** | 烟雾报警、CO报警 |

**常见问题及成本**：

| 问题 | 严重性 | 修复成本 |
|-----|-------|---------|
| 地基裂缝 | ⭐⭐⭐⭐⭐ | $5K-$50K+ |
| 屋顶需更换 | ⭐⭐⭐⭐ | $15K-$40K |
| 电箱升级 | ⭐⭐⭐ | $2K-$5K |
| 管道更换 | ⭐⭐⭐⭐ | $5K-$20K |
| HVAC 更换 | ⭐⭐⭐ | $5K-$15K |
| 窗户更换 | ⭐⭐ | $10K-$30K |

### 1.2 Pest/Termite Inspection

**Section 1 vs Section 2**：

| 类型 | 定义 | 处理 |
|-----|-----|-----|
| **Section 1** | 活跃侵害或损坏 | 通常卖家修复 |
| **Section 2** | 有利于侵害的条件 | 可谈判 |

**常见问题**：

| 问题 | 成本 |
|-----|-----|
| 白蚁处理 | $1K-$3K |
| 干腐修复 | $2K-$10K+ |
| 木材更换 | 变化大 |

### 1.3 专项检查

| 检查 | 何时需要 | 成本 |
|-----|---------|-----|
| **Sewer Scope** | 老房子 | $200-400 |
| **Roof** | 屋顶问题 | $300-500 |
| **Foundation** | 裂缝/沉降 | $400-800 |
| **Chimney** | 有壁炉 | $150-300 |
| **Pool** | 有泳池 | $200-400 |

## 第二阶段：产权检查

### Phase 2: Title Review

### 2.1 Preliminary Title Report

**审查要点**：

| 项目 | 检查内容 |
|-----|---------|
| **所有权** | 卖家是否真正拥有？ |
| **Liens** | 有无未披露的债务？ |
| **Easements** | 他人有无使用权？ |
| **Encumbrances** | 有无其他限制？ |
| **Judgments** | 卖家有无判决？ |

**常见问题**：

| 问题 | 影响 | 处理 |
|-----|-----|-----|
| 未付税 | 需清偿才能过户 | 从卖方款项扣除 |
| 机械留置权 | 承包商欠款 | 卖方解决 |
| 判决留置权 | 影响产权 | 卖方解决 |
| Easement | 可能限制使用 | 评估影响 |

### 2.2 产权保险

**两种类型**：

| 类型 | 保护对象 | 谁付费 |
|-----|---------|-------|
| **Owner's Policy** | 买家 | 地区惯例 |
| **Lender's Policy** | 贷款方 | 买家 |

## 第三阶段：法律合规

### Phase 3: Legal Compliance

### 3.1 许可审查

**检查清单**：

- ☐ 所有改建有许可？
- ☐ 许可已完结 (finaled)？
- ☐ 无违章建筑？
- ☐ ADU/JADU 合法？
- ☐ 车库改建合规？

**如何检查**：
1. 向城市 Building Department 索取许可历史
2. 对比物业现状与许可记录
3. 注意任何差异

### 3.2 Zoning 合规

| 检查项 | 问题 |
|-------|-----|
| 用途 | 当前用途是否允许？ |
| Setbacks | 建筑是否符合退线？ |
| 覆盖率 | 是否超过允许覆盖？ |
| 高度 | 是否超过限制？ |
| 停车 | 是否满足要求？ |

### 3.3 HOA 审查（如适用）

**关键文件**：

| 文件 | 内容 |
|-----|-----|
| **CC&Rs** | 使用限制、规则 |
| **Bylaws** | 治理规则 |
| **Budget** | 财务状况 |
| **Reserves** | 储备金情况 |
| **Meeting Minutes** | 最近议题 |

**红旗**：
- ☐ 储备金不足
- ☐ 计划中的特别评估
- ☐ 进行中的诉讼
- ☐ 高欠费率
- ☐ 保险不足

## 第四阶段：财务验证

### Phase 4: Financial Verification

### 4.1 租金验证

| 验证项 | 方法 |
|-------|-----|
| 当前租金 | 索取租约副本 |
| 租金历史 | 银行对账单 |
| 市场租金 | Comp 分析 |
| 租户质量 | 信用/付款历史 |

### 4.2 费用验证

| 费用 | 验证来源 |
|-----|---------|
| 房产税 | County Assessor |
| 保险 | 当前保单/报价 |
| HOA | HOA Budget |
| 公用事业 | 历史账单 |
| 维修 | 维修记录 |

### 4.3 现金流重算

```
验证后的现金流计算：

年租金收入：$......
- 空置 (5%)：$......
= 有效租金：$......

- 房产税：$......
- 保险：$......
- HOA：$......
- 公用事业：$......
- 维修储备 (5%)：$......
- PM (如需)：$......
= NOI：$......

- 债务服务：$......
= 现金流：$......
```

## 第五阶段：物理再访

### Phase 5: Physical Re-Visit

### 最后检查

**成交前 Walk-Through**：

- ☐ 商定的修复已完成？
- ☐ 包含的设备还在？
- ☐ 无新损坏？
- ☐ 房屋已清空？
- ☐ 公用设施正常？

### 多时段访问

| 时间 | 检查内容 |
|-----|---------|
| 工作日早晨 | 通勤噪音 |
| 工作日晚间 | 邻居活动 |
| 周末 | 街区氛围 |
| 雨天 | 排水问题 |

## 尽调决策树

### Due Diligence Decision Tree

```
检查发现问题
      ↓
问题可解决吗？
├── 否 → 退出交易
└── 是 → 解决成本多少？
          ↓
     成本在安全边际内？
     ├── 否 → 要求信用或降价
     │         ↓
     │    卖家同意？
     │    ├── 否 → 退出或接受
     │    └── 是 → 继续
     └── 是 → 继续
                ↓
          移除 Contingency
```

## 尽调红旗总结

### Red Flags Summary

### 致命红旗（考虑退出）

| 红旗 | 原因 |
|-----|-----|
| 重大地基问题 | 成本不可控 |
| 大规模违建 | 法律风险 |
| 产权纠纷 | 无法清晰过户 |
| 环境污染 | 修复成本巨大 |
| 严重结构问题 | 安全风险 |

### 可谈判红旗

| 红旗 | 处理 |
|-----|-----|
| 屋顶需更换 | 要求信用 |
| 电气升级需求 | 要求信用 |
| 小规模违建 | 评估合法化成本 |
| 维修需求 | 计入成本 |

## 本章总结

### Chapter Summary

尽职调查是"知彼"的过程：

| 阶段 | 核心任务 |
|-----|---------|
| **物理检查** | Home, Pest, 专项 |
| **产权检查** | Title, 留置权, Easement |
| **法律合规** | 许可, Zoning, HOA |
| **财务验证** | 租金, 费用, 现金流 |
| **物理再访** | 最后确认 |

**核心原则**：

> "知彼知己，百战不殆。"

尽调不充分就是"不知彼"，失败的风险大增。
