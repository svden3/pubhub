## 九地概述 | Overview of Nine Terrains

孙子将地形分为九种，每种地形有不同的风险特征和应对策略。投资同样如此——不同的市场环境、资产类型、杠杆水平构成不同的"地形"，需要不同的风险控制策略。

### 投资的九种地形

| 地形 | 孙子原意 | 投资映射 | 风险等级 | 策略 |
| ---- | -------- | -------- | -------- | ---- |
| **散地** | 诸侯在自己领地作战 | 熟悉领域、小仓位试探 | 低 | 可以试错 |
| **轻地** | 刚进入敌境不深 | 新领域、轻仓位 | 低-中 | 谨慎前进 |
| **争地** | 双方都想占领的要地 | 热门赛道、竞争激烈 | 中 | 需要优势 |
| **交地** | 四通八达之地 | 流动性好的资产 | 中 | 保持灵活 |
| **衢地** | 多国交界的战略要地 | 受多重因素影响的资产 | 中-高 | 多方结盟 |
| **重地** | 深入敌境 | 重仓位、高 conviction | 高 | 必须有把握 |
| **圮地** | 山林险阻难行 | 流动性差的资产 | 高 | 谨慎进入 |
| **围地** | 进路狭窄、退路迂回 | 杠杆投资、锁定期长 | 极高 | 需有退路 |
| **死地** | 背水一战之地 | All-in、高杠杆、无退路 | 致命 | 避免！ |

---

## 详细解析

### 1. 散地 (Dispersive Ground) — 安全区域

**孙子原意**："诸侯自战其地，为散地。"在自己的领地作战。

**投资映射**：在自己熟悉的领域，用小仓位进行试探性投资。

**特征**：
- 你对这个行业/公司非常了解
- 仓位很小（< 5%）
- 即使全部损失也不影响整体

**策略**："散地则无战" — 在这种地形下，不需要全力以赴，可以用来学习和试错。

**例子**：
- 一个科技从业者买入自己熟悉的科技股，仓位 3%
- 一个地产开发商第一次尝试 ADU 项目

---

### 2. 轻地 (Facile Ground) — 初涉领域

**孙子原意**："入人之地不深者，为轻地。"刚进入敌方领土。

**投资映射**：刚开始接触的新领域，轻仓位试水。

**特征**：
- 领域相对陌生
- 仓位较轻（5-10%）
- 容易撤退

**策略**："轻地则无止" — 不要在这里恋战，如果情况不对，快速撤退。

**例子**：
- 第一次投资加密货币，只投入可承受损失的金额
- 第一次做地产开发，选择小规模项目

---

### 3. 争地 (Contentious Ground) — 竞争激烈

**孙子原意**："我得则利，彼得亦利者，为争地。"双方都想占领的战略要地。

**投资映射**：热门赛道，所有人都想进入，竞争激烈。

**特征**：
- 赛道看起来很好
- 但竞争者众多
- 估值可能已经很高

**策略**："争地则无攻" — 不要硬抢，除非你有明显优势。

**例子**：
- 2021 年的电动车赛道
- 热门学区的地产

---

### 4. 交地 (Open Ground) — 流动性好

**孙子原意**："我可以往，彼可以来者，为交地。"四通八达之地。

**投资映射**：流动性极好的资产，随时可以买卖。

**特征**：
- 高流动性
- 价格透明
- 进出自由

**策略**："交地则无绝" — 保持灵活，不要切断退路。

**例子**：
- 大盘蓝筹股
- 国债
- 主流 ETF

---

### 5. 衢地 (Focal Ground) — 多方博弈

**孙子原意**："诸侯之地三属，先至而得天下之众者，为衢地。"多国交界的战略要地。

**投资映射**：受多重宏观因素影响的资产。

**特征**：
- 受政策、利率、汇率等多重因素影响
- 需要关注的变量多
- 不确定性高

**策略**："衢地则合交" — 需要多方面的分析和对冲。

**例子**：
- 跨国公司股票
- 新兴市场资产
- 大型地产开发（受政策、利率、就业多重影响）

---

### 6. 重地 (Serious Ground) — 深度介入

**孙子原意**："入人之地深，背城邑多者，为重地。"深入敌境。

**投资映射**：重仓位持有，高 conviction bet。

**特征**：
- 仓位较重（20%+）
- 已经深度研究
- 很难快速撤退

**策略**："重地则掠" — 必须有高确定性，并且准备好持久战。

**警示**：进入"重地"之前，必须完成"五事庙算"的全部评估。

**例子**：
- 巴菲特对苹果的持仓（45%+）
- 将大部分资金投入一个开发项目

---

### 7. 圮地 (Difficult Ground) — 流动性陷阱

**孙子原意**："山林、险阻、沮泽，凡难行之道者，为圮地。"山林险阻难行。

**投资映射**：流动性差的资产，难以退出。

**特征**：
- 买容易，卖难
- 定价不透明
- 可能需要大幅折价才能退出

**策略**："圮地则行" — 快速通过，不要逗留太久。

**例子**：
- 小盘股
- 私募股权
- 非核心区域地产
- 收藏品

---

### 8. 围地 (Hemmed-in Ground) — 杠杆陷阱

**孙子原意**："所由入者隘，所从归者迂，彼寡可以击吾之众者，为围地。"进路狭窄，退路迂回。

**投资映射**：杠杆投资、锁定期长、有对手方风险。

**特征**：
- 使用了杠杆
- 有锁定期或惩罚性退出条款
- 依赖对手方（如贷款机构）

**策略**："围地则谋" — 必须有详细的逃生计划。

**例子**：
- 带杠杆的股票投资
- Construction Loan（有到期风险）
- 对冲基金锁定期

---

### 9. 死地 (Desperate Ground) — 致命区域

**孙子原意**："疾战则存，不疾战则亡者，为死地。"背水一战之地。

**投资映射**：All-in、高杠杆、没有退路。

**特征**：
- 全部资金押注
- 高杠杆
- 如果失败，将彻底出局

**策略**："死地则战" — 但更好的策略是**永远不要让自己进入死地**！

**例子**：
- 用借来的钱满仓加杠杆
- 把所有身家押在一个项目上
- Bill Hwang / Archegos 的覆灭

## 自我诊断：你在哪种地形？

### 诊断清单

| 问题 | 是 | 否 |
| ---- | -- | -- |
| 单一持仓超过 25%？ | → 重地 | |
| 使用了 2 倍以上杠杆？ | → 围地 | |
| 持有流动性差的资产超过 30%？ | → 圮地 | |
| 现金储备不足 3 个月支出？ | → 围地/死地 | |
| 最大单笔投资如果归零，能否生存？ | → 判断是否在死地 | |

### 风险等级矩阵

```
                    低杠杆                    高杠杆
                  ┌─────────────────────────────────┐
熟悉领域          │  散地 (安全)    │  围地 (危险)   │
                  ├─────────────────┼────────────────┤
陌生领域          │  轻地 (谨慎)    │  死地 (致命)   │
                  └─────────────────┴────────────────┘
```

## 【华尔街案例】Archegos 的"死地"之亡

### 背景
- Bill Hwang 的家族办公室 Archegos Capital
- 使用 Total Return Swap（TRS）隐藏真实仓位
- 杠杆率估计 5-8 倍
- 集中持仓：ViacomCBS、Discovery、百度等

### 地形分析
- **起点**：重地（集中持仓）
- **加杠杆**：进入围地
- **继续加杠杆 + 集中**：进入死地
- **市场下跌**：Margin Call → 强制平仓 → 归零

### 时间线
- 2021年3月22日：ViacomCBS 增发股票
- 股价开始下跌
- 3月26日：Goldman、Morgan Stanley 开始平仓
- 一周内：$200 亿 → $0

### 教训
> "永远不要让自己进入死地——即使进入，也要有'死地求生'的预案。"

## 【地产案例】2008 加州开发商的"围地"困境

### 背景
- 2006-2007：加州地产泡沫高峰
- 许多开发商使用 Construction Loan 开发项目
- 典型结构：80% LTV，2-3 年期限

### 围地特征
- **进路狭窄**：资金来源单一（银行贷款）
- **退路迂回**：项目未完工无法销售
- **对手方风险**：银行可以拒绝续贷

### 2008 发生了什么
1. 市场崩盘，房价下跌 30-50%
2. 项目完工，但无法销售
3. Construction Loan 到期
4. 银行拒绝续贷
5. **结果**：进入"围地"的开发商破产

### 幸存者特征
- 低杠杆（50% LTV 以下）
- 多元化融资来源
- 现金储备充足
- 能够转为出租等待

---

## 如何避免进入"死地"

### 预防措施

1. **杠杆控制**
   - 总资产 < 2x 净资产
   - 避免 Margin Loan

2. **仓位分散**
   - 单一持仓 < 25%
   - 相关性高的资产合并计算

3. **流动性管理**
   - 现金储备 > 12 个月支出
   - 流动资产占比 > 30%

4. **退出路径**
   - 任何投资都要有 Plan B
   - 地产：能否转为出租？
   - 股票：流动性是否足够？

5. **压力测试**
   - 如果市场跌 50%，你能生存吗？
   - 如果利率涨 3%，你能承受吗？

### 地形监控仪表盘

| 指标 | 当前值 | 安全线 | 围地线 | 死地线 |
| ---- | ------ | ------ | ------ | ------ |
| 总杠杆率 | | < 1.5x | > 2x | > 3x |
| 最大单一仓位 | | < 20% | > 30% | > 50% |
| 现金占比 | | > 15% | < 10% | < 5% |
| 流动资产占比 | | > 40% | < 30% | < 20% |

## 本章总结

### Chapter Summary

孙子的"九地"理论告诉我们：**不同的地形需要不同的策略，最重要的是知道自己在哪里。**

**核心原则**：

1. **识别你的位置**：定期评估你的投资组合处于哪种"地形"
2. **避免死地**：永远不要让自己陷入无路可退的境地
3. **谨慎进入围地**：杠杆、锁定期、对手方风险都是"围地"特征
4. **保持机动性**：即使在重地，也要保留撤退的可能

> "知地知天，胜乃可全。" — 孙子
