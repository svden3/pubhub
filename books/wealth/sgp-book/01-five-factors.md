# 第一章：五事 — 看透企业的五个维度

## Chapter 1: The Five Factors — Five Dimensions to See Through an Enterprise

---

> "故经之以五事，校之以计而索其情：一曰道，二曰天，三曰地，四曰将，五曰法。"
>
> "Therefore, in your deliberations, when seeking to determine the military conditions, let them be made the basis of a comparison, in this wise: the Moral Law, Heaven, Earth, the Commander, Method and Discipline."
>
> — 《孙子兵法·始计篇》

---

## 五事概述 | Overview of the Five Factors

孙子开篇即强调，在开战之前，必须通过"五事"来衡量敌我双方的实力、预测战争胜负。这五事分别是：**道、天、地、将、法**。

在投资领域，这"五事"对应着我们评估一家企业的五个核心维度：

| 五事 | 孙子原意 | 投资映射 | 核心问题 |
| ---- | -------- | -------- | -------- |
| **道** | 政治与民心 | 治理与愿景 | 管理层与股东利益一致吗？ |
| **天** | 天时气候 | 宏观周期 | 现在是春夏还是秋冬？ |
| **地** | 地理环境 | 护城河 | 竞争优势能持续吗？ |
| **将** | 将领素质 | 管理层 | 领导者有智、信、仁、勇、严吗？ |
| **法** | 组织制度 | 财务健康 | 现金流充足、制度健全吗？ |

---

## 一、道 (Dao) — 治理与愿景

### The Moral Law — Governance & Vision

**孙子原意**：
> "道者，令民与上同意也，故可以与之死，可以与之生，而不畏危。"

民众与统治者的意愿高度一致，这样民众就能与君主同生共死，不畏惧危险。

**投资映射**：企业的使命愿景，以及**管理层是否与股东利益一致**。

### 评估要点

1. **利益一致性 (Alignment of Interests)**
   - 管理层持股比例如何？
   - 高管薪酬是否与长期业绩挂钩？
   - 是否存在大股东侵害中小股东的历史？

2. **使命与价值观 (Mission & Values)**
   - 公司有清晰的长期使命吗？
   - 这个使命是真实践行还是墙上标语？
   - 员工是否认同公司文化？

3. **诚信记录 (Integrity Track Record)**
   - 管理层过去是否有失信行为？
   - 财务报告是否透明？
   - 是否有过监管处罚？

### 红旗信号 🚩

- 大股东频繁减持
- 高管薪酬与业绩脱钩
- 关联交易复杂
- 财务报告"创意"会计
- 员工评价普遍负面

### 【华尔街案例】恩隆 (Enron) 的"无道"覆灭

- **表象**：能源巨头，华尔街宠儿，连续六年被《财富》评为"美国最具创新力公司"
- **真相**：管理层利用复杂会计手法隐藏巨额债务，CEO 和 CFO 疯狂套现
- **"道"的崩塌**：上下不同欲——管理层掏空公司，员工养老金化为乌有
- **结果**：2001 年破产，股价从 $90 跌至 $0.26
- **教训**："道"不正，一切皆空

### 【地产案例】评估开发项目的"道"

在加州地产投资中，"道"的评估包括：

- **开发商诚信**：过去项目是否如期交付？是否有纠纷历史？
- **合伙人利益一致**：GP 是否有 Co-invest？利润分配结构是否公平？
- **退出条款**：如果出现分歧，Operating Agreement 中的解决机制是什么？

## 二、天 (Tian) — 宏观周期

### Heaven — Macro Cycles

**孙子原意**：
> "天者，阴阳，寒暑、时制也。"

即昼夜、四季气候变化及自然规律。

**投资映射**：**宏观经济周期**和**市场情绪周期**。

### 评估要点

1. **经济周期 (Economic Cycle)**
   - 当前处于扩张期还是衰退期？
   - 利率是上升还是下降趋势？
   - 通胀水平如何？

2. **行业周期 (Industry Cycle)**
   - 该行业处于成长期、成熟期还是衰退期？
   - 技术变革对行业的影响？
   - 监管政策的趋势？

3. **市场情绪 (Market Sentiment)**
   - 当前是贪婪还是恐惧？
   - 估值水平处于历史什么位置？
   - VIX 恐慌指数如何？

### 判断"天时"的指标

| 指标 | 春夏（扩张） | 秋冬（收缩） |
| ---- | ------------ | ------------ |
| GDP 增长 | 正增长 | 负增长或放缓 |
| 失业率 | 下降 | 上升 |
| 利率趋势 | 低或下降 | 高或上升 |
| 企业盈利 | 增长 | 下滑 |
| 市场估值 | 合理或低估 | 高估 |
| VIX | < 20 | > 30 |

### 【华尔街案例】巴菲特 1969 年的"识天"

- **背景**：1960 年代末，美股牛市持续多年，估值泡沫
- **巴菲特的判断**："天"已入秋冬——找不到符合安全边际的股票
- **行动**：1969 年解散合伙基金，退还投资者资金
- **结果**：完美躲过 1973-74 熊市（道琼斯跌 45%）
- **教训**："非利不动，非得不用，非危不战"

### 【地产案例】加州地产的"天时"

| 周期信号 | 春夏（买入机会） | 秋冬（谨慎或卖出） |
| -------- | ---------------- | ------------------ |
| 利率 | 低或下降 | 7%+ 且上升 |
| 库存 | 高，买家市场 | 低，卖家市场 |
| 科技就业 | 裁员潮后回暖 | 大规模招聘高峰 |
| Permit 活动 | 低迷 | 过热 |

## 三、地 (Di) — 护城河

### Earth — Competitive Moats

**孙子原意**：
> "地者，远近、险易、广狭、死生也。"

即路途远近、地势险要平坦、战场宽窄以及死地或生地。

**投资映射**：企业的**竞争优势**和**行业格局**。

### 评估要点

1. **护城河深度 (Moat Depth)**
   - 品牌护城河：消费者是否有品牌忠诚？
   - 网络效应：用户越多，价值越大？
   - 成本优势：规模经济、独特资源？
   - 转换成本：客户离开的代价高吗？
   - 专利/监管壁垒：法律保护有多强？

2. **行业竞争格局 (Industry Structure)**
   - 市场集中度如何？
   - 竞争激烈程度？
   - 新进入者威胁？
   - 替代品威胁？
   - 供应商/客户议价能力？

3. **市场空间 (Total Addressable Market)**
   - 行业天花板有多高？
   - 企业的市场份额和增长空间？
   - 是朝阳产业还是夕阳产业？

### 护城河评分表

| 护城河类型 | 深度 | 持久性 | 举例 |
| ---------- | ---- | ------ | ---- |
| 品牌 | ⭐⭐⭐ | 高 | 可口可乐、茅台 |
| 网络效应 | ⭐⭐⭐⭐ | 高 | 微信、Visa |
| 转换成本 | ⭐⭐⭐ | 中-高 | SAP、彭博终端 |
| 成本优势 | ⭐⭐ | 中 | 台积电、Costco |
| 专利/监管 | ⭐⭐ | 中（有期限） | 制药公司 |

### 【华尔街案例】用"地"分析 Apple

- **品牌**：⭐⭐⭐⭐⭐ 全球最具价值品牌之一
- **网络效应**：⭐⭐⭐⭐ iOS 生态系统
- **转换成本**：⭐⭐⭐⭐ 用户迁移到安卓成本高
- **总评**：占据"高阳之位"——定价权强，竞争壁垒深

### 【地产案例】加州地产的"地利"

| 区域 | 地利优势 | 风险 |
| ---- | -------- | ---- |
| Palo Alto / Menlo Park | 学区顶尖，科技中心，供应极度有限 | 价格高，租金回报率低 |
| San Jose | 科技就业密集，价格相对合理 | 对科技周期敏感 |
| East Bay (Oakland, Fremont) | 价格较低，租金回报更好 | 通勤依赖，治安差异大 |
| Gilroy / Morgan Hill | 土地多，开发空间大 | 需求弱于核心区域 |

## 四、将 (Jiang) — 管理层

### The Commander — Management Quality

**孙子原意**：
> "将者，智、信、仁、勇、严也。"

孙子提出为将者的五个核心标准。

**投资映射**：**CEO 及管理团队的能力画像**。

### 将的五个维度

| 维度 | 孙子原意 | 管理层评估 | 具体表现 |
| ---- | -------- | ---------- | -------- |
| **智** | 足智多谋 | 战略眼光 | 能否看清行业趋势、把握机会？ |
| **信** | 赏罚有信 | 兑现承诺 | 过去的业绩指引是否兑现？ |
| **仁** | 爱护士卒 | 善待利益相关者 | 对员工、股东、社区的态度？ |
| **勇** | 果敢决断 | 危机中的决策 | 困难时刻是否敢于变革？ |
| **严** | 军纪严明 | 内控与成本 | 管理是否严格、成本控制如何？ |

### 评估方法

1. **阅读年报中的股东信**：是否坦诚、有深度？
2. **观看财报电话会议**：回答问题时的态度和逻辑
3. **追踪历史承诺**：过去说的是否做到？
4. **员工评价**：Glassdoor、LinkedIn 上的真实反馈
5. **离职高管动向**：核心团队是否稳定？

### 【华尔街案例】"将"的正反案例

**正面：杰夫·贝索斯 (Jeff Bezos)**
- 智：长期思维，敢于投资未来（AWS）
- 信：一直强调"Day 1"文化，言行一致
- 勇：亏损多年坚持战略，顶住华尔街压力

**反面：亚当·诺伊曼 (Adam Neumann) - WeWork**
- 智：愿景宏大但缺乏盈利路径
- 信：承诺频繁变化
- 仁：巨额套现同时裁员
- 严：公司治理混乱

### 【地产案例】评估总包商的"将"

| 维度 | 评估内容 |
| ---- | -------- |
| 智 | 是否有解决问题的创造力？ |
| 信 | 预算和工期是否可靠？ |
| 仁 | 与分包商关系如何？ |
| 勇 | 遇到问题是否主动沟通？ |
| 严 | 施工质量控制如何？ |

## 五、法 (Fa) — 财务与制度

### Method and Discipline — Financial Health & Systems

**孙子原意**：
> "法者，曲制、官道、主用也。"

即军队的组织编制、各级职责管理、军需物资供应。

**投资映射**：企业的**商业模式**、**财务健康**和**运营效率**。

### 评估要点

1. **现金流 (Cash Flow) — "主用"**
   - 经营现金流是否为正？
   - 自由现金流能否覆盖资本支出？
   - 现金储备能维持多长时间？

2. **资本结构 (Capital Structure)**
   - 负债率是否合理？
   - 债务到期时间分布？
   - 利息覆盖倍数？

3. **运营效率 (Operational Efficiency) — "曲制"**
   - 毛利率趋势？
   - 运营杠杆？
   - 资产周转率？

4. **激励机制 (Incentive Structure) — "官道"**
   - 员工期权计划？
   - 高管激励与长期业绩挂钩？
   - 内部晋升机制？

### 格雷厄姆的财务筛选标准

| 指标 | 防御型标准 | 进攻型标准 |
| ---- | ---------- | ---------- |
| 流动比率 | > 2.0 | > 1.5 |
| 长期负债 | < 净流动资产 | < 2x 净利润 |
| 连续盈利 | 10 年 | 5 年 |
| 连续分红 | 20 年 | 5 年 |
| EPS 增长 | 10 年 > 33% | 5 年 > 25% |

### 【华尔街案例】"法"的崩塌 — 雷曼兄弟

- **表象**：百年投行，华尔街五大行之一
- **"法"的问题**：
  - 主用（现金流）：高度依赖短期融资
  - 曲制（风控）：杠杆率高达 30 倍
- **结果**：2008 年 9 月破产，全球金融危机导火索
- **教训**：再大的机构，"法"不健全也会一夜崩溃

### 【地产案例】项目的"法"评估

| 维度 | 评估内容 |
| ---- | -------- |
| 主用 | 项目现金流预测是否合理？融资是否到位？ |
| 曲制 | 成本控制机制？Contingency 预留多少？ |
| 官道 | 激励机制？GC 是否有利润分享？ |

## 五事庙算评分表

### Five Factors Scorecard

| 维度 | 评估问题 | 评分 (1-5) | 权重 | 加权分 |
| ---- | -------- | ---------- | ---- | ------ |
| **道** | 管理层与股东利益一致吗？ | | 20% | |
| **天** | 当前周期有利吗？ | | 15% | |
| **地** | 护城河深吗？ | | 25% | |
| **将** | 管理层优秀吗？ | | 20% | |
| **法** | 财务健康吗？ | | 20% | |
| | | **总分** | | |

**评分标准**：
- 5 分：卓越
- 4 分：良好
- 3 分：一般
- 2 分：较差
- 1 分：危险

**决策规则**：
- 总分 > 4.0：可以考虑投资
- 总分 3.0-4.0：需要更多研究
- 总分 < 3.0：不投资
- 任一维度 < 2：一票否决

## 本章总结

### Chapter Summary

孙子认为："**凡此五者，将莫不闻，知之者胜，不知者不胜。**"

在"孙子-格雷厄姆"投资体系中，"五事"就是你的**投资前尽职调查清单 (Due Diligence Checklist)**。

**记住**：
- **道**：上下是否同欲？
- **天**：时机是否有利？
- **地**：护城河是否深厚？
- **将**：管理层是否可靠？
- **法**：财务是否健康？

只有在这五个维度上都经过了严密的"庙算"（计算），投资者才能做到"多算胜"，从而确立"先为不可胜"的安全边际。
