## 退休的三个数字

### 数字一：你需要多少钱？

**简单公式**：
```
退休储蓄 = 年支出 × 25
```

这叫 **4% 法则**——每年从退休储蓄中提取 4%，理论上可以用 30 年以上。

**举例**：
- 年支出 $60,000 → 需要 $1,500,000
- 年支出 $80,000 → 需要 $2,000,000
- 年支出 $100,000 → 需要 $2,500,000

妈妈说：
> "儿子，这只是理论。实际上，你不会花一样多的钱。60 岁和 80 岁，开支完全不同。"

---

### 数字二：你还有多少时间？

**复利的魔法**：
```
25 岁开始 vs 35 岁开始

假设每月存 $500，年回报 7%：

25 岁开始 → 65 岁有 $1,200,000
35 岁开始 → 65 岁有 $567,000
```

**10 年的差距 = 63 万美元**

Sarah 说：
> "复利是时间的函数。开始得越早，终点越远。"

---

### 数字三：你的收入来源有几个？

**理想的退休收入来源**：
```
1. 社保（Social Security）  —— 基础保障
2. 退休账户（401k/IRA）    —— 税务优惠
3. 投资收入（股息/利息）    —— 被动收入
4. 兼职/顾问                —— 如果你想
```

**Mike 的教训**：
> "我所有的钱都在 Enron 股票里。退休金、401k、个人账户。
> 我以为我是百万富翁。
> 三个月后，我一无所有。"

**规则**：退休收入至少要有 2-3 个独立来源。

## 退休账户：硅谷人的税务武器

### 401(k)：公司帮你存

```
2024 年限额：$23,000（50 岁以上 +$7,500）

优点：
✓ 税前存入，当年省税
✓ 公司 Match（免费的钱！）
✓ 自动扣款，无需纪律

缺点：
✗ 59.5 岁前取出要罚款
✗ 取出时按收入纳税
✗ 投资选择有限
```

**妈妈的命令**：
> "儿子，401k 至少存到公司 Match 的上限。不存就是白送钱给政府。"

---

### Roth IRA：年轻人的最佳选择

```
2024 年限额：$7,000（50 岁以上 +$1,000）

优点：
✓ 税后存入，增长免税
✓ 取出时完全免税
✓ 退休后灵活度高
✓ 没有强制取出要求

缺点：
✗ 收入太高不能存（$161k 单身 / $240k 夫妻）
✗ 限额较低
```

**Sarah 的建议**：
> "年轻时收入低，税率低。用 Roth IRA。
> 老了收入高，税率高。用 Traditional 401k。
> 这叫税务套利。"

---

### Backdoor Roth：高收入者的后门

如果收入超过 Roth IRA 限额：

```
步骤：
1. 存入 Traditional IRA（非抵税）
2. 立即转换成 Roth IRA
3. 缴少量转换税
4. 享受终身免税增长
```

**注意**：要咨询税务专家，有 Pro Rata 规则。

## 退休的四个阶段

### 第一阶段：积累期（25-45 岁）

**目标**：存得越多越好

```
策略：
- 最大化 401k + Roth IRA
- 投资主要放在股票
- 承受波动，追求增长
- 不要碰退休账户
```

---

### 第二阶段：过渡期（45-55 岁）

**目标**：降低风险，保护本金

```
策略：
- 逐步从股票转向债券
- 开始规划退休后的生活
- 计算你的"数字"
- 考虑提前退休的可能
```

Sarah 在 45 岁时：
> "我开始认真算了。股票从 80% 降到 60%。债券从 10% 升到 30%。现金保持 10%。"

---

### 第三阶段：退休初期（55-70 岁）

**目标**：活跃、自由、有选择

```
策略：
- 健康是最大的投资
- 可以做自己想做的事
- 支出可能比工作时更高（旅行、爱好）
- 保持一定的股票配置（对抗通胀）
```

---

### 第四阶段：退休后期（70 岁以后）

**目标**：简化、安心、传承

```
策略：
- 支出通常下降
- 医疗费用上升
- 考虑遗产规划
- 简化投资组合
```

妈妈说：
> "儿子，到了这个年纪，钱不是最重要的了。重要的是有人陪你吃饭。"

## 加州房产：退休的隐形炸弹

2022 年，Mike 的邻居 David 要退休了。

"我 1995 年买的房子，$300,000。现在值 $2,500,000。"

"恭喜啊！"

"**恭喜什么？我想卖掉搬去德州，但联邦资本利得税要交 $440,000。**"

这就是加州退休的第一课：**你的房子可能是金手铐**。

---

### Prop 13：加州最伟大的法案

**1978 年的规则**：
```
Prop 13 核心：
- 房产税基于购买价，不是市场价
- 每年最多涨 2%
- 房子不卖，税基不变
```

**David 的例子**：
```
1995 年买入：$300,000
2024 年市值：$2,500,000

按 Prop 13：每年房产税 ≈ $5,000（基于调整后的购买价 ~$420,000）
按市价算：每年房产税 ≈ $30,000

每年省：$25,000
```

妈妈说：
> "儿子，在加州买房，就像买了终身打折卡。但你一卖，折扣就没了。"

---

### 1031 Exchange：延税的艺术

**联邦税法第 1031 条**：
```
规则：
- 卖投资房产，买同类房产
- 资本利得税可以延迟
- 不是免税，是递延

时间要求：
- 45 天内确定新房产
- 180 天内完成交易

限制：
- 只适用于投资房产
- 不适用于自住房（有其他免税额）
- 必须"同类交换"（Like-Kind）
```

**David 的想法**：
> "我能不能用 1031 把 Palo Alto 的房子换成德州的房子？"

**答案**：可以。但你会失去 Prop 13 的保护。而且德州虽然没有州税，但房产税是加州的 3 倍。

---

### Prop 19 陷阱：2020 年的糖衣毒药

**表面的"好处"**：
```
Prop 19（2020 年生效）：

"好处"：
- 55 岁以上可以带税基搬家（全州范围）
- 残疾人、灾民也可以
- 最多可用 3 次

听起来不错？等等...
```

**真正的陷阱**：
```
Prop 19 的代价：

旧规则（Prop 58）：
- 父母房产传给子女，税基不变
- 子女继承 $300,000 的税基
- 继续享受低房产税

新规则（Prop 19）：
- 继承后如果不自住，重新评估市值
- 子女继承 $2,500,000 市值的房子
- 房产税从 $5,000 跳到 $30,000
- 除非子女搬进去住
```

**Mike 的分析**：
> "Prop 19 就是政府的把戏。
> 表面上给老人搬家的便利。
> 实际上取消了子女继承的税基保护。
> 加州政府每年多收几十亿房产税。"

```
□ 401k 至少存到公司 Match 上限
□ 开设 Roth IRA 或 Backdoor Roth
□ 了解你的退休"数字"（年支出 × 25）
□ 退休收入至少 2-3 个来源

投资配置：
□ 股债配置随年龄调整
□ 不要把退休金全押一个股票
□ 指数基金为核心，长期持有

加州房产（如适用）：
□ 理解 Prop 13 的价值
□ 了解 Prop 19 对继承的影响
□ 考虑 1031 Exchange 的利弊
□ 计算卖房 vs 持有的税务成本

心态准备：
□ 想清楚退休后要做什么
□ 买对不难，拿住才难
□ 时间比钱贵
```

# 第二十六章：医保是隐形的财富

> **妈妈说："儿子，健康的人有一千个愿望，生病的人只有一个。"**

---

## Sarah 的算术课（续）

2020 年，Sarah 48 岁，准备退休。

"等等，" 我说，"你才 48。Medicare 要 65 岁才有。这中间 17 年怎么办？"

Sarah 拿出一张纸：

```
48-65 岁医保方案：

选项 1：COBRA（离职后延续公司保险）
- 每月 $2,200（家庭）
- 只能用 18 个月
- 太贵了

选项 2：ACA（奥巴马医保）
- 每月 $800-1,500（看收入和补贴）
- 可以一直用到 65 岁
- 这就是我的选择

选项 3：配偶保险
- 如果配偶还在工作
- 这是最划算的
```

"**医保是提前退休最大的障碍。**很多人不是没钱退休，是没保险不敢退。"

## 美国医保系统：一张图看懂

```
                    美国医保全景图

┌─────────────────────────────────────────────────────┐

├─────────────────────────────────────────────────────┤
```

## 雇主保险：硅谷的隐形福利

### 为什么大公司的保险这么好

**Google 的医保（Sarah 的例子）：**

```
Google 员工医保福利：

- 保费：公司付 90%+，员工付很少
- 免赔额：$500 或更低
- 自付最高限额：$3,000
- 包括：牙科、眼科、心理健康
- 额外：健身房、按摩、营养咨询

每年价值：$20,000-30,000（如果自己买）
```

**这是隐形的年薪。**

很多人只看 base salary，忘了医保福利值多少钱。

---

### 小公司 vs 大公司

| 项目 | 大公司（Google） | 小公司/创业公司 |
|------|-----------------|-----------------|
| 保费分担 | 公司 90%+ | 公司 50-70% |
| 免赔额 | $500 | $2,000-5,000 |
| 网络 | PPO（自由选医生） | HMO（限制多） |
| 额外福利 | 很多 | 很少 |

**Mike 的教训：**

> "我去了一家创业公司，工资涨了 20%。
> 但医保从 PPO 变成 High Deductible。
> 第一年我儿子住院，自付 $8,000。
> 那 20% 涨薪全没了。"

---

## HSA：三重免税的超级账户

### 什么是 HSA

**Health Savings Account（健康储蓄账户）**

只有 High Deductible Health Plan（HDHP）才能开。

### 三重免税

```
HSA 的超能力：

1. 存入免税    — 从税前收入扣除
2. 增长免税    — 投资收益不交税
3. 取出免税    — 用于医疗支出不交税

这是美国税法里唯一的"三重免税"账户。
比 401k 还好（401k 取出要交税）。
比 Roth IRA 还好（Roth 存入要交税）。
```

### 2024 年限额

| 类型 | 限额 |
|------|------|
| 个人 | $4,150 |
| 家庭 | $8,300 |
| 55 岁以上追加 | +$1,000 |

### 隐藏的退休账户

**Sarah 的 HSA 策略：**

> "我不用 HSA 付现在的医疗费。
> 我用现金付，保留收据。
> HSA 的钱全部投资指数基金。
>
> 20 年后，这些钱免税增长。
> 65 岁以后，可以用于任何支出（非医疗支出交普通税，但没罚款）。
>
> **HSA 是隐形的 Super Roth IRA。**"

---

## ACA（奥巴马医保）：提前退休的救星

### 什么是 ACA

**Affordable Care Act**，2010 年通过。

核心规则：
- 不能因为既往病史拒保
- 必须覆盖基本医疗服务
- 低收入有补贴

### 补贴怎么算

```
ACA 补贴规则（2024 年）：

你的收入 vs 联邦贫困线（FPL）

收入 < 150% FPL  →  最多付收入的 0-2%
收入 150-200% FPL →  最多付收入的 2-4%
收入 200-250% FPL →  最多付收入的 4-6%
收入 250-400% FPL →  最多付收入的 6-8.5%
收入 > 400% FPL  →  目前没有上限（但 2025 年可能恢复）

2024 年 FPL（48 州）：
- 个人：$15,060
- 2 人家庭：$20,440
- 4 人家庭：$31,200
```

### 提前退休的关键

**Sarah 的操作：**

```
2020 年退休时：

Google 股票：大部分是长期资本利得
每年卖一点：控制在 $60,000 左右
家庭收入：$60,000（约 192% FPL）

ACA 保费（2 人家庭）：
- 市场价：$1,800/月
- 补贴后：$400/月

每年省：$16,800
```

**关键技巧：控制 AGI（调整后总收入）**

- 卖股票要算资本利得
- 401k 取出要算收入
- Roth IRA 取出不算收入

"**Roth IRA 是提前退休的好朋友。取出不算收入，不影响 ACA 补贴。**"

---

## Medicare：65 岁的礼物

### 四个部分

```
Medicare 结构：

Part A（住院）
- 大部分人免费（工作满 10 年）
- 覆盖：住院、护理院、临终关怀
- 有免赔额和自付

Part B（门诊）
- 每月 $174.70（2024 年标准）
- 高收入更高（IRMAA 附加费）
- 覆盖：医生、检查、门诊手术

Part C（Medicare Advantage）
- 私人保险公司提供
- 整合 A + B + 通常包括 D
- 可能有额外福利（牙科、眼科）

Part D（处方药）
- 单独购买或包含在 Part C
- 不同计划覆盖不同药物
```

### IRMAA：高收入的惩罚

**Income-Related Monthly Adjustment Amount**

```
2024 年 IRMAA 门槛（个人）：

收入 ≤ $103,000        →  Part B: $174.70
$103,000 - $129,000    →  Part B: $244.60
$129,000 - $161,000    →  Part B: $349.40
$161,000 - $193,000    →  Part B: $454.20
$193,000 - $500,000    →  Part B: $559.00
> $500,000             →  Part B: $594.00

Part D 也有类似的附加费。
```

**注意：用的是 2 年前的收入！**

Sarah：
> "我 2020 年从 Google 退休，当年卖了很多股票。
> 2022 年我满 65 岁，Medicare 用我 2020 年的收入算。
> Part B 保费从 $170 变成 $500 多。
>
> **教训：退休前两年要规划好收入。**"

## 牙科和眼科：被遗忘的角落

### 问题

Medicare 不包括：
- 常规牙科
- 常规眼科（验光、眼镜）
- 助听器

这些要单独买保险，或者自付。

### 解决方案

```
选项 1：单独买牙科/眼科保险
- 每月 $30-50
- 覆盖有限，主要是预防性检查

选项 2：Medicare Advantage（Part C）
- 很多计划包括牙科/眼科
- 选计划时要仔细看

选项 3：自付
- 每年两次洗牙：$200-400
- 验光 + 眼镜：$300-500
- 可能比保费便宜

选项 4：牙科学校
- 便宜 50-70%
- 但需要时间和耐心
```

---

## 长期护理：最大的隐形风险

### 数字说话

```
长期护理费用（2024 年）：

护理院（Nursing Home）
- 全国平均：$9,000/月
- 加州：$12,000/月
- 纽约：$14,000/月

辅助生活（Assisted Living）
- 全国平均：$4,500/月

居家护理
- $25-35/小时
- 全职：$8,000-12,000/月

需要长期护理的概率：
- 65 岁以上：约 70% 需要某种形式的长期护理
- 平均需要时间：女性 3.7 年，男性 2.2 年
```

**Medicare 不覆盖长期护理！**

只覆盖短期康复（最多 100 天）。

### 选项

```
选项 1：长期护理保险（LTCI）
- 最好 50-60 岁买
- 保费随年龄增加
- 保险公司可能涨价或倒闭

选项 2：自己存钱
- 假设需要 $300,000-500,000
- 放在独立账户

选项 3：Medicaid（最后手段）
- 必须"花光"资产
- 只有 $2,000 以下才有资格
- 选择有限

选项 4：混合保险
- 人寿保险 + 长期护理
- 如果没用上，留给继承人
```

**妈妈的态度：**

> "儿子，我不要去护理院。
> 我要在自己家里走。
> 你帮我请个人就行。"

## 医疗旅游：硅谷人的秘密

### 为什么

美国医疗费用是全球最高。

```
同样的手术：

膝关节置换：
- 美国：$50,000
- 墨西哥：$12,000
- 泰国：$10,000
- 印度：$7,000

牙科植牙（单颗）：
- 美国：$5,000
- 墨西哥：$1,500
- 哥斯达黎加：$1,000
```

### Sarah 的经历

> "我去墨西哥做了牙科。
> 洛斯阿尔戈多内斯（Los Algodones），边境小镇。
> 那里有 300 多个牙医诊所。
>
> 四颗牙冠 + 洗牙：$2,000

>
> 医生是美国牙科学院毕业的。
> 设备是德国进口的。
> 唯一的区别是价格。"

### 注意事项

- 做好调查（看评价、认证）
- 复杂手术要三思
- 计算旅行成本
- 考虑术后护理

## 健康才是最大的财富

### Mike 的故事

2018 年，Mike 52 岁。

体检发现：2 型糖尿病。

```
Mike 的医疗账单（每年）：

药物：$3,000
检查：$1,500
专科：$2,000
总计：$6,500/年

如果 10 年前开始锻炼呢？
```

**妈妈说：**

> "儿子，健身房会员费 $50/月，
> 糖尿病药 $300/月。
> 你自己算。"

### 预防 vs 治疗

```
投资健康的回报：

运动（每天 30 分钟）
- 成本：$0-50/月
- 回报：降低心脏病、糖尿病、癌症风险

健康饮食
- 成本：可能比垃圾食品贵 20%
- 回报：更少的医疗支出

年度体检
- 成本：通常保险全覆盖
- 回报：早发现、早治疗

这是真正的"投资"。
```

```
工作时：
□ 最大化公司医保福利

提前退休（65 岁前）：
□ 计算 ACA 保费和补贴
□ 规划收入以最大化补贴
□ 用 Roth IRA 取款（不算收入）
□ 保留 HSA，让它继续增长

65 岁后：
□ 提前 3 个月申请 Medicare
□ 选择 Original Medicare 还是 Medicare Advantage
□ 考虑 Medigap 补充保险
□ 注意 IRMAA（用 2 年前收入算）

长期规划：
□ 考虑长期护理风险
□ 预防胜于治疗
□ 健康是最好的投资
```
