# 第二十六章：医保是隐形的财富

> **妈妈说："儿子，健康的人有一千个愿望，生病的人只有一个。"**

## Sarah 的算术课（续）

2020 年，Sarah 48 岁，准备退休。

"等等，" 我说，"你才 48。Medicare 要 65 岁才有。这中间 17 年怎么办？"

Sarah 拿出一张纸：

```
48-65 岁医保方案：

选项 1：COBRA（离职后延续公司保险）
- 每月 $2,200（家庭）
- 只能用 18 个月
- 太贵了

选项 2：ACA（奥巴马医保）
- 每月 $800-1,500（看收入和补贴）
- 可以一直用到 65 岁
- 这就是我的选择

选项 3：配偶保险
- 如果配偶还在工作
- 这是最划算的
```

"**医保是提前退休最大的障碍。**很多人不是没钱退休，是没保险不敢退。"

## 雇主保险：硅谷的隐形福利

### 为什么大公司的保险这么好

**Google 的医保（Sarah 的例子）：**

```
Google 员工医保福利：

- 保费：公司付 90%+，员工付很少
- 免赔额：$500 或更低
- 自付最高限额：$3,000
- 包括：牙科、眼科、心理健康
- 额外：健身房、按摩、营养咨询

每年价值：$20,000-30,000（如果自己买）
```

**这是隐形的年薪。**

很多人只看 base salary，忘了医保福利值多少钱。

---

### 小公司 vs 大公司

| 项目 | 大公司（Google） | 小公司/创业公司 |
|------|-----------------|-----------------|
| 保费分担 | 公司 90%+ | 公司 50-70% |
| 免赔额 | $500 | $2,000-5,000 |
| 网络 | PPO（自由选医生） | HMO（限制多） |
| 额外福利 | 很多 | 很少 |

**Mike 的教训：**

> "我去了一家创业公司，工资涨了 20%。
> 但医保从 PPO 变成 High Deductible。
> 第一年我儿子住院，自付 $8,000。
> 那 20% 涨薪全没了。"

## HSA：三重免税的超级账户

### 什么是 HSA

**Health Savings Account（健康储蓄账户）**

只有 High Deductible Health Plan（HDHP）才能开。

### 三重免税

```
HSA 的超能力：

1. 存入免税    — 从税前收入扣除
2. 增长免税    — 投资收益不交税
3. 取出免税    — 用于医疗支出不交税

这是美国税法里唯一的"三重免税"账户。
比 401k 还好（401k 取出要交税）。
比 Roth IRA 还好（Roth 存入要交税）。
```

### 2024 年限额

| 类型 | 限额 |
|------|------|
| 个人 | $4,150 |
| 家庭 | $8,300 |
| 55 岁以上追加 | +$1,000 |

### 隐藏的退休账户

**Sarah 的 HSA 策略：**

> "我不用 HSA 付现在的医疗费。
> 我用现金付，保留收据。
> HSA 的钱全部投资指数基金。
>
> 20 年后，这些钱免税增长。
> 65 岁以后，可以用于任何支出（非医疗支出交普通税，但没罚款）。
>
> **HSA 是隐形的 Super Roth IRA。**"

## ACA（奥巴马医保）：提前退休的救星

### 什么是 ACA

**Affordable Care Act**，2010 年通过。

核心规则：
- 不能因为既往病史拒保
- 必须覆盖基本医疗服务
- 低收入有补贴

### 补贴怎么算

```
ACA 补贴规则（2024 年）：

你的收入 vs 联邦贫困线（FPL）

收入 < 150% FPL  →  最多付收入的 0-2%
收入 150-200% FPL →  最多付收入的 2-4%
收入 200-250% FPL →  最多付收入的 4-6%
收入 250-400% FPL →  最多付收入的 6-8.5%
收入 > 400% FPL  →  目前没有上限（但 2025 年可能恢复）

2024 年 FPL（48 州）：
- 个人：$15,060
- 2 人家庭：$20,440
- 4 人家庭：$31,200
```

### 提前退休的关键

**Sarah 的操作：**

```
2020 年退休时：

Google 股票：大部分是长期资本利得
每年卖一点：控制在 $60,000 左右
家庭收入：$60,000（约 192% FPL）

ACA 保费（2 人家庭）：
- 市场价：$1,800/月
- 补贴后：$400/月

每年省：$16,800
```

**关键技巧：控制 AGI（调整后总收入）**

- 卖股票要算资本利得
- 401k 取出要算收入
- Roth IRA 取出不算收入

"**Roth IRA 是提前退休的好朋友。取出不算收入，不影响 ACA 补贴。**"

## Medicare：65 岁的礼物

### 四个部分

```
Medicare 结构：

Part A（住院）
- 大部分人免费（工作满 10 年）
- 覆盖：住院、护理院、临终关怀
- 有免赔额和自付

Part B（门诊）
- 每月 $174.70（2024 年标准）
- 高收入更高（IRMAA 附加费）
- 覆盖：医生、检查、门诊手术

Part C（Medicare Advantage）
- 私人保险公司提供
- 整合 A + B + 通常包括 D
- 可能有额外福利（牙科、眼科）

Part D（处方药）
- 单独购买或包含在 Part C
- 不同计划覆盖不同药物
```

### IRMAA：高收入的惩罚

**Income-Related Monthly Adjustment Amount**

```
2024 年 IRMAA 门槛（个人）：

收入 ≤ $103,000        →  Part B: $174.70
$103,000 - $129,000    →  Part B: $244.60
$129,000 - $161,000    →  Part B: $349.40
$161,000 - $193,000    →  Part B: $454.20
$193,000 - $500,000    →  Part B: $559.00
> $500,000             →  Part B: $594.00

Part D 也有类似的附加费。
```

**注意：用的是 2 年前的收入！**

Sarah：
> "我 2020 年从 Google 退休，当年卖了很多股票。
> 2022 年我满 65 岁，Medicare 用我 2020 年的收入算。
> Part B 保费从 $170 变成 $500 多。
>
> **教训：退休前两年要规划好收入。**"

## 牙科和眼科：被遗忘的角落

### 问题

Medicare 不包括：
- 常规牙科
- 常规眼科（验光、眼镜）
- 助听器

这些要单独买保险，或者自付。

### 解决方案

```
选项 1：单独买牙科/眼科保险
- 每月 $30-50
- 覆盖有限，主要是预防性检查

选项 2：Medicare Advantage（Part C）
- 很多计划包括牙科/眼科
- 选计划时要仔细看

选项 3：自付
- 每年两次洗牙：$200-400
- 验光 + 眼镜：$300-500
- 可能比保费便宜

选项 4：牙科学校
- 便宜 50-70%
- 但需要时间和耐心
```

## 长期护理：最大的隐形风险

### 数字说话

```
长期护理费用（2024 年）：

护理院（Nursing Home）
- 全国平均：$9,000/月
- 加州：$12,000/月
- 纽约：$14,000/月

辅助生活（Assisted Living）
- 全国平均：$4,500/月

居家护理
- $25-35/小时
- 全职：$8,000-12,000/月

需要长期护理的概率：
- 65 岁以上：约 70% 需要某种形式的长期护理
- 平均需要时间：女性 3.7 年，男性 2.2 年
```

**Medicare 不覆盖长期护理！**

只覆盖短期康复（最多 100 天）。

### 选项

```
选项 1：长期护理保险（LTCI）
- 最好 50-60 岁买
- 保费随年龄增加
- 保险公司可能涨价或倒闭

选项 2：自己存钱
- 假设需要 $300,000-500,000
- 放在独立账户

选项 3：Medicaid（最后手段）
- 必须"花光"资产
- 只有 $2,000 以下才有资格
- 选择有限

选项 4：混合保险
- 人寿保险 + 长期护理
- 如果没用上，留给继承人
```

**妈妈的态度：**

> "儿子，我不要去护理院。
> 我要在自己家里走。
> 你帮我请个人就行。"

## 医疗旅游：硅谷人的秘密

### 为什么

美国医疗费用是全球最高。

```
同样的手术：

膝关节置换：
- 美国：$50,000
- 墨西哥：$12,000
- 泰国：$10,000
- 印度：$7,000

牙科植牙（单颗）：
- 美国：$5,000
- 墨西哥：$1,500
- 哥斯达黎加：$1,000
```

### Sarah 的经历

> "我去墨西哥做了牙科。
> 洛斯阿尔戈多内斯（Los Algodones），边境小镇。
> 那里有 300 多个牙医诊所。
>
> 四颗牙冠 + 洗牙：$2,000
> 美国报价：$8,000
>
> 医生是美国牙科学院毕业的。
> 设备是德国进口的。
> 唯一的区别是价格。"

### 注意事项

- 做好调查（看评价、认证）
- 复杂手术要三思
- 计算旅行成本
- 考虑术后护理

## 健康才是最大的财富

### Mike 的故事

2018 年，Mike 52 岁。

体检发现：2 型糖尿病。

```
Mike 的医疗账单（每年）：

药物：$3,000
检查：$1,500
专科：$2,000
总计：$6,500/年

如果 10 年前开始锻炼呢？
```

**妈妈说：**

> "儿子，健身房会员费 $50/月，
> 糖尿病药 $300/月。
> 你自己算。"

### 预防 vs 治疗

```
投资健康的回报：

运动（每天 30 分钟）
- 成本：$0-50/月
- 回报：降低心脏病、糖尿病、癌症风险

健康饮食
- 成本：可能比垃圾食品贵 20%
- 回报：更少的医疗支出

年度体检
- 成本：通常保险全覆盖
- 回报：早发现、早治疗

这是真正的"投资"。
```

## 阿甘的医保清单

```
工作时：
□ 最大化公司医保福利
□ 如果有 HDHP，开 HSA 并存满
□ 了解 COBRA 规则（离职时可延续 18 个月）
□ 考虑配偶保险作为备选

提前退休（65 岁前）：
□ 计算 ACA 保费和补贴
□ 规划收入以最大化补贴
□ 用 Roth IRA 取款（不算收入）
□ 保留 HSA，让它继续增长

65 岁后：
□ 提前 3 个月申请 Medicare
□ 选择 Original Medicare 还是 Medicare Advantage
□ 考虑 Medigap 补充保险
□ 注意 IRMAA（用 2 年前收入算）

长期规划：
□ 考虑长期护理风险
□ 预防胜于治疗
□ 健康是最好的投资
```
