# 第十九章：公立还是私立

> "妈妈说：花钱能买到好教育，但最好的老师是免费的——那就是你自己。"

---

## 问题

"公立学校还是私立学校？"

这是每个硅谷家长的问题。

答案不是"哪个更好"，而是"什么适合你"。

---

## 数字说话

### 费用对比（加州 2025 年）

| 类型 | 年费用 | K-12 总费用 |
|------|-------|------------|
| 公立 | $0 | $0 |
| 私立普通 | $25,000-40,000 | $325,000-520,000 |
| 私立顶尖 | $50,000-65,000 | $650,000-845,000 |

**私立 K-12 可能比大学还贵。**

### 机会成本

如果把 $50,000/年 投到指数基金，18 年后：

```
每年 $50,000
× 18 年
× 7% 年化回报
= ~$1,700,000
```

**一百七十万美元。**

这是选择私立学校的真实成本。

---

## 什么时候值得

### 公立学校足够好的情况

| 条件 | 说明 |
|------|------|
| 你住在好学区 | Cupertino、Palo Alto、Fremont |
| 孩子没有特殊需求 | 标准课程能满足 |
| 你愿意补充课外 | 课后班、夏令营、家庭教育 |

### 私立学校值得考虑的情况

| 条件 | 说明 |
|------|------|
| 孩子有特殊需求 | 学习障碍、天才儿童 |
| 公立学区很差 | 搬家成本太高 |
| 宗教/价值观 | 特定教育理念 |
| 你真的负担得起 | 不影响退休储蓄 |

---

## 隐藏的选项

### 选项 1：买房进好学区

在好学区买房可能比私立学校更划算：

```
Cupertino 房子 vs Los Gatos 房子差价：~$500,000
但这是资产，不是消费
18 年后房子可能增值
```

**房子会涨，学费不会回来。**

### 选项 2：公立 + 补充

公立学校 + 课外活动：

```
公立学校：$0
钢琴课：$5,000/年
编程课：$3,000/年
夏令营：$5,000/年
家教：$5,000/年
---
总计：~$18,000/年
```

**比私立便宜一半，灵活度更高。**

### 选项 3：只念私立高中

有些家庭的策略：

- K-8：公立
- 9-12：私立

高中 4 年私立：$200,000 vs K-12 私立：$700,000

**省了 $500,000。**

---

## 真正的问题

问自己：

1. **为什么想让孩子上私立？**
   - 如果是"大家都上"，那不是好理由
   - 如果是"孩子有特殊需求"，那值得考虑

2. **这会影响你的财务安全吗？**
   - 如果要借钱、动用退休金 → 不值得
   - 如果轻松负担 → 可以考虑

3. **孩子真的需要吗？**
   - 有些孩子在任何环境都能成功
   - 有些孩子需要特定环境

## 妈妈的观点

妈妈在 Baton Rouge 的公立学校读书。

她说："**学校教知识，家庭教做人。好学校是加分，不是必须。**"

Sarah 从 Berkeley 毕业，公立大学。

她说："**我认识的最成功的人，不是从最好的学校出来的。是那些最努力的人。**"
