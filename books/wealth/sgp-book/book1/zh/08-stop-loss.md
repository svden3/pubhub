# 第八章：止损的艺术

> "小敌之坚，大敌之擒也。"
>

## 孙子论"知难而退"

孙子说：弱小的军队如果只知道硬拼，就会被强大的敌人俘虏。

投资中的道理相同：**不认输的小亏，会变成吃掉你的大亏**。

| 原文 | 直译 | 投资映射 |
|-----|------|---------|
| 小敌之坚 | 弱小却坚持硬拼 | 死扛不止损 |
| 大敌之擒 | 被强敌俘虏 | 被市场消灭 |

---

## 为什么止损这么难

### 心理障碍

| 心理 | 表现 | 结果 |
|-----|------|------|
| **损失厌恶** | 亏损的痛苦是盈利快乐的2倍 | 不愿意"确认"亏损 |
| **锚定效应** | 盯着买入价不放 | "回本就卖" |
| **沉没成本** | "已经亏这么多了" | 继续加仓 |
| **确认偏误** | 只看好消息 | 忽略警示信号 |
| **禀赋效应** | 自己的股票更值钱 | 高估持仓价值 |

### 常见借口

```
"不卖就不算亏" → 账面亏损也是真金白银

"总会涨回来的" → 有些永远涨不回来（柯达、安然）

"已经跌这么多了" → 跌90%后还能再跌90%

"我研究过的，会涨" → 市场不在乎你的研究

"再等等看" → 等待通常只会更糟
```

## 止损的数学逻辑

### 亏损越大，回本越难

```
亏损幅度 vs 回本需求：

亏损 10% → 需要涨 11%
亏损 20% → 需要涨 25%
亏损 30% → 需要涨 43%
亏损 40% → 需要涨 67%
亏损 50% → 需要涨 100%
亏损 60% → 需要涨 150%
亏损 70% → 需要涨 233%
亏损 80% → 需要涨 400%
亏损 90% → 需要涨 900%
```

### 早止损，保存资本

```
情景对比：

方案A：亏损 20% 止损
- 剩余本金：$80
- 回本需要：涨 25%
- 难度：中等

方案B：死扛到亏 60%
- 剩余本金：$40
- 回本需要：涨 150%
- 难度：极难
- 同时错失其他机会

结论：早止损，保存弹药，才能重新出发。
```

## 什么时候该止损

### 规则一：投资逻辑改变

```
买入时的假设不再成立：

□ 公司基本面恶化
□ 竞争优势消失
□ 管理层变化且不利
□ 行业结构性衰退
□ 出现会计问题

例：
- 买入时：公司是行业龙头
- 现在：市场份额持续下滑
- 行动：止损，不等"反弹"
```

### 规则二：技术止损

```
基于价格的机械止损：

保守型：
- 跌破买入价 10% 止损
- 跌破重要支撑位止损

进取型：
- 跌破买入价 15-20% 止损
- 考虑更长期基本面

规则：
- 买入时就设定止损位
- 不要事后修改
- 机械执行
```

### 规则三：机会成本

```
问自己：

"如果我现在没有这只股票，
手里有这笔钱，
我还会买它吗？"

如果答案是"不"→ 应该卖出

资金永远有更好的去处。
```

## 止损的方法

### 方法一：硬止损

```
买入时设定，不修改：

例：
买入价：$100
止损价：$85 (15%)

执行：
- 跌到 $85，无条件卖出
- 不看新闻
- 不听分析
- 机械执行
```

### 方法二：移动止损

```
随着盈利上移止损位：

例：
买入价：$100
初始止损：$85

涨到 $120：
- 新止损：$100 (保本)

涨到 $140：
- 新止损：$120 (锁定利润)

规则：
- 止损只能上移，不能下移
- 确保不会从盈利变亏损
```

### 方法三：时间止损

```
持有一定时间没涨，卖出：

例：
- 买入时设定：12个月观察期
- 如果12个月后没有明显上涨
- 考虑卖出，重新评估

适用于：
- 催化剂明确的投资
- 事件驱动型投资
```

### 方法四：分批止损

```
不确定时，分批退出：

例：
持仓 $10,000

第一批：跌 10%，卖 1/3
第二批：跌 15%，再卖 1/3
第三批：跌 20%，卖剩余

好处：
- 如果反弹，还有仓位
- 如果继续跌，损失可控
```

## 不该止损的情况

### 情况一：短期波动

```
市场正常波动不应止损：

标普500历史：
- 年内回撤 10% 是常态
- 年内回撤 20% 每3-4年一次
- 如果每次回撤都止损，会被洗出市场

区分：
- 短期波动 ≠ 基本面变化
- 价格下跌 ≠ 投资失败
```

### 情况二：你更了解

```
当市场暂时误解时：

例：
- 公司发布利好，股价却下跌
- 短期财报不及预期，但长期战略不变
- 市场恐慌抛售优质公司

巴菲特：
"别人恐惧时我贪婪"

条件：
- 你真的比市场更了解
- 不是自我欺骗
```

### 情况三：长期投资的核心持仓

```
指数基金等核心资产：

- 不因短期波动止损
- 用"时间"而非"止损"管理风险
- 持有期限：10年以上

例：
2020年3月标普暴跌 34%
- 止损 → 错过反弹
- 持有 → 1年后创新高
```

## 案例分析

### 案例一：正确的止损

**安然 (Enron) 2001年**

```
时间线：
- 2000年：股价 $90
- 2001年初：开始有负面报道
- 2001年中：CFO被调查
- 2001年10月：股价 $30
- 2001年12月：破产，股价 $0.20

如果在 $70 止损（跌 22%）：
- 保住了 $70
- 避免了归零

如果死扛：
- $90 → $0.20
- 亏损 99.8%
```

### 案例二：不该止损

**苹果 2013年**

```
时间线：
- 2012年9月：$100（分拆调整）
- 2013年4月：$56（跌 44%）
- 担忧：iPhone 增长见顶

如果在 2013 年止损：
- 卖在 $56

如果持有到 2024年：
- 股价 $190+
- 涨幅 3.4 倍

区别：
- 苹果的竞争优势没变
- 只是短期增长放缓
- 不应止损
```

### 如何区分

| 应该止损 | 不该止损 |
|---------|---------|
| 基本面恶化 | 只是短期波动 |
| 竞争优势消失 | 护城河仍在 |
| 会计问题 | 只是增长放缓 |
| 管理层诚信问题 | 只是市场恐慌 |
| 结构性衰退 | 周期性低谷 |

## 止损清单

### 买入时

```
□ 设定明确的止损位
□ 明确投资逻辑
□ 知道什么情况会卖
□ 把止损写下来
```

### 持有时

```
□ 定期检查投资逻辑是否成立
□ 是否接近止损位
□ 是否有更好的机会
□ 资金是否有更好的用途
```

### 考虑止损时

```
□ 投资逻辑是否已变？
□ 如果现在没有，会买吗？
□ 持有的理由是什么？
□ 是在"希望"还是"知道"？
```

## 本章总结

### 止损的核心

| 原则 | 说明 |
|-----|------|
| **承认错误** | 止损不是失败，是纠错 |
| **保存资本** | 本金在，机会在 |
| **机会成本** | 资金可以去更好的地方 |
| **机械执行** | 不要让情绪干预 |

### 何时止损，何时不止

| 止损 | 不止损 |
|-----|-------|
| 投资逻辑改变 | 只是短期波动 |
| 基本面恶化 | 护城河仍在 |
| 有更好机会 | 长期信念不变 |
| 触及预设止损 | 市场暂时误解 |

### 核心心法

> "小敌之坚，大敌之擒也。"

小亏不认，就会变成大亏。
承认错误，才能保存实力。
活着，才能等到下一次机会。
