## 孙子论"正奇"

孙子说：作战的基本原则是**以正兵交战，以奇兵取胜**。

| 概念 | 原意 | 投资映射 |
|-----|------|---------|
| **正兵** | 正面对敌的主力 | 核心持仓 |
| **奇兵** | 出奇制胜的奇袭 | 机会持仓 |
| **预备** | 随时待命的后备 | 现金储备 |

---

## 60-70-30 资产配置法则

### 基本框架

```
资产配置 = 正兵 + 奇兵 + 预备

正兵（60-70%）：核心持仓
├── 指数基金 (40-50%)
└── 价值蓝筹 (20%)

奇兵（20-30%）：机会持仓
├── 成长股 (10-15%)
├── 周期股 (5-10%)
└── 特殊机会 (5%)

预备（10-20%）：现金储备
├── 货币基金
└── 短期国债
```

### Graham 的建议

> "防守型投资者应该将资金分配在高等级债券和普通股之间，股票占比在 25%-75% 之间浮动。"

**简化版**：

| 市场状态 | 股票比例 | 债券/现金比例 |
|---------|---------|--------------|
| 极度低估 | 75% | 25% |
| 正常估值 | 50% | 50% |
| 极度高估 | 25% | 75% |

## 正兵：核心持仓

### 什么是核心持仓

核心持仓是投资组合的**主力军**，特点：

| 特征 | 说明 |
|-----|------|
| 稳定 | 不受短期波动影响 |
| 长期 | 持有期限 5-10 年以上 |
| 低换手 | 几乎不调整 |
| 分散 | 覆盖多个行业 |

### 核心持仓的构建

**方法一：指数基金为主**

```
推荐配置：

美股指数 (40-50%)
├── VTI (全美股市) 或
├── VOO (S&P 500) 或
└── VIG (股息增长)

国际股票 (10-15%)
├── VXUS (除美国外全球)
└── VWO (新兴市场)

债券 (10-20%)
└── BND (全美债券)
```

**方法二：个股精选**

| 类型 | 数量 | 占比 | 选股标准 |
|-----|------|------|---------|
| 大型价值股 | 8-10只 | 30% | P/E < 15, 分红 > 20年 |
| 消费龙头 | 3-5只 | 15% | 品牌护城河 |
| 金融龙头 | 2-3只 | 10% | ROE > 15% |
| 科技龙头 | 2-3只 | 10% | 现金流充沛 |

## 奇兵：机会持仓

### 什么是机会持仓

机会持仓是投资组合的**突击队**，特点：

| 特征 | 说明 |
|-----|------|
| 灵活 | 随市场调整 |
| 短期 | 持有期限 1-3 年 |
| 择时 | 在特定机会出现时介入 |
| 集中 | 单只可占更大比例 |

### 机会持仓的类型

**类型一：周期股抄底**

```
时机：行业低谷
目标：周期性行业龙头

例：
2020年3月：能源股 (XOM, CVX)
2022年底：科技股 (META, GOOG)
2023年：银行股 (JPM, BAC)

规则：
- 仅在 P/B < 1 时考虑
- 持有至估值恢复正常
- 占比不超过 10%
```

**类型二：事件驱动**

```
类型：
- 分拆 (Spin-off)
- 合并套利
- 清算价值股

例：
2021年：VMware 从 Dell 分拆
2023年：GE 一分为三

规则：
- 需要专业研究
- 持有期限明确
- 单只不超过 5%
```

**类型三：成长股仓位**

```
时机：成长股估值合理时

标准：
- PEG < 1
- 收入增长 > 20%
- 有明确的护城河

占比：
- 总体不超过 15%
- 单只不超过 5%
```

## 预备：现金储备

### 为什么需要现金

> "机会总是留给有准备的人。"

| 用途 | 说明 |
|-----|------|
| **防守** | 应对市场下跌 |
| **进攻** | 抓住突然出现的机会 |
| **心理** | 有钱在手，心不慌 |
| **生活** | 应对紧急支出 |

### 现金储备的比例

| 市场状态 | 现金比例 | 原因 |
|---------|---------|------|
| 极度低估 | 5% | 满仓进攻 |
| 正常估值 | 10-15% | 保持灵活 |
| 高估 | 20-30% | 等待机会 |
| 泡沫 | 30-50% | 大幅防守 |

### 巴菲特的现金管理

```
伯克希尔的现金规则：

永远保留：至少 $300亿 现金（约 10% 资产）

实际操作：
- 2008年：$370亿 → 抄底高盛、GE
- 2020年：$1370亿 → 等待机会
- 2022年：$1050亿 → 抄底石油股

现金既是防守武器，也是进攻弹药。
```

## 仓位管理的具体规则

### 规则一：单只股票上限

```
防守型投资者：
- 单只股票 < 5%
- 任何单一行业 < 15%

进取型投资者：
- 单只股票 < 10%
- 核心持仓单只 < 8%
- 机会持仓单只 < 5%

绝对规则：
- 永远不要单只 > 15%
- 除非你是巴菲特
```

### 规则二：加仓与减仓

**加仓规则**：

```
场景一：股价下跌

如果：
- 基本面没变
- 估值更便宜
- 占比未超限

可以：分批加仓（每跌 10% 加一次）

限制：
- 总占比不超过 8%
- 有明确止损线
```

**减仓规则**：

```
场景一：估值过高
- P/E 超过历史均值 50%
- P/B > 3
- 分批减仓

场景二：基本面恶化
- 竞争力下降
- 管理层变化
- 会计问题

场景三：更好的机会
- 机会成本考虑
- 调整到更好的标的
```

### 规则三：再平衡

**定期再平衡**：

```
频率：每年 1-2 次

方法：
1. 检查各资产类别占比
2. 如偏离目标 > 5%，进行调整
3. 卖出超配，买入低配

例：
目标：股票 60% / 债券 40%
当前：股票 70% / 债券 30%
行动：卖出 10% 股票，买入债券
```

**阈值再平衡**：

```
规则：当偏离度 > 20% 时调整

例：
目标股票占比：50%
触发点：< 40% 或 > 60%
```

## 不同人生阶段的配置

### 年轻人（25-40岁）

```
正兵 (80%)
├── 指数基金：60%
└── 成长股：20%

奇兵 (15%)
└── 高成长/高风险

预备 (5%)
└── 最低现金

理由：时间长，可承受波动
```

### 中年人（40-55岁）

```
正兵 (70%)
├── 指数基金：50%
└── 价值蓝筹：20%

奇兵 (15%)
└── 周期股/事件驱动

预备 (15%)
└── 增加现金缓冲

理由：平衡增长与安全
```

### 退休前（55-65岁）

```
正兵 (60%)
├── 指数基金：30%
├── 股息股：20%
└── 债券：10%

奇兵 (10%)
└── 保守机会

预备 (30%)
└── 高比例现金/债券

理由：保护本金，准备退休
```

## 实战案例

### 案例：2022年的仓位管理

```
2022年1月（市场高点）：
- 正兵：55%（已减配科技）
- 奇兵：15%（周期股、能源）
- 预备：30%（高现金）

2022年6月（市场下跌20%）：
- 开始加仓科技龙头
- 预备金降到 20%

2022年10月（市场底部）：
- 继续加仓
- 预备金降到 10%

2023年结果：
- 科技股反弹 50%+
- 正奇配合得当
```

## 本章总结

### 正奇预备的核心

| 类型 | 占比 | 特点 | 作用 |
|-----|------|------|------|
| 正兵 | 60-70% | 稳定、长期 | 主力收益 |
| 奇兵 | 20-30% | 灵活、择时 | 超额收益 |
| 预备 | 10-20% | 安全、流动 | 防守进攻 |

### 核心要点

1. **正兵为主**：核心持仓占主导，提供稳定收益
2. **奇兵为辅**：机会持仓提供超额收益
3. **预备不可少**：现金是防守和进攻的武器
4. **动态平衡**：根据市场状态调整比例
5. **纪律执行**：遵守仓位上限规则
