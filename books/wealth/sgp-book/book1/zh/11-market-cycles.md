## 周期无处不在

### 自然界的周期

```
日夜交替 → 24小时
四季轮回 → 12个月
月亮盈亏 → 29天
潮汐涨落 → 12小时
```

### 经济与市场的周期

```
经济周期 → 7-10年
信贷周期 → 5-7年
房地产周期 → 18-20年
股市周期 → 4-7年
行业周期 → 各不相同
```

## Howard Marks 论周期

### 橡树资本的周期大师

> "我们不知道周期会去哪里，但我们知道周期存在，并且我们可以利用它。"
>
> — Howard Marks

Howard Marks 是橡树资本的联合创始人，他的《周期》一书是理解市场周期的经典。

### 周期的特征

```
周期的基本规律：

1. 均值回归
   - 极端必将回归中间
   - 过高必跌，过低必涨

2. 过度反应
   - 上涨时涨过头
   - 下跌时跌过头

3. 心理驱动
   - 乐观推动上涨
   - 悲观推动下跌

4. 忘记历史
   - "这次不一样"
   - 每次都有人这么说
```

---

## 市场周期的四个阶段

### 阶段一：底部积累

```
特征：
- 估值处于历史低位
- 新闻全是坏消息
- 没人想买股票
- 机构在悄悄买入

心理：
- 极度悲观
- "股市永远不会涨"
- 投降式抛售

正确做法：
- 这是最佳买点
- 分批建仓
- 贪婪于恐惧
```

### 阶段二：上升期

```
特征：
- 价格开始上涨
- 经济数据改善
- 盈利开始增长
- 更多人开始关注

心理：
- 从悲观转向谨慎乐观
- 怀疑"是否是反弹"
- 逐渐接受"牛市来了"

正确做法：
- 持有核心仓位
- 享受上涨
- 不要过早卖出
```

### 阶段三：顶部狂热

```
特征：
- 估值处于历史高位
- 新闻全是好消息
- 人人都在谈股票
- 新手涌入市场

心理：
- 极度乐观
- "这次不一样"
- FOMO（害怕错过）

正确做法：
- 这是卖出时机
- 逐步减仓
- 恐惧于贪婪
```

### 阶段四：下跌期

```
特征：
- 价格开始下跌
- 经济数据恶化
- 盈利开始下滑
- 人们开始恐慌

心理：
- 从乐观转向恐惧
- "要跌到哪里？"
- 恐慌性抛售

正确做法：
- 等待底部信号
- 准备现金
- 耐心等待
```

## 如何判断周期位置

### 估值指标

| 指标 | 底部 | 正常 | 顶部 |
|-----|------|------|------|
| **S&P 500 P/E** | < 12 | 15-18 | > 22 |
| **CAPE（席勒市盈率）** | < 15 | 16-25 | > 30 |
| **股息率** | > 4% | 2-3% | < 2% |
| **Buffett 指标** | < 70% | 80-100% | > 120% |

### 情绪指标

| 指标 | 底部 | 顶部 |
|-----|------|------|
| **VIX** | > 30 | < 15 |
| **CNN恐惧贪婪指数** | < 20 | > 80 |
| **散户情绪** | 极度悲观 | 极度乐观 |
| **IPO数量** | 极少 | 创纪录 |

### 周期检查清单

```
底部信号（贪婪时机）：
□ P/E < 历史平均 30%
□ VIX > 30
□ 媒体全是悲观报道
□ 朋友都在说"股票危险"
□ 投资论坛冷清
□ 基金赎回创新高

顶部信号（恐惧时机）：
□ P/E > 历史平均 50%
□ VIX < 15
□ 媒体全是牛市报道
□ 朋友都在说买了什么股票
□ 投资论坛火爆
□ IPO 创纪录
```

## 历史周期回顾

### 美股主要周期

| 底部 | 顶部 | 周期长度 | 涨幅 |
|-----|------|---------|------|
| 1982 | 2000 | 18年 | 1400% |
| 2009 | 2021 | 12年 | 500% |
| 2002 | 2007 | 5年 | 100% |

### 主要底部

```
1932年：大萧条底部
- 道琼斯从 381 跌到 41
- P/E < 6
- "美国完了"
- 后续：20年牛市

1974年：石油危机底部
- S&P 500 跌 48%
- P/E = 7
- "滞胀永远不会结束"
- 后续：牛市开始

1982年：利率周期底部
- 利率超过 20%
- P/E = 8
- "股票已死"
- 后续：18年大牛市

2009年：金融危机底部
- S&P 500 跌 57%
- 银行差点倒闭
- "资本主义崩溃"
- 后续：12年牛市
```

### 主要顶部

```
1929年：大萧条前顶部
- "永久繁荣"
- 杠杆投机疯狂
- 后续：跌 89%

2000年：科技泡沫顶部
- "新经济"
- 纳斯达克 P/E = 175
- 后续：跌 78%

2007年：金融危机前顶部
- "房价永远涨"
- 金融创新
- 后续：跌 57%

2021年：疫情后顶部
- "科技改变一切"
- SPAC 狂热
- Meme 股疯狂
- 后续：成长股跌 50-80%
```

## 利用周期的策略

### 策略一：资产配置调整

```
根据周期位置调整股债比例：

周期底部：
- 股票 70-80%
- 债券/现金 20-30%

周期正常：
- 股票 50-60%
- 债券/现金 40-50%

周期顶部：
- 股票 30-40%
- 债券/现金 60-70%
```

### 策略二：行业轮动

```
经济周期中的行业表现：

衰退期 → 防守行业（公用事业、医疗、消费必需品）
复苏初期 → 周期行业（金融、工业、可选消费）
扩张期 → 成长行业（科技、通信）
过热期 → 商品行业（能源、原材料）
```

### 策略三：现金管理

```
周期底部：
- 用光现金储备
- 满仓进攻

周期正常：
- 保持正常现金比例（10-20%）

周期顶部：
- 增加现金储备（30%+）
- 等待机会
```

## 周期投资的陷阱

### 陷阱一：择时失败

```
问题：
- 永远不知道确切的底部在哪
- 可能太早或太晚

解决：
- 分批买入/卖出
- 不追求完美择时
- 只在极端时行动
```

### 陷阱二：反应过度

```
问题：
- 周期判断错误
- 频繁调仓亏损

解决：
- 只在极端估值时调整
- 保持核心仓位稳定
- 调整幅度适度（20-30%）
```

### 陷阱三："这次不一样"

```
每个周期顶部都有人说：
- 2000年："新经济改变一切"
- 2007年："风险分散了"
- 2021年："科技改变估值逻辑"

事实：
- 每次都是一样的
- 估值终会回归
- 周期不会消失
```

## 本章总结

### 周期的核心规律

| 规律 | 说明 |
|-----|------|
| **周期必然** | 周期一定会发生 |
| **极端回归** | 极端必将回归中间 |
| **情绪驱动** | 贪婪和恐惧交替 |
| **历史重演** | 但具体形式不同 |

### 实操建议

```
1. 了解自己在周期的哪里
2. 极端时才大幅调仓
3. 分批操作，不追求完美
4. 保持核心仓位稳定
5. "这次不一样"是危险信号
```

### Howard Marks 的智慧

> "我们不能预测周期，但我们可以准备。
> 当市场走极端时，逆向而行通常是对的。"
