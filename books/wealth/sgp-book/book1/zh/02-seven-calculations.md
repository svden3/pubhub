---

> "主孰有道？将孰有能？天地孰得？法令孰行？兵众孰强？士卒孰练？赏罚孰明？吾以此知胜负矣。"

## 孙子的"七计"

孙子通过七个问题来比较敌我双方：

| 七计 | 原意 | 投资映射 |
|-----|------|---------|
| **主孰有道** | 谁的君主更得民心 | 谁的治理更好 |
| **将孰有能** | 谁的将领更有才能 | 谁的管理层更强 |
| **天地孰得** | 谁占天时地利 | 谁的外部环境更有利 |
| **法令孰行** | 谁的法令更严明 | 谁的执行力更强 |
| **兵众孰强** | 谁的军队更强大 | 谁的资源更充足 |
| **士卒孰练** | 谁的士兵更训练有素 | 谁的团队更专业 |
| **赏罚孰明** | 谁的赏罚更分明 | 谁的激励机制更好 |

## 一、主孰有道 — 治理质量比较

### 评估指标

| 指标 | 5分 | 3分 | 1分 |
|-----|-----|-----|-----|
| 管理层持股 | > 10% | 3-10% | < 1% |
| 独立董事比例 | > 50% | 30-50% | < 30% |
| 关联交易 | 几乎没有 | 偶尔有 | 频繁 |
| 股东回报政策 | 清晰一致 | 时有时无 | 不透明 |

### 案例：可口可乐 vs 百事

| 维度 | 可口可乐 | 百事 |
|-----|---------|------|
| 专注度 | 纯饮料业务 | 饮料+食品 |
| 资本配置 | 回购+分红为主 | 多元化收购 |
| 股东信 | 坦诚直接 | 较多公关语言 |
| 评分 | 4.5 | 3.5 |

## 二、将孰有能 — 管理层比较

### 评估维度

| 维度 | 关键问题 |
|-----|---------|
| **战略眼光** | 5年前的预测准确吗？ |
| **执行能力** | 说到做到吗？ |
| **资本配置** | ROE 趋势如何？ |
| **危机处理** | 上次危机处理得如何？ |
| **人才吸引** | 能吸引顶尖人才吗？ |

### 量化指标

| 指标 | 优秀 | 及格 | 差 |
|-----|------|------|-----|
| CEO 任期内 ROE | > 15% | 10-15% | < 10% |
| 承诺完成率 | > 80% | 50-80% | < 50% |
| 员工留存率 | > 90% | 70-90% | < 70% |
| Glassdoor 评分 | > 4.0 | 3.0-4.0 | < 3.0 |

## 三、天地孰得 — 外部环境比较

### 天：宏观环境

| 因素 | 有利 | 中性 | 不利 |
|-----|------|------|------|
| 利率趋势 | 下降 | 稳定 | 上升 |
| 监管环境 | 放松 | 稳定 | 收紧 |
| 技术变革 | 受益者 | 不受影响 | 被颠覆者 |

### 地：竞争地位

| 因素 | 有利 | 中性 | 不利 |
|-----|------|------|------|
| 市场份额趋势 | 上升 | 稳定 | 下降 |
| 定价能力 | 强 | 中等 | 弱 |
| 供应商议价 | 强势 | 平衡 | 弱势 |

---

## 四、法令孰行 — 执行效率比较

### 运营指标对比

| 指标 | 公式 | 意义 |
|-----|------|------|
| **库存周转** | 销售成本 / 平均库存 | 运营效率 |
| **应收账款周转** | 营收 / 平均应收 | 收款效率 |
| **资产周转率** | 营收 / 平均资产 | 资产利用率 |
| **SG&A/营收** | 销售管理费用 / 营收 | 成本控制 |

### 案例：沃尔玛 vs 西尔斯 (2010)

| 指标 | 沃尔玛 | 西尔斯 |
|-----|--------|--------|
| 库存周转 | 8.5x | 3.2x |
| SG&A/营收 | 19% | 28% |
| 同店销售增长 | +2% | -5% |
| 执行力评分 | 5 | 2 |

**结果**：西尔斯2018年破产，沃尔玛市值持续增长。

## 六、士卒孰练 — 团队专业度比较

### 人才指标

| 维度 | 评估方法 |
|-----|---------|

|-----|------|------|

| 创新速度 | 20%时间创新制度 | 官僚化严重 |
| 产品质量 | 搜索最优 | 门户思维 |
| 评分 | 5 | 2.5 |

## 七、赏罚孰明 — 激励机制比较

### 激励结构分析

| 维度 | 好的设计 | 差的设计 |
|-----|---------|---------|
| 高管薪酬 | 与长期业绩挂钩 | 固定高薪 |
| 股权激励 | 有锁定期和业绩条件 | 无条件赠予 |
| 绩效考核 | 客观量化指标 | 主观评价为主 |
| 问责机制 | 失败有后果 | 只奖不罚 |

### 红旗信号 🚩

- 业绩下滑但高管薪酬上涨
- 短期激励比重过大
- 金降落伞条款过于慷慨
- 回购时机与高管卖出重合

## 七计实战应用

### 步骤一：确定可比公司

选择 3-5 家主要竞争对手进行比较。

### 步骤二：填写评分表

| 七计 | 权重 | 公司A | 公司B | 公司C |
|-----|------|-------|-------|-------|
| 道 | 15% | | | |
| 将 | 20% | | | |
| 天 | 10% | | | |
| 地 | 20% | | | |
| 法 | 15% | | | |
| 兵 | 10% | | | |
| 练 | 10% | | | |
| **加权总分** | 100% | | | |

### 步骤三：解读结果

| 相对排名 | 投资建议 |
|---------|---------|
| 第一名且绝对分 > 4 | 考虑投资 |
| 第一名但绝对分 < 4 | 行业可能不好 |
| 非第一名 | 除非有特殊理由，否则选行业龙头 |

## 案例：2020年电动车三国演义

### 比较对象

- Tesla（特斯拉）
- NIO（蔚来）
- XPeng（小鹏）

### 七计评分 (2020年)

| 七计 | Tesla | NIO | XPeng |
|-----|-------|-----|-------|
| 道 | 4 | 3 | 3 |
| 将 | 5 | 4 | 4 |
| 天 | 5 | 4 | 4 |
| 地 | 5 | 3 | 3 |
| 法 | 4 | 3 | 3 |
| 兵 | 4 | 2 | 2 |
| 练 | 5 | 3 | 3 |
| **总分** | **4.5** | **3.1** | **3.1** |

### 分析

- **Tesla**：全面领先，尤其在技术(练)和品牌(地)
- **NIO/XPeng**：中国市场有优势，但整体实力差距大
- **结论**：同样投资电动车，Tesla 是更安全的选择

### 后续表现 (2020-2024)

| 公司 | 2020年股价 | 2024年股价 | 变化 |
|-----|-----------|-----------|------|
| Tesla | ~$100 | ~$250 | +150% |
| NIO | ~$5 | ~$5 | 0% |
| XPeng | ~$20 | ~$10 | -50% |

**七计分析验证了结果。**

## 本章总结

孙子的"七计"提供了一个系统的竞争对手比较框架：

| 七计 | 核心问题 | 关键指标 |
|-----|---------|---------|

| 兵 | 谁的弹药更多？ | 现金、负债率 |
| 练 | 谁的团队更专业？ | 人才密度、创新能力 |

> "知彼知己，百战不殆。"
