## 情绪是投资的头号敌人

### 人类的情绪本能

```
进化赋予我们的本能：

恐惧：看到狮子 → 立即逃跑
贪婪：看到食物 → 立即抢夺
从众：跟随族群 → 更安全

这些本能帮助祖先在荒野生存。
但在投资市场，它们是毁灭性的。
```

### 市场中的情绪陷阱

| 情绪 | 触发 | 导致行为 | 后果 |
|-----|------|---------|------|
| 恐惧 | 市场下跌 | 恐慌卖出 | 卖在底部 |
| 贪婪 | 市场上涨 | 追高买入 | 买在顶部 |
| 从众 | 别人都在买 | 跟风 | 泡沫中被套 |
| 后悔 | 错过上涨 | FOMO 追入 | 高位接盘 |
| 愤怒 | 亏损 | 报复性交易 | 亏损加剧 |

## 常见情绪陷阱

### 一、恐惧 (Fear)

```
表现：
- 市场下跌时恐慌
- "天要塌了"
- 不顾价格卖出
- 发誓永不碰股票

心理机制：
- 损失厌恶（亏损的痛是盈利快乐的2倍）
- 短期思维（只看眼前）
- 从众心理（别人都在卖）

解药：
- 记住：恐慌往往是机会
- 问：基本面变了吗？
- 有计划：预先设定应对方案
```

### 二、贪婪 (Greed)

```
表现：
- 市场上涨时亢奋
- "这次不一样"
- 加杠杆追高
- 觉得自己是天才

心理机制：
- 过度自信
- 近因偏误（最近的经历权重过大）
- 社交炫耀（朋友圈晒收益）

解药：
- 记住：没有只涨不跌的市场
- 问：估值合理吗？
- 有纪律：设定卖出规则
```

### 三、FOMO (Fear of Missing Out)

```
表现：
- 看到别人赚钱就眼红
- "我也要上车"
- 不顾价格买入
- 后悔"怎么没早买"

心理机制：
- 社交比较
- 损失厌恶（害怕错过）
- 羊群效应

解药：
- 记住：永远有下一次机会
- 问：这是我的能力圈吗？
- 有耐心：等待自己的机会
```

### 四、沉没成本谬误

```
表现：
- "已经亏这么多了，不能卖"
- 加仓摊薄成本
- 死扛到底
- 无视新的证据

心理机制：
- 厌恶"确认"亏损
- 觉得沉没成本能回来
- 承认错误太痛苦

解药：
- 记住：过去的成本无法改变
- 问：如果现在没有，会买吗？
- 有勇气：该止损就止损
```

### 五、确认偏误

```
表现：
- 只看支持自己观点的信息
- 忽略反面证据
- "分析师不懂"
- 找理由合理化持仓

心理机制：
- 保护自尊
- 减少认知失调
- 懒得重新思考

解药：
- 记住：反面观点也可能对
- 问：反对者怎么说？
- 有开放：主动寻找反面证据
```

## 情绪控制的方法

### 方法一：预先计划

```
在平静时制定规则：

买入规则：
- 什么条件触发买入
- 分几批买入
- 每批买多少

卖出规则：
- 什么条件触发卖出
- 止损设在哪里
- 盈利多少开始减仓

调仓规则：
- 多久检查一次
- 什么触发再平衡
- 极端情况怎么办

然后：机械执行，不临时改动
```

### 方法二：设置冷却期

```
任何重大决策前，等待 24-48 小时。

规则：
- 想买入：等 24 小时
- 想卖出：等 24 小时
- 想大幅调仓：等 48 小时

目的：
- 让情绪冷却
- 给理性思考时间
- 避免冲动决策

例外：
- 已经触发预设的止损
- 已经计划好的操作
```

### 方法三：减少刺激

```
减少看盘频率：

建议频率：
- 长期投资者：每周或每月看一次
- 中期投资者：每天看一次
- 不要实时盯盘

减少新闻消费：
- 不看财经电视
- 不刷投资论坛
- 只读高质量的分析

好处：
- 减少情绪波动
- 减少冲动交易
- 生活更健康
```

### 方法四：写投资日记

```
记录每一笔交易：

买入时记录：
- 买入理由
- 预期目标
- 什么情况会卖出
- 当时的情绪

卖出时记录：
- 卖出理由
- 实际收益/亏损
- 与预期对比
- 当时的情绪

定期回顾：
- 什么决策是对的
- 什么决策是错的
- 情绪如何影响了我
- 如何改进
```

### 方法五：建立支持系统

```
找到可以讨论的人：

好的讨论伙伴：
- 有投资经验
- 思维理性
- 能提出反对意见
- 不只会附和

避免：
- 只会说"买买买"的人
- 只会传播恐惧的人
- 推荐"内幕消息"的人
```

## 情绪控制清单

### 买入前

```
□ 我是因为分析还是因为 FOMO？
□ 我等了 24 小时了吗？
□ 这是我能力圈内的吗？
□ 我有完整的投资论文吗？
□ 如果跌 30% 我能承受吗？
```

### 持有时

```
□ 我是不是看盘太频繁了？
□ 我是因为短期波动焦虑吗？
□ 我的投资逻辑变了吗？
□ 我在"希望"还是在"分析"？
```

### 卖出前

```
□ 我是因为恐惧还是因为理性？
□ 我等了 24 小时了吗？
□ 基本面变了吗？
□ 这是触发了预设的止损吗？
□ 如果卖了，钱会投到哪里？
```

## 案例分析

### 案例：2020年3月的两种投资者

```
投资者 A（情绪驱动）：

2月底：
- 看到新闻恐慌
- "疫情要毁灭一切"
- 全部卖出

3月底：
- 市场开始反弹
- "会二次探底"
- 继续等待

年底：
- 市场创新高
- "等回调再买"
- 踏空整年

结果：卖在底部，错过反弹

---

投资者 B（纪律驱动）：

2月底：
- 看到市场下跌
- 检查基本面
- 保持仓位

3月底：
- VIX 创新高
- "恐惧时贪婪"
- 分批买入

年底：
- 持续持有
- 收益 30%+

结果：遵循纪律，获得收益
```

## 本章总结

### 情绪控制的核心

| 情绪 | 危害 | 对策 |
|-----|------|------|
| 恐惧 | 卖在底部 | 预先计划 |
| 贪婪 | 买在顶部 | 设定规则 |
| FOMO | 追高 | 保持耐心 |
| 沉没成本 | 死扛 | 勇于止损 |
| 确认偏误 | 忽视风险 | 寻找反面 |

### 五大方法

```
1. 预先计划 —— 在平静时做决策
2. 冷却期 —— 等 24-48 小时
3. 减少刺激 —— 少看盘，少看新闻
4. 投资日记 —— 记录和反思
5. 支持系统 —— 找理性的讨论伙伴
```

### 核心心法

> "主不可以怒而兴师，将不可以愠而致战。"

不要在愤怒时做决定。
不要在恐惧时卖出。
不要在狂热时买入。

情绪平静时做的决定，才是好决定。
