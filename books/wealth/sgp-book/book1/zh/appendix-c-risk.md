## 投资组合风险检查

### 每月检查

```
组合概览：

总资产：$_____________
现金比例：______%
股票比例：______%
债券比例：______%
房产比例：______%
其他：______%

与目标配置对比：
□ 偏离 < 5%：正常
□ 偏离 5-10%：考虑再平衡
□ 偏离 > 10%：需要调整
```

### 集中度检查

```
单一股票集中度：

最大持仓：__________ 占比 ____%
第二大持仓：__________ 占比 ____%
第三大持仓：__________ 占比 ____%

检查：
□ 单一股票 < 5%？
□ 前 5 大持仓 < 25%？

如果集中度过高：
→ 考虑减仓
→ 或增加其他持仓

---

单一行业集中度：

行业分布：
科技：____%
金融：____%
医疗：____%
消费：____%
工业：____%
能源：____%
其他：____%

检查：
□ 单一行业 < 20%？

如果行业集中度过高：
→ 增加其他行业暴露
→ 或减少过度集中的行业
```

### 相关性检查

```
持仓之间的相关性：

高度相关的风险：
- 同一行业的多只股票
- 同一地区的多项投资
- 同一周期阶段的资产

检查清单：
□ 有没有"一损俱损"的风险？
□ 经济衰退时哪些会同时下跌？
□ 利率上升时哪些会受影响？

例子：
如果你持有：
- 苹果、微软、谷歌（都是科技大盘股）
- 高度相关，实际上不够分散

更好的分散：
- 苹果（科技）
- 强生（医疗）
- 宝洁（消费）
- JPMorgan（金融）
```

## 单一投资风险检查

### 买入前风险检查

```
买入 [股票名称] 前：

业务风险：
□ 主要业务风险是什么？
□ 竞争对手威胁有多大？
□ 技术变化风险？
□ 监管风险？
□ 客户集中风险？

财务风险：
□ 负债风险？
□ 流动性风险？
□ 货币风险？
□ 利率风险？

估值风险：
□ 如果盈利低于预期 20%，股价会跌多少？
□ 如果 P/E 回到历史平均，股价会跌多少？
□ 最坏情况下可能亏损多少？

我能承受的最大亏损：____%
最坏情况预估亏损：____%

如果最坏情况 > 你能承受的 → 不买或减少仓位
```

### 持有期风险监控

```
每季度检查：

基本面风险信号：
□ 收入下滑？
□ 利润率恶化？
□ 市场份额流失？
□ 管理层变动？
□ 诉讼或监管问题？

财务风险信号：
□ 负债增加？
□ 现金流恶化？
□ 资本支出大幅增加？
□ 融资需求增加？

估值风险信号：
□ 估值是否变得很高？
□ 股价涨幅是否远超基本面？

如果出现多个风险信号 → 重新评估，考虑减仓
```

## 杠杆风险检查

### 杠杆使用检查

```
当前杠杆情况：

投资组合：
□ 是否使用保证金贷款？
□ 保证金使用率：____%
□ 维持保证金要求：____%

房产：
□ 总房产价值：$_____________
□ 总房产贷款：$_____________
□ LTV 比率：____%

其他债务：
□ 信用卡债务：$_____________
□ 个人贷款：$_____________
□ 学生贷款：$_____________

---

杠杆风险评估：

保守（低风险）：
□ 投资组合无杠杆
□ 房产 LTV < 60%
□ 无高息债务

中等风险：
□ 投资组合杠杆 < 1.5x
□ 房产 LTV 60-75%
□ 债务可控

高风险：
□ 投资组合杠杆 > 1.5x
□ 房产 LTV > 75%
□ 有大额高息债务

如果在高风险区 → 制定降杠杆计划
```

### 强制平仓风险

```
检查被迫卖出的风险：

保证金账户：
□ 距离强平线还有多少缓冲？
□ 如果市场跌 20%，会被强平吗？
□ 有现金可以追加吗？

房产贷款：
□ 有浮动利率贷款吗？
□ 利率上升 2% 还能负担吗？
□ 有贷款即将到期吗？

个人现金流：
□ 紧急基金有多少个月支出？
□ 如果失业，能撑多久？
□ 会被迫卖投资吗？

如果存在被迫卖出风险 → 立即降低风险敞口
```

## 情绪风险检查

### 当前情绪状态

```
市场情绪：

□ VIX 水平：_____
   - < 15：平静，可能自满
   - 15-25：正常
   - 25-35：紧张
   - > 35：恐慌

□ 新闻情绪：
   - 极度乐观
   - 乐观
   - 中性
   - 悲观
   - 极度悲观

□ 周围人的情绪：
   - 大家都在谈股票（警惕）
   - 没人关心股市（可能是机会）

---

你的情绪状态：

最近感受：
□ 焦虑
□ 贪婪
□ 恐惧
□ FOMO
□ 平静
□ 自信过度

如果情绪激动：
→ 不做任何决策
→ 等 24-48 小时
→ 回顾投资计划
```

### 行为偏误检查

```
检查自己是否有这些偏误：

过度自信：
□ 觉得自己比市场聪明？
□ 忽视反对意见？
□ 集中投资少数股票？

确认偏误：
□ 只看支持自己观点的信息？
□ 忽略负面消息？
□ 拒绝承认错误？

近因偏误：
□ 因为最近涨了就更乐观？
□ 因为最近跌了就更悲观？
□ 过度重视短期表现？

损失厌恶：
□ 因为怕承认亏损而死扛？
□ 过早卖出盈利的持仓？
□ 不愿意止损？

如果发现有偏误 → 暂停决策，寻求外部意见
```

## 系统性风险检查

### 宏观风险评估

```
经济周期：

当前位置判断：
□ 早期复苏
□ 中期扩张
□ 晚期扩张
□ 衰退

风险指标：
□ 收益率曲线：正常/平坦/倒挂
□ 失业率趋势：下降/稳定/上升
□ 企业盈利趋势：增长/稳定/下滑
□ 通胀水平：低/中/高

如果多个指标指向晚期或衰退：
→ 考虑增加现金比例
→ 减少高风险资产
→ 避免杠杆
```

### 黑天鹅准备

```
极端事件准备：

问自己：
□ 如果市场明天跌 30%，我会怎么做？
□ 如果某个持仓归零，我能承受吗？
□ 如果失业 6 个月，投资会受影响吗？
□ 如果房市跌 30%，我的房产投资会怎样？

准备措施：
□ 紧急基金 6-12 个月支出
□ 投资组合分散化
□ 不过度使用杠杆
□ 有备用收入来源
□ 保险覆盖充分

心理准备：
□ 接受黑天鹅会发生
□ 有预定的应对计划
□ 不会恐慌卖出
```

## 风险行动计划

### 如果发现高风险

```
降低风险的步骤：

1. 识别最大风险
   - 集中度？
   - 杠杆？
   - 单一投资？

2. 制定计划
   - 卖出多少？
   - 分几次卖？
   - 卖出后资金去哪？

3. 执行
   - 按计划行动
   - 不临时改变
   - 记录过程

4. 后续监控
   - 确认风险降低
   - 持续监控
```

### 风险检查频率

```
日常：
□ 不需要（除非特殊情况）

每周：
□ 快速浏览持仓
□ 有无重大新闻

每月：
□ 组合概览
□ 集中度检查
□ 情绪状态

每季度：
□ 完整风险检查
□ 再平衡评估
□ 宏观环境评估

每年：
□ 全面审视
□ 调整策略
□ 更新计划
```
