## 危机概述

### 发生了什么

```
2008年金融危机时间线：

2007年：
- 次贷违约开始上升
- 房价开始下跌
- 对冲基金开始爆仓

2008年3月：
- 贝尔斯登被收购

2008年9月15日：
- 雷曼兄弟破产
- 危机全面爆发

2008年10月：
- S&P 500 单月跌 17%
- VIX 创历史新高 80

2009年3月：
- S&P 500 触底 666
- 从高点跌了 57%
```

### 影响有多大

```
股市：
- S&P 500：-57%（2007.10 - 2009.3）
- 道琼斯：-54%
- 纳斯达克：-56%

房市：
- 全美房价：-33%
- 部分地区：-50% 以上

经济：
- 失业率：从 5% 升到 10%
- 银行倒闭：465 家（2008-2012）
- GDP：-4.3%（2009）

个人：
- 数百万人失去房子
- 数万亿美元财富蒸发
- 退休计划被迫推迟
```

## 投资者的反应

### 恐慌卖出者

```
典型行为：
- 2008年9-10月恐慌卖出
- "这次不一样，世界要崩溃"
- 卖在底部附近
- 错过后来的反弹

结果：
如果在 2009年3月卖出：
- 锁定了 -57% 的亏损
- S&P 500 后来涨了 6 倍
- 完美地卖在底部

教训：
恐慌时卖出，往往是最差的决策。
```

### 坚持持有者

```
典型行为：
- 承受了巨大的账面亏损
- 没有卖出
- 继续定投
- 等待复苏

结果：
如果从 2007年10月持有到 2024年：
- S&P 500：+400% 以上
- 年化收益：约 10%

教训：
长期持有，最终会复苏。
但需要钢铁般的意志。
```

### 逆向投资者

```
典型行为：
- 2008年底开始买入
- 2009年初加仓
- "别人恐惧时我贪婪"

结果：
如果在 2009年3月买入 S&P 500：
- 到 2024年：+700%
- 年化收益：约 15%

巴菲特的行动：
- 2008年10月写文章《买美国》
- 大量买入高盛、GE 等
- 获得优惠条款

教训：
危机是最好的买入机会。
但需要勇气和现金。
```

## S-G-P 框架分析

### 孙子视角

```
"兵无常势，水无常形。"

危机的本质：
- 市场情绪极端化
- 价格严重偏离价值
- 恐惧压倒理性

孙子的应对：
1. 预先准备（有现金储备）
2. 不被情绪左右
3. 在混乱中寻找机会
4. 等待形势逆转
```

### Graham 视角

```
"市场先生" 理论：

2008年的市场先生：
- 极度恐惧
- 愿意以任何价格卖出
- 给出荒谬的低价

Graham 会怎么做：
1. 检查安全边际
   - 很多优质公司跌到了清算价值以下

2. 分散投资
   - 不知道哪些公司会活下来

3. 逐步买入
   - 不猜底，分批建仓

4. 等待价值回归
   - 市场最终会恢复理性
```

### 实践启示

```
危机中该做什么：

1. 检查投资组合
   □ 有没有会破产的公司
   □ 有没有过度杠杆
   □ 现金储备够不够

2. 不要恐慌卖出
   □ 除非基本面彻底改变
   □ 除非需要用钱
   □ 否则持有

3. 考虑买入
   □ 如果有现金
   □ 如果有勇气
   □ 选择最优质的公司

4. 继续定投
   □ 不停止定投计划
   □ 危机时买入成本更低
   □ 长期受益
```

## 具体案例

### 案例一：银行股

```
2008-2009年银行股：

倒闭的：
- 雷曼兄弟（破产）
- 华盛顿互惠（破产）
- 华科维亚（被收购）

活下来的：
- 摩根大通：从 $50 跌到 $15，后涨到 $200
- 富国银行：从 $35 跌到 $8，后涨到 $50
- 高盛：从 $200 跌到 $47，后涨到 $400

教训：
1. 有些公司会破产——分散很重要
2. 活下来的会大涨——不要全部卖出
3. 难以预测谁会活——买行业 ETF 更安全
```

### 案例二：科技股

```
2008-2009年科技股：

苹果：
- 从 $28 跌到 $11（-60%）
- 后涨到 $200+（18倍）

亚马逊：
- 从 $90 跌到 $34（-62%）
- 后涨到 $3000+（88倍）

谷歌：
- 从 $350 跌到 $125（-64%）
- 后涨到 $2800+（22倍）

教训：
优质科技公司在危机中跌得很惨，
但恢复后涨得更多。
关键是持有优质公司。
```

### 案例三：房地产

```
2008-2012年房市：

全美：
- 房价跌 33%
- 2012年触底

加州：
- 部分地区跌 50%+
- 法拍房大量涌现

聪明的投资者：
- 2010-2012年买入法拍房
- 以重置成本 50% 的价格买入
- 出租收益率 10%+
- 房价后来涨了 2-3 倍

教训：
房产危机是最好的买入机会，
但需要现金和耐心。
```

## 从危机中学习

### 什么有效

```
有效的策略：

1. 分散投资
   - 不会因单一公司破产而归零
   - 组合整体更稳定

2. 现金储备
   - 有钱才能抄底
   - 不会被迫卖出

3. 长期视角
   - 知道危机会过去
   - 不被短期恐慌影响

4. 定投
   - 危机时买入更多份额
   - 平均成本更低
```

### 什么无效

```
无效的策略：

1. 全部卖出
   - 卖在底部
   - 错过反弹

2. 杠杆投资
   - 被强制平仓
   - 没有机会等反弹

3. 集中持仓
   - 如果持有雷曼，归零
   - 没有分散保护

4. 择时交易
   - 无法准确预测底部
   - 往往卖在低点买在高点
```

## 为下一次危机做准备

### 危机会再来

```
历史规律：
- 1987年黑色星期一
- 2000年互联网泡沫
- 2008年金融危机
- 2020年新冠暴跌
- 未来还会有...

平均每 7-10 年一次大调整。

问题不是"会不会来"，
而是"你准备好了吗"。
```

### 准备清单

```
现在就准备：

资金准备：
□ 紧急基金 6-12 个月支出
□ 投资组合有 10-20% 现金/债券
□ 没有高息债务

心理准备：
□ 接受波动是正常的
□ 预期会有 50% 的跌幅
□ 制定危机应对计划

策略准备：
□ 分散的投资组合
□ 不用杠杆
□ 定投计划
□ 再平衡规则

当危机来临时：
□ 不恐慌卖出
□ 检查持仓质量
□ 考虑加仓
□ 坚持定投
```

## 本章总结

### 2008危机的教训

| 教训 | 说明 |
|-----|------|
| 恐慌卖出 | 锁定亏损，错过反弹 |
| 坚持持有 | 最终复苏，需要意志 |
| 逆向买入 | 最佳机会，需要现金和勇气 |
| 分散投资 | 保护免于破产 |

### 核心原则

```
1. 危机是机会的另一面
2. 现金是危机时的弹药
3. 分散保护你免于灾难
4. 长期视角帮你度过恐惧
5. 纪律让你在混乱中保持理性
```

### 孙子的智慧

> "兵无常势，水无常形。"

市场没有固定的形态。
危机会来，也会过去。
能够随势而变的投资者，
才能在危机中生存并获利。
