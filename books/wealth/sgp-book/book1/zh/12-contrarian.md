## 孙子论"不随众"

孙子说：善于作战的人，**调动敌人而不被敌人调动**。

投资中，这意味着：**主动选择，不随大众**。

| 原文 | 直译 | 投资映射 |
|-----|------|---------|
| 致人而不致于人 | 调动别人，不被调动 | 自己思考，不随大众 |

## 巴菲特论"恐惧与贪婪"

### 最著名的投资格言

> "在别人贪婪时恐惧，在别人恐惧时贪婪。"
>
> — Warren Buffett

```
大众行为          你应该做的
━━━━━━━━━━━━━━━━━━━━━━━━━━━━
贪婪（追高）   →   恐惧（减仓）
恐惧（抛售）   →   贪婪（买入）
```

### 为什么逆向有效

```
市场是人组成的，人会：

情绪化
├── 乐观时过度乐观
├── 悲观时过度悲观
└── 推动价格偏离价值

羊群效应
├── 别人买我也买
├── 别人卖我也卖
└── 造成极端价格

短期思维
├── 只看眼前
├── 忽视长期
└── 错误定价

逆向投资就是利用这些人性弱点。
```

## 逆向投资的条件

### 不是什么都要逆向

```
逆向投资 ≠ 盲目逆向

正确的逆向：
✓ 基本面支持
✓ 估值有吸引力
✓ 市场过度反应
✓ 有安全边际

错误的逆向：
✗ 只因为跌了就买
✗ 不了解公司
✗ 抄底"落刀"
✗ 忽视结构性问题
```

### 区分"暂时困难"与"结构性衰退"

| 暂时困难 | 结构性衰退 |
|---------|-----------|
| 短期利空 | 长期趋势恶化 |
| 竞争力不变 | 护城河消失 |
| 可以恢复 | 无法逆转 |
| 值得逆向买入 | 不应该逆向 |

**例子**：

```
暂时困难（值得逆向）：
- 2008年金融危机中的苹果
- 2020年疫情中的迪士尼
- 产品召回导致的短期下跌

结构性衰退（不要逆向）：
- 柯达面对数码相机
- 诺基亚面对智能手机
- 传统零售面对电商
```

## 逆向投资的实践

### 情景一：市场恐慌

```
信号：
- VIX > 30
- 媒体全是末日预言
- 朋友都在说"卖光了"
- 基金大规模赎回

行动：
1. 检查手头现金
2. 列出想买的优质公司
3. 分批买入
4. 持有等待恢复

历史案例：
- 2009年3月
- 2020年3月
- 2022年10月
```

### 情景二：市场狂热

```
信号：
- VIX < 15
- 媒体全是牛市预言
- 朋友都在说赚了多少
- IPO 热潮

行动：
1. 检查持仓估值
2. 逐步减仓高估的
3. 增加现金储备
4. 等待下次机会

历史案例：
- 2000年3月
- 2021年年初
```

### 情景三：行业被抛弃

```
信号：
- 整个行业估值创新低
- 分析师纷纷下调评级
- "这个行业没有未来"

行动：
1. 研究行业是否有结构性问题
2. 如果只是周期，考虑龙头
3. 分批建仓
4. 耐心持有

历史案例：
- 2020年能源股
- 2016年银行股
- 2023年中概股
```

## 经典逆向案例

### 案例一：巴菲特买高盛

```
背景：
- 2008年9月：雷曼倒闭
- 市场恐慌到极点
- "资本主义要完了"

巴菲特的行动：
- 投资高盛 $50亿
- 条款：10%优先股股息 + 认股权

市场反应：
- "巴菲特疯了"
- "银行都要倒闭"

结果：
- 高盛存活
- 巴菲特获利超 $50亿

心法：
- 当优质公司陷入恐慌
- 价格远低于价值
- 这就是逆向的时机
```

### 案例二：2020年3月

```
背景：
- 新冠疫情爆发
- 全球封锁
- S&P 500 一个月跌 34%

恐惧者：
- "世界末日"
- 清仓离场
- 等待更低

贪婪者：
- "优质公司不会消失"
- 分批买入
- AAPL, MSFT, GOOG

结果：
- 9个月后创新高
- 最快的熊市复苏
```

### 案例三：逆向失败——安然

```
背景：
- 2001年：安然股价暴跌
- 从 $90 跌到 $30

逆向买入者：
- "跌这么多，是机会"
- 不了解会计问题
- 买入抄底

结果：
- 安然破产
- 股价归零
- 逆向失败

教训：
- 不是跌了就值得买
- 要区分暂时困难 vs 结构性问题
- 会计问题是红线
```

## 逆向投资的心理建设

### 为什么逆向这么难

```
逆向意味着：
- 当所有人都在卖时买入
- 承受"你疯了"的质疑
- 短期可能继续亏损
- 独自承担决策后果

这需要：
- 独立思考能力
- 对自己分析的信心
- 承受短期痛苦的能力
- 长期视角
```

### 如何培养逆向思维

```
1. 建立自己的分析框架
   - 不依赖别人的观点
   - 有自己的估值方法
   - 能独立做出判断

2. 研究历史
   - 每次恐慌后都恢复了
   - 每次泡沫最终都破了
   - 历史会重演

3. 控制情绪
   - 知道自己会害怕
   - 预先制定计划
   - 机械执行

4. 从小仓位开始
   - 先用小钱练习
   - 积累经验
   - 逐步增加仓位
```

## 逆向投资检查清单

### 考虑买入时

```
市场情绪检查：
□ 市场是否极度恐惧？
□ 媒体是否一片悲观？
□ 大众是否在抛售？

基本面检查：
□ 公司的竞争优势还在吗？
□ 是暂时困难还是结构性问题？
□ 管理层是否可信？

估值检查：
□ 估值是否处于历史低位？
□ 安全边际是否足够？
□ 最坏情况下能承受吗？

执行检查：
□ 分批买入还是一次性？
□ 仓位占比合理吗？
□ 有没有止损计划？
```

### 考虑卖出时

```
市场情绪检查：
□ 市场是否极度贪婪？
□ 媒体是否一片乐观？
□ 大众是否在追高？

估值检查：
□ 估值是否处于历史高位？
□ 安全边际是否消失？
□ "这次不一样"的声音多吗？

执行检查：
□ 分批卖出还是一次性？
□ 减仓到什么比例？
□ 留多少现金等待机会？
```

## 本章总结

### 逆向的核心

| 原则 | 说明 |
|-----|------|
| **别人恐惧时贪婪** | 市场恐慌是买入机会 |
| **别人贪婪时恐惧** | 市场狂热是卖出信号 |
| **不是盲目逆向** | 需要基本面支持 |
| **需要耐心** | 价值回归需要时间 |

### 孙子的智慧

> "致人而不致于人。"

不被市场情绪带着走，主动选择自己的位置。

### 逆向的三个条件

```
1. 市场情绪走极端
2. 基本面支持
3. 估值有安全边际

三者缺一不可。
```
