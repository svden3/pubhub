## 为什么需要纪律

### 市场的特点

```
市场每天都在诱惑你：
- 热门股票的上涨
- 专家的预测
- 朋友的"内幕"
- 新闻的噪音

没有纪律，你会：
- 追涨杀跌
- 频繁交易
- 偏离策略
- 最终亏损
```

### 纪律的作用

```
纪律是：
- 预先制定的规则
- 不受情绪影响
- 一致性执行
- 长期制胜的关键

纪律帮助你：
- 避免情绪化决策
- 减少交易成本
- 保持策略一致
- 获得复利效应
```

## 投资纪律的核心原则

### 原则一：只在能力圈内投资

```
规则：
- 只投资你真正理解的公司
- 能用简单语言解释商业模式
- 知道公司如何赚钱
- 了解行业的竞争格局

检验方法：
- 能不能向10岁孩子解释清楚
- 能不能写出完整的投资论文
- 知不知道主要风险在哪

例外：
- 指数基金不需要深入理解个股
- 但要理解资产配置的逻辑
```

### 原则二：永远保持安全边际

```
规则：
- 只在有安全边际时买入
- 至少 30% 的折扣
- 估值高时不买

实践：
- P/E < 15 或行业平均
- P/B < 1.5
- 股息率有吸引力

不妥协：
- 再好的公司，贵了也不买
- 宁可错过，不可买贵
```

### 原则三：分散但不过度

```
规则：
- 持股 20-30 只
- 单只股票 < 5%
- 单一行业 < 20%
- 保持现金储备

不做的事：
- 全仓单只股票
- 持有 100 只股票
- 集中于单一行业
```

### 原则四：长期持有

```
规则：
- 默认持有期：5年以上
- 不因短期波动卖出
- 只在特定条件下卖出

卖出条件：
- 投资逻辑改变
- 估值过高
- 有更好的机会
- 触发止损

不卖的情况：
- 价格下跌但逻辑不变
- 短期财报不及预期
- 市场整体下跌
```

### 原则五：定期检查，及时调整

```
规则：
- 季度检查持仓
- 年度再平衡
- 持续学习和改进

检查内容：
- 各持仓的投资逻辑还成立吗
- 资产配置是否偏离目标
- 是否有需要止损的
- 是否有更好的机会

调整原则：
- 不频繁调整
- 有明确的理由
- 分批执行
```

## 建立你的投资系统

### 第一步：定义你的策略

```
回答以下问题：

1. 我的投资目标是什么？
   - 退休储蓄
   - 财务自由
   - 孩子教育

2. 我的风险承受能力？
   - 能承受多大回撤
   - 多长的投资期限
   - 这笔钱什么时候用

3. 我的投资方法？
   - 主动还是被动
   - 价值还是成长
   - 集中还是分散

写下来，形成书面策略。
```

### 第二步：制定具体规则

```
买入规则：
□ 只买 ___ 行业的公司
□ P/E < ___
□ ROE > ___
□ 连续盈利 ___ 年
□ 分红 ___ 年以上
□ 初次建仓 ___%
□ 分 ___ 批买入

持有规则：
□ 每 ___ 个月检查一次
□ 单只占比超过 ___% 减仓
□ 年度再平衡

卖出规则：
□ 止损线：-___%
□ 估值上限：P/E > ___
□ 基本面恶化时卖出
□ 分 ___ 批卖出
```

### 第三步：建立检查清单

```
买入检查清单：
□ 在能力圈内？
□ 通过财务检查？
□ 有安全边际？
□ 仓位规划合理？
□ 写下投资论文？

持有检查清单：
□ 投资逻辑还成立？
□ 基本面有变化吗？
□ 估值还合理吗？
□ 仓位需要调整吗？

卖出检查清单：
□ 有明确的卖出理由？
□ 不是因为情绪？
□ 卖出后资金去哪？
```

### 第四步：执行与记录

```
交易日志：
- 日期
- 操作（买/卖）
- 标的
- 价格
- 数量
- 理由
- 当时情绪

定期回顾：
- 每季度回顾交易
- 分析对错
- 总结经验
- 优化规则
```

## 纪律的敌人

### 敌人一：过度自信

```
表现：
- "我比市场聪明"
- "这次不一样"
- 忽视风险

对策：
- 保持谦卑
- 记住过去的错误
- 控制仓位
```

### 敌人二：短期思维

```
表现：
- 每天看账户
- 被日波动影响
- 频繁交易

对策：
- 减少看盘
- 关注年度回报
- 坚持长期策略
```

### 敌人三：社交压力

```
表现：
- 朋友在炫耀收益
- 论坛都在谈某只股票
- FOMO 心理

对策：
- 不讨论短期收益
- 远离噪音
- 专注自己的策略
```

### 敌人四：信息过载

```
表现：
- 看太多新闻
- 太多"专家"观点
- 无法决策

对策：
- 限制信息来源
- 只看高质量分析
- 相信自己的系统
```

## 案例：投资系统示例

```
某投资者的系统：

身份：40岁，中产，长期投资

目标：
- 20年后退休
- 每年增长 8-10%
- 能承受 30% 回撤

策略：
- 60% 股票 / 30% 债券 / 10% 现金
- 股票以美股指数 + 精选个股
- 每年再平衡一次

买入规则：
- 指数基金：每月定投
- 个股：P/E < 18，ROE > 12%，分红 > 10年
- 初次建仓 2%，分 3 批

卖出规则：
- 个股止损：-20%
- 估值过高：P/E > 25
- 每年审视一次

纪律：
- 每季度检查持仓
- 每年再平衡
- 不因短期波动操作
- 交易日志记录每笔交易
```

## 本章总结

### 纪律的五大原则

| 原则 | 说明 |
|-----|------|
| 能力圈 | 只投资你理解的 |
| 安全边际 | 永远要有折扣 |
| 分散 | 20-30只，不过度 |
| 长期持有 | 5年以上视角 |
| 定期检查 | 季度检查，年度调整 |

### 建立系统的四步

```
1. 定义策略 —— 写下投资哲学
2. 制定规则 —— 具体的买卖规则
3. 建立清单 —— 检查清单辅助决策
4. 执行记录 —— 日志和定期回顾
```

### 核心心法

> "兵者，诡道也。"

市场充满诱惑和陷阱。
纪律是你的盔甲。
系统是你的武器。

有纪律的平庸策略，胜过无纪律的天才想法。
