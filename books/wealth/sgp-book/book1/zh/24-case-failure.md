---

> "知己知彼，百战不殆；不知彼而知己，一胜一负；不知彼，不知己，每战必殆。"
>
> — 《孙子兵法·谋攻篇》

## 为什么研究失败

### 成功有运气，失败有规律

```
成功的投资：
- 可能是能力
- 也可能是运气
- 难以区分

失败的投资：
- 几乎都有共同原因
- 可以总结规律
- 更容易学习

查理·芒格：
"我只想知道我会死在哪里，
然后我就不去那里。"

研究失败，是为了避免失败。
```

## 失败案例集

### 案例一：安然（Enron）

```
背景：
- 曾是全美第七大公司
- 2000年市值 700亿美元
- 被《财富》评为"最受尊敬的公司"

问题：
- 会计造假
- 隐藏债务
- 复杂的关联交易
- 管理层欺诈

结局：
- 2001年破产
- 股价从 $90 跌到 $0.26
- 员工养老金全部归零
- 高管被判刑

红旗信号（事后看）：
- 业务模式难以理解
- 利润来源不清晰
- CFO 频繁变更
- 高管大量卖股

教训：
□ 不投资你不理解的公司
□ 警惕复杂的会计
□ 管理层诚信很重要
□ 如果太好到不真实，可能不真实
```

### 案例二：雷曼兄弟

```
背景：
- 158年历史的投资银行
- 2007年市值 600亿美元
- 华尔街五大投行之一

问题：
- 极端杠杆（30:1以上）
- 集中在房地产相关资产
- 流动性管理不善
- 过度自信

结局：
- 2008年9月破产
- 历史上最大的破产案
- 股价从 $67 跌到 $0.06
- 引发全球金融危机

红旗信号：
- 资产负债表高杠杆
- 对单一领域过度暴露
- 管理层否认问题
- 同行开始出问题

教训：
□ 杠杆是双刃剑
□ 集中持仓风险巨大
□ 流动性危机致命
□ 不要相信"大到不能倒"
```

### 案例三：诺基亚

```
背景：
- 曾占全球手机市场 40%
- 2007年市值 1500亿美元
- 似乎坚不可摧

问题：
- 忽视智能手机趋势
- 操作系统落后
- 组织僵化
- 过度自信

结局：
- 2013年手机业务卖给微软
- 股价从 $60 跌到 $5
- 从行业领导者变成边缘角色

红旗信号：
- iPhone 发布后仍不重视
- 市场份额开始流失
- 竞争对手快速崛起
- 管理层仍在否认

教训：
□ 护城河可以被填平
□ 技术变革颠覆一切
□ 傲慢是衰落的开始
□ 要持续观察竞争格局
```

### 案例四：长期资本管理公司（LTCM）

```
背景：
- 两位诺贝尔奖得主
- "史上最聪明的投资团队"
- 前几年年化收益 40%

问题：
- 极端杠杆（25:1）
- 过度相信模型
- 集中押注
- 忽视尾部风险

结局：
- 1998年四个月亏损 46亿美元
- 需要美联储协调救助
- 公司倒闭

红旗信号：
- 杠杆越来越高
- 收益来源不清晰
- 没有考虑极端情况
- 过度自信

教训：
□ 聪明人也会犯错
□ 模型有局限性
□ 黑天鹅事件会发生
□ 杠杆放大一切，包括错误
```

### 案例五：个人投资者的典型错误

```
案例 A：追涨杀跌的小张

2021年1月：
- 听朋友说 GameStop 暴涨
- FOMO，在 $300 买入
- "这次不一样"

2021年2月：
- 股价跌到 $40
- 恐慌卖出
- 亏损 87%

---

案例 B：全仓 meme 股的老刘

2021年：
- 把所有积蓄投入 meme 股
- AMC、GME、BBBY
- 不分散，不止损

2022年：
- 大部分 meme 股跌 80%+
- 账户缩水 75%
- 后悔莫及

---

案例 C：杠杆交易的小王

2020年3月：
- 用 3 倍杠杆做多
- 市场继续下跌
- 被强制平仓

结果：
- 如果没有杠杆，持有到年底赚 70%
- 但因为杠杆，亏了 80%
```

## 失败的共同模式

### 模式一：过度杠杆

```
问题：
- 放大收益，也放大亏损
- 可能被强制平仓
- 没有时间等待恢复

案例：雷曼、LTCM、个人杠杆交易

规则：
□ 不用杠杆，或最多 1.5x
□ 确保不会被强制平仓
□ 永远有现金储备
```

### 模式二：过度集中

```
问题：
- 一只股票出问题就归零
- 没有风险分散
- "鸡蛋都在一个篮子"

案例：安然员工、单一行业投资者

规则：
□ 单只股票 < 5%
□ 单一行业 < 20%
□ 至少 20-30 只股票
```

### 模式三：忽视警示信号

```
问题：
- 不愿接受坏消息
- 合理化负面信息
- 等到太晚才行动

案例：诺基亚投资者、安然投资者

规则：
□ 主动寻找负面信息
□ 问"如果我错了呢"
□ 设定止损规则
```

### 模式四：不理解投资标的

```
问题：
- 不知道公司做什么
- 不知道钱怎么赚的
- 只因为别人推荐

案例：安然、复杂金融产品

规则：
□ 能用简单语言解释
□ 知道风险在哪里
□ 在能力圈内投资
```

### 模式五：情绪化交易

```
问题：
- 恐惧时卖出
- 贪婪时买入
- 追涨杀跌

案例：2020年3月卖出者、meme股追高者

规则：
□ 预先制定计划
□ 不因情绪改变策略
□ 设置冷却期
```

## 如何从失败中学习

### 第一步：承认错误

```
承认是改变的开始：

错误的态度：
- "市场不公平"
- "我只是运气不好"
- "分析师误导了我"

正确的态度：
- "我的决策有问题"
- "我没有做足功课"
- "我被情绪控制了"

只有承认错误，才能改进。
```

### 第二步：分析原因

```
问五个"为什么"：

例子：我在 GameStop 上亏了钱

为什么亏钱？
→ 因为我在高点买入

为什么在高点买入？
→ 因为 FOMO

为什么 FOMO？
→ 因为朋友都在赚钱

为什么朋友影响了我？
→ 因为我没有自己的投资计划

为什么没有计划？
→ 因为我没有认真学习投资

根本原因：缺乏投资知识和纪律
```

### 第三步：制定规则

```
把教训变成规则：

教训：不要追涨
规则：不买过去一个月涨超 20% 的股票

教训：不要集中
规则：单只股票最多 5%

教训：不要用杠杆
规则：永远不使用保证金

写下规则，贴在看得见的地方。
```

### 第四步：执行检查

```
定期回顾：

每月检查：
□ 这个月遵守规则了吗
□ 有没有情绪化决策
□ 有没有犯老错误

每年回顾：
□ 今年的错误是什么
□ 需要增加什么规则
□ 规则执行得如何

持续改进，永不停止。
```

## 本章总结

### 失败的五大模式

| 模式 | 案例 | 防范规则 |
|-----|------|---------|
| 过度杠杆 | 雷曼、LTCM | 不用或低杠杆 |
| 过度集中 | 安然员工 | 分散投资 |
| 忽视警示 | 诺基亚 | 主动寻找反面 |
| 不理解 | 复杂产品 | 能力圈内 |
| 情绪化 | meme股 | 预先计划 |

### 从失败中学习的步骤

```
1. 承认错误 —— 不找借口
2. 分析原因 —— 问五个为什么
3. 制定规则 —— 把教训变规则
4. 执行检查 —— 定期回顾
```

### 核心心法

> "不知彼，不知己，每战必殆。"

知己 = 知道自己的弱点
知彼 = 知道市场的风险

研究失败，是为了不重蹈覆辙。
承认错误，是为了持续进步。

最好的投资者不是不犯错，
而是从错误中学习最多的人。
