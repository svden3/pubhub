# 第六章：形兵之极 — 分散与集中

---

> "形兵之极，至于无形。无形则深间不能窥，智者不能谋。"
>
> — 《孙子兵法·虚实篇》

---

## 孙子论"形兵之极"

孙子说：用兵的最高境界是**无形**——让敌人看不透你的部署。

在投资中，这意味着：**合理分散，让市场的任何单一风险都无法击溃你**。

| 原文 | 投资映射 |
|-----|---------|
| 形兵之极，至于无形 | 分散到无法被单一风险击倒 |
| 深间不能窥 | 不依赖任何单一信息 |
| 智者不能谋 | 市场无法预测你的弱点 |

---

## Graham 的分散原则

### 20-30 股原则

> "足够的分散化意味着持有不同行业的 20-30 只股票。"
>
> — Benjamin Graham

**为什么是 20-30 只？**

```
分散的边际效益：

持股数    风险降低（相对单只）
1         0%
5         50%
10        70%
20        85%
30        90%
50        92%
100       95%

20-30只已经捕获了大部分分散效益。
再多，管理成本上升，边际收益递减。
```

### 分散的三个维度

| 维度 | 说明 | 目标 |
|-----|------|------|
| **数量** | 持股数量 | 20-30只 |
| **行业** | 行业分布 | 5-7个行业 |
| **地域** | 地理分布 | 2-3个市场 |

---

## 分散投资的真正含义

### 分散 ≠ 持有一堆相关的股票

**错误的分散**：

```
以为分散了：
- 苹果
- 微软
- Google
- Meta
- Amazon

实际：全是科技股，高度相关
2022年：全部一起跌 30-60%
```

**正确的分散**：

```
真正的分散：
- 科技：苹果、微软
- 金融：JPMorgan
- 医疗：强生
- 消费：宝洁、可口可乐
- 能源：埃克森
- 工业：3M

2022年：科技跌，能源涨，整体稳定
```

### 相关性是关键

| 组合类型 | 相关性 | 分散效果 |
|---------|--------|---------|
| 全科技股 | 0.8+ | 差 |
| 同行业不同公司 | 0.6-0.8 | 中 |
| 不同行业 | 0.3-0.5 | 好 |
| 股票 + 债券 | 0-0.3 | 很好 |
| 美股 + 黄金 | 负相关 | 最好 |

## 集中与分散的平衡

### 巴菲特论集中

> "分散是对无知的保护。如果你知道自己在做什么，分散就没有意义了。"
>
> — Warren Buffett

**巴菲特的做法**：

```
伯克希尔前五大持仓（2024年）：

苹果：45%
美国银行：10%
美国运通：8%
可口可乐：7%
雪佛龙：5%

前五大 = 75%

这是高度集中，但是：
- 巴菲特研究了几十年
- 对这些公司极其了解
- 有足够的财务实力承受波动
```

### 适合你的分散度

| 类型 | 持股数 | 单只上限 | 适合谁 |
|-----|--------|---------|--------|
| **高度分散** | 50+ | 2% | 完全被动投资者 |
| **Graham式** | 20-30 | 5% | 业余价值投资者 |
| **适度集中** | 10-20 | 10% | 认真研究的投资者 |
| **高度集中** | 5-10 | 20% | 专业投资者 |

**规则**：**研究深度与集中度成正比**。

## 地域分散

### 为什么需要地域分散

| 原因 | 说明 |
|-----|------|
| **单一市场风险** | 日本 1990 年至今还没回到高点 |
| **货币风险** | 美元可能贬值 |
| **增长机会** | 新兴市场增长更快 |
| **估值差异** | 不同市场估值不同 |

### 建议配置

```
美国投资者：

美国：60-70%
发达国际：15-25%（欧洲、日本）
新兴市场：5-15%（中国、印度）

中国投资者：

A股：40-50%
港股：20-30%
美股：20-30%
```

### 实现方式

| 方式 | 优点 | 缺点 |
|-----|------|------|

| **ADR** | 个股选择 | 有限选择 |
| **海外账户** | 完全自由 | 管理复杂 |

## 分散的误区

### 误区一：数量 = 分散

```
错误：持有100只科技股
正确：持有20只不同行业的股票

数量不重要，相关性才重要。
```

### 误区二：过度分散

```
问题：持有50只股票
- 无法深入研究每一只
- 收益接近指数，但费用更高
- 不如直接买指数基金

彼得·林奇：
"拥有50只股票的人，是不知道自己在做什么。"
```

### 误区三：假分散

```
以为分散了：
- 持有10只主动基金

实际：
- 这些基金可能持有相同的股票
- 都是"大盘成长"风格
- 费用还很高

解决：检查底层持仓的重叠度
```

## 本章总结

### 分散的核心原则

| 原则 | 说明 |
|-----|------|
| **20-30 股** | 数量上足够分散 |
| **5-7 行业** | 行业不过度集中 |
| **低相关性** | 资产之间相关性低 |
| **与能力匹配** | 研究深度决定集中度 |
| **简单为美** | 不要过度复杂化 |

### 形兵之极

> "形兵之极，至于无形。"

真正的分散是：
1. 任何单一公司出问题，你不会破产
2. 任何单一行业崩溃，你能承受
3. 任何单一国家危机，你有备选
4. 任何单一资产类别下跌，你仍然生存

**但同时**：
- 不要分散到无法管理
- 不要分散到失去超额收益
- 不要分散到变成昂贵的指数基金

> "分散是为了生存，集中是为了致富。"
