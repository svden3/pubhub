## 为什么要读财报

### Graham 的基本功

> "投资者的基本功是阅读财务报表。"
>
> — Benjamin Graham

财报是公司的"体检报告"。**不读财报就投资，如同不做体检就手术**。

---

## 三张核心报表

### 一、利润表 (Income Statement)

```
公司赚不赚钱？

收入 (Revenue)
  - 成本 (COGS)
  ─────────────
= 毛利润 (Gross Profit)
  - 运营费用 (Operating Expenses)
  ─────────────
= 营业利润 (Operating Income)
  - 利息和税
  ─────────────
= 净利润 (Net Income)

关键指标：
- 毛利率 = 毛利 / 收入
- 营业利润率 = 营业利润 / 收入
- 净利润率 = 净利 / 收入
```

### 二、资产负债表 (Balance Sheet)

```
公司有什么？欠什么？

资产 (Assets)
├── 流动资产 (Current Assets)
│   ├── 现金
│   ├── 应收账款
│   └── 存货
└── 非流动资产 (Non-current Assets)
    ├── 固定资产
    └── 无形资产

负债 (Liabilities)
├── 流动负债 (Current Liabilities)
│   └── 短期借款、应付账款
└── 长期负债 (Long-term Debt)

股东权益 (Shareholders' Equity)
= 资产 - 负债

关键指标：
- 流动比率 = 流动资产 / 流动负债
- 负债率 = 总负债 / 总资产
- 净资产 = 股东权益
```

### 三、现金流量表 (Cash Flow Statement)

```
公司现金从哪来？到哪去？

经营活动现金流 (Operating Cash Flow)
├── 日常运营产生的现金
└── 最重要的指标

投资活动现金流 (Investing Cash Flow)
├── 资本支出 (CapEx)
└── 收购、投资

融资活动现金流 (Financing Cash Flow)
├── 借款
├── 发行股票
└── 分红

自由现金流 (Free Cash Flow)
= 经营现金流 - 资本支出
= 真正可分配给股东的钱
```

## Graham 的七大财务检查

### 检查一：规模

```
标准：年收入 > $100M（中小公司可放宽）

原因：
- 规模提供稳定性
- 更容易获取信息
- 流动性更好
```

### 检查二：财务稳健

```
标准：
- 流动比率 > 2
- 长期负债 < 净流动资产

原因：
- 短期偿债能力强
- 长期财务安全
```

### 检查三：盈利稳定

```
标准：过去 10 年每年都盈利

原因：
- 证明商业模式可行
- 有能力度过经济周期
```

### 检查四：分红记录

```
标准：连续 20 年分红

原因：
- 管理层愿意回报股东
- 有稳定的现金流
```

### 检查五：盈利增长

```
标准：10 年盈利增长 > 1/3（年化 > 3%）

原因：
- 跑赢通胀
- 证明公司在成长
```

### 检查六：估值合理

```
标准：
- P/E < 15
- 或 P/E × P/B < 22.5

原因：
- 不为增长付出过高价格
- 留足安全边际
```

### 检查七：资产价值

```
标准：P/B < 1.5

原因：
- 资产提供保护
- 即使清算也有价值
```

## 关键指标深度解析

### 毛利率 (Gross Margin)

```
公式：毛利率 = (收入 - 成本) / 收入

解读：
- 反映定价能力和成本控制
- 越高越好
- 行业差异大

标杆：
- 软件：70-80%
- 消费品：40-60%
- 零售：20-30%
- 航空：10-20%

警示信号：
- 毛利率持续下降
- 低于行业平均
```

### 营业利润率 (Operating Margin)

```
公式：营业利润率 = 营业利润 / 收入

解读：
- 反映运营效率
- 扣除了运营费用后的盈利能力

标杆：
- 优秀：> 20%
- 良好：15-20%
- 一般：10-15%
- 较差：< 10%

警示信号：
- 持续低于行业平均
- 趋势向下
```

### 净资产收益率 ROE

```
公式：ROE = 净利润 / 股东权益

解读：
- 反映股东资本的回报率
- 巴菲特最看重的指标

标杆：
- 优秀：> 20%
- 良好：15-20%
- 一般：10-15%
- 较差：< 10%

杜邦分解：
ROE = 净利率 × 周转率 × 杠杆

注意：
- 高杠杆带来的高 ROE 要警惕
- 关注 ROE 的稳定性
```

### 自由现金流 (FCF)

```
公式：FCF = 经营现金流 - 资本支出

解读：
- 真正可分配给股东的钱
- 最难造假的指标

检查：
- FCF 是否持续为正
- FCF 是否与净利润接近
- FCF 是否在增长

警示信号：
- 净利润高但 FCF 低
- FCF 持续为负
- 大量资本支出
```

## 财务造假的红旗

### 红旗一：收入问题

```
警示信号：
- 收入增长远超行业
- 应收账款增长快于收入
- 季度末大量发货
- 退货率上升

可能问题：
- 虚增收入
- 提前确认收入
- 塞货给渠道
```

### 红旗二：现金流问题

```
警示信号：
- 净利润高但经营现金流低
- 资本化支出异常高
- 应付账款大幅增加
- 存货周转变慢

可能问题：
- 利润质量差
- 费用资本化
- 挤压供应商
- 产品滞销
```

### 红旗三：关联交易

```
警示信号：
- 大量关联方交易
- 复杂的公司结构
- 难以理解的商业模式
- 频繁变更审计师

可能问题：
- 利益输送
- 隐藏负债
- 虚构交易
```

### 红旗四：管理层问题

```
警示信号：
- CFO 离职
- 审计师更换
- 高管大量减持
- 业绩指引频繁调整

可能问题：
- 内部问题
- 财务压力
- 失去信心
```

## 实战案例

### 案例：分析苹果公司

```
关键数据（2023财年）：

收入：$383B
毛利率：44%（软件+硬件混合）
营业利润率：30%（极其优秀）
净利润：$97B
ROE：约 160%（主要靠回购）

现金流：
- 经营现金流：$110B
- 资本支出：$11B
- 自由现金流：$99B

资产负债：
- 现金：$62B
- 长期债务：$111B
- 但债务主要用于回购，成本极低

评估：
- 极强的盈利能力
- 巨额自由现金流
- 资产负债表健康
- 唯一问题：估值是否合理
```

## 财报分析清单

### 基本检查

```
□ 收入增长率（过去5年）
□ 毛利率趋势
□ 营业利润率趋势
□ ROE（过去5年平均）
□ 自由现金流（是否为正）
□ 负债率（是否过高）
```

### 质量检查

```
□ 经营现金流 ≥ 净利润？
□ 应收账款增速 < 收入增速？
□ 存货周转正常？
□ 资本支出合理？
□ 有无大额非经常性项目？
```

### 红旗检查

```
□ 审计意见是否清洁？
□ 关联交易是否合理？
□ 高管是否在增持或减持？
□ 业绩指引是否频繁调整？
□ 是否有难以理解的项目？
```
