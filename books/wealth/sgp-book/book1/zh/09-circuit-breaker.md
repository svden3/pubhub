# 第九章：熔断机制

---

> "主不可以怒而兴师，将不可以愠而致战。合于利而动，不合于利而止。"
>
> — 《孙子兵法·火攻篇》

---

## 孙子论"止"

孙子说：君主不可因愤怒而发动战争，将领不可因怨恨而作战。**合乎利益就行动，不合利益就停止**。

这是最高的纪律：**知道什么时候该停**。

| 原文 | 直译 | 投资映射 |
|-----|------|---------|
| 主不可以怒而兴师 | 不因愤怒而打仗 | 不因冲动而交易 |
| 将不可以愠而致战 | 不因怨恨而作战 | 不因报复心理加仓 |
| 合于利而动 | 有利才行动 | 符合策略才交易 |
| 不合于利而止 | 不利就停止 | 触发熔断就休息 |

---

## 什么是熔断机制

### 市场熔断

```
美股市场熔断规则：

Level 1：下跌 7%  → 暂停 15 分钟
Level 2：下跌 13% → 暂停 15 分钟
Level 3：下跌 20% → 当天收市

目的：
- 阻止恐慌蔓延
- 让市场冷静
- 防止踩踏
```

### 个人熔断

**你也需要自己的熔断机制：当事情失控时，强制停止。**

| 市场熔断 | 个人熔断 |
|---------|---------|
| 保护市场 | 保护你的账户 |
| 强制暂停 | 强制停止交易 |
| 冷静思考 | 冷静反思 |
| 防止踩踏 | 防止冲动决策 |

## 个人熔断的类型

### 类型一：单日熔断

```
当日亏损触发：

阈值：
- 黄灯：当日亏损 > 2%
- 红灯：当日亏损 > 5%
- 熔断：当日亏损 > 10%

行动：
- 黄灯：减少交易
- 红灯：停止新交易
- 熔断：当天不再看盘

规则：宁可错过，不可冲动
```

### 类型二：月度熔断

```
月度亏损触发：

阈值：
- 月亏损 > 5%：反思策略
- 月亏损 > 10%：暂停交易一周
- 月亏损 > 15%：暂停交易一个月

目的：
- 阻止连续亏损
- 避免"回本心理"
- 强制反思
```

### 类型三：交易频率熔断

```
交易次数触发：

阈值：
- 每周交易 > 3次：警示
- 每周交易 > 5次：熔断

为什么：
- 频繁交易往往是坏习惯
- 交易成本累积
- 往往是情绪化交易

例外：
- 有明确策略的再平衡
- 预先计划的建仓/减仓
```

### 类型四：情绪熔断

```
情绪状态触发：

当你感到：
- 极度兴奋 → 停止
- 极度恐惧 → 停止
- 想"报复"市场 → 停止
- 无法入睡 → 停止
- 不断刷新账户 → 停止

行动：
1. 关闭交易软件
2. 离开电脑/手机
3. 做其他事情
4. 第二天再决定
```

## 熔断后怎么办

### 步骤一：冷静下来

```
身体层面：
- 离开电脑
- 去散步
- 做运动
- 睡一觉

心理层面：
- 接受已经发生的事
- 不要责怪自己
- 不要急于"弥补"
```

### 步骤二：分析原因

```
问自己：

□ 是市场系统性下跌，还是我选错了？
□ 我有没有违反自己的规则？
□ 我是根据计划交易，还是冲动？
□ 我的仓位管理有问题吗？
□ 我有没有忽略警示信号？
```

### 步骤三：调整策略

```
根据分析，可能的调整：

仓位问题：
- 降低总体仓位
- 减少单只集中度

风控问题：
- 设置更严格的止损
- 增加现金比例

心理问题：
- 减少看盘频率
- 设置交易规则

策略问题：
- 反思投资逻辑
- 考虑是否需要改变
```

### 步骤四：恢复交易

```
熔断后的复出规则：

1. 至少等待规定的冷却期
2. 只用一半仓位开始
3. 从最保守的标的开始
4. 逐步恢复正常

例：
月度熔断后：
- 第一周：只观察，不交易
- 第二周：50% 仓位
- 第三周：75% 仓位
- 第四周：恢复正常
```

## 案例分析

### 案例一：2020年3月——没有熔断的灾难

```
Mike 的故事：

2020年2月：
- 满仓科技股
- 觉得"会反弹"

2020年3月初：
- 开始亏损
- 加仓"抄底"

2020年3月中：
- 亏损 30%
- 用杠杆加仓

2020年3月20日：
- 亏损 50%
- 被强制平仓

如果有熔断：
- 月亏损 15% 时停止
- 保住本金的 85%
- 等待真正的底部
```

### 案例二：有纪律的熔断

```
Sarah 的故事：

2020年2月末：
- 看到市场信号
- 减仓到 60%

2020年3月初：
- 月亏损触及 10%
- 启动熔断
- 不再交易

2020年3月中：
- 冷静观察
- 分析机会

2020年3月底：
- 确认底部信号
- 分批建仓

结果：
- 避开最大跌幅
- 抄到相对低点
- 2020年底盈利 40%
```

## 系统性风控框架

### 三层防护

```
第一层：仓位控制
├── 单只股票 < 5%
├── 单一行业 < 20%
├── 股票总仓位 < 70%
└── 保持现金储备

第二层：止损机制
├── 个股止损：-15%
├── 组合止损：-10%/月
└── 移动止损保护盈利

第三层：熔断机制
├── 日熔断：-5%
├── 月熔断：-15%
└── 情绪熔断
```

### 熔断恢复流程

```
触发熔断
    ↓
冷却期（1天-1周）
    ↓
原因分析
    ↓
策略调整
    ↓
减仓恢复（50%仓位）
    ↓
逐步恢复
    ↓
正常交易
```

## 本章总结

### 熔断的核心

| 原则 | 说明 |
|-----|------|
| **强制停止** | 当情况失控时，停下来 |
| **冷静反思** | 不在激动时做决定 |
| **保护本金** | 活着比赚钱更重要 |
| **纪律执行** | 规则是事先定的，不是事后改的 |

### 孙子的智慧

> "合于利而动，不合于利而止。"

知道什么时候该停，比知道什么时候该做更重要。

### 熔断的三个层次

```
1. 日熔断 —— 控制单日损失
2. 月熔断 —— 控制连续亏损
3. 情绪熔断 —— 控制冲动交易
```

### 最终心法

投资是马拉松，不是百米冲刺。
偶尔停下来休息，不是软弱。
是为了走更远的路。

> "怒可以复喜，愠可以复悦，亡国不可以复存，死者不可以复生。"

账户归零了，就没有第二次机会。
熔断，是给自己留一条后路。
