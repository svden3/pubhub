# 附录D：推荐书单

---

## 必读经典

### 价值投资基础

```
《聪明的投资者》
The Intelligent Investor
作者：Benjamin Graham
出版：1949年（多次修订）

为什么读：
- 价值投资的"圣经"
- 建立正确的投资心态
- 理解"市场先生"
- 学习安全边际概念

核心观点：
- 投资与投机的区别
- 防御型 vs 进取型投资者
- 市场先生的比喻
- 安全边际原则

建议：先读修订版，有 Jason Zweig 的注释

---

《证券分析》
Security Analysis
作者：Benjamin Graham & David Dodd
出版：1934年（多次修订）

为什么读：
- 价值投资的技术基础
- 深入的财务分析方法
- 债券和股票分析

难度：较高，适合进阶读者

---

《巴菲特致股东的信》
Berkshire Hathaway Letters to Shareholders
作者：Warren Buffett
来源：www.berkshirehathaway.com（免费）

为什么读：
- 巴菲特亲笔的投资智慧
- 真实案例的分析
- 理解长期投资思维

建议：从最新的开始，倒读历史信件
```

### 投资思维

```
《穷查理宝典》
Poor Charlie's Almanack
作者：Charlie Munger

为什么读：
- 跨学科思维
- 心理学与投资
- 清单式思考
- 避免错误的智慧

核心观点：
- 多元思维模型
- 逆向思考
- 能力圈
- 误判心理学

---

《随机漫步的傻瓜》
Fooled by Randomness
作者：Nassim Nicholas Taleb

为什么读：
- 理解随机性
- 区分能力和运气
- 避免过度自信

核心观点：
- 幸存者偏差
- 黑天鹅事件
- 随机性的欺骗

---

《思考，快与慢》
Thinking, Fast and Slow
作者：Daniel Kahneman

为什么读：
- 理解认知偏误
- 改善决策质量
- 行为金融学基础

核心观点：
- 系统1和系统2思维
- 损失厌恶
- 锚定效应
```

## 进阶读物

### 估值与分析

```
《股市真规则》
The Five Rules for Successful Stock Investing
作者：Pat Dorsey

为什么读：
- Morningstar 的投资方法
- 护城河分析
- 行业分析框架

适合：想学习系统化分析的投资者

---

《投资最重要的事》
The Most Important Thing
作者：Howard Marks

为什么读：
- 橡树资本创始人的智慧
- 周期思维
- 风险管理

核心观点：
- 第二层思维
- 市场周期
- 风险控制

---

《护城河投资》
The Little Book That Builds Wealth
作者：Pat Dorsey

为什么读：
- 专注讲护城河
- 简洁实用
- 案例丰富

核心观点：
- 五种护城河类型
- 如何识别真假护城河
```

### 市场历史

```
《漫步华尔街》
A Random Walk Down Wall Street
作者：Burton Malkiel

为什么读：
- 市场效率理论
- 指数投资的理论基础
- 泡沫历史

核心观点：
- 主动投资难以战胜市场
- 指数基金是好选择
- 市场有时非理性

---

《非理性繁荣》
Irrational Exuberance
作者：Robert Shiller

为什么读：
- 泡沫的形成机制
- CAPE 估值方法
- 行为金融学视角

核心观点：
- 估值对长期回报很重要
- 市场可以长期非理性
- 社会和心理因素影响市场

---

《股票作手回忆录》
Reminerta Trader
作者：Edwin Lefèvre

为什么读：
- 投机大师的故事
- 市场人性不变
- 投资纪律的重要性

经典语录：
"华尔街没有新鲜事"
"关键是坐着不动"
```

## 阅读建议

### 阅读顺序

```
入门阶段（第1年）：
1. 《聪明的投资者》
2. 《漫步华尔街》
3. 《小狗钱钱》或《百万富翁的简单财富之道》

进阶阶段（第2-3年）：
4. 《股市真规则》
5. 《投资最重要的事》
6. 《穷查理宝典》

深入阶段（第3年+）：
7. 《证券分析》
8. 巴菲特致股东的信
9. 《思考，快与慢》

持续阅读：
- 年报（巴菲特信、公司年报）
- 投资者会议记录
- 高质量财经文章
```

### 如何阅读

```
有效阅读方法：

1. 带着问题读
   - 这本书要解决什么问题？
   - 我能学到什么？
   - 如何应用？

2. 做笔记
   - 标记重要段落
   - 写下自己的想法
   - 总结核心观点

3. 重复阅读
   - 经典要多读几遍
   - 每次有新收获
   - 不同阶段理解不同

4. 应用实践
   - 读完立即应用
   - 在实践中验证
   - 形成自己的方法

5. 讨论分享
   - 和他人讨论
   - 教会别人
   - 加深理解
```

## 最后的建议

```
阅读的目的：

不是为了：
- 收集书单
- 炫耀读了多少书
- 寻找神奇公式

而是为了：
- 建立正确的思维框架
- 学习前人的智慧
- 避免重复犯错
- 形成自己的方法

读得少而精，胜过读得多而浅。
理解并应用一本书，胜过浏览十本书。

最好的投资是投资自己。
阅读是投资自己最好的方式之一。
```
