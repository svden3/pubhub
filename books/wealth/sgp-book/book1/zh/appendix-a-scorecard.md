# 附录A：S-G-P 评分表

---

## 投资评分系统

### 如何使用这个评分表

```
1. 为每个候选投资打分
2. 每个部分满分如标注
3. 总分 100 分
4. 建议：70分以上考虑投资

评分原则：
- 诚实评估
- 如有疑问，分数从低
- 定期重新评估持仓
```

---

## 第一部分：业务质量（30分）

### 1.1 护城河（10分）

```
评估公司的竞争优势：

强大护城河（8-10分）：
□ 品牌力量（如：苹果、可口可乐）
□ 网络效应（如：Facebook、Visa）
□ 转换成本（如：微软、Oracle）
□ 成本优势（如：Costco、Amazon）
□ 监管许可（如：公用事业）

中等护城河（5-7分）：
□ 有一定品牌认知
□ 有一些转换成本
□ 规模有一定优势
□ 但竞争对手可以追赶

弱护城河（1-4分）：
□ 竞争激烈
□ 产品同质化
□ 容易被替代
□ 没有明显优势

我的评分：___/10
评分理由：_____________________
```

```
困难行业（1-4分）：
```

## 第二部分：财务质量（30分）

### 2.1 盈利稳定性（10分）

```
评估盈利记录：

优秀（8-10分）：
□ 过去 10 年每年盈利
□ 盈利稳定增长
□ 经济衰退时也盈利
□ 盈利可预测

良好（5-7分）：
□ 大部分年份盈利
□ 有一定波动
□ 偶尔亏损但能恢复
□ 周期性但可控

较差（1-4分）：
□ 盈利不稳定
□ 经常亏损
□ 依赖非经常性项目
□ 难以预测

我的评分：___/10
评分理由：_____________________
```

### 2.2 资产负债表（10分）

```
评估财务健康：

健康（8-10分）：
□ 负债率 < 40%
□ 流动比率 > 2
□ 现金充裕
□ 无即将到期的大额债务
□ 利息覆盖率 > 5

中等（5-7分）：
□ 负债率 40-60%
□ 流动比率 1-2
□ 有一定现金
□ 债务可控
□ 利息覆盖率 3-5

高风险（1-4分）：
□ 负债率 > 60%
□ 流动比率 < 1
□ 现金紧张
□ 大额债务即将到期
□ 利息覆盖率 < 3

我的评分：___/10
评分理由：_____________________
```

```
评估现金流：

优秀（8-10分）：

较差（1-4分）：
□ 经营现金流 < 净利润
□ 自由现金流为负
□ 资本支出过高
□ 依赖融资
```

## 第三部分：估值（20分）

### 3.1 相对历史估值（10分）

```
与公司历史估值比较：

低估（8-10分）：
□ 当前 P/E < 历史平均 20%+
□ 当前 P/B < 历史平均 20%+
□ 在历史估值区间下沿

合理（5-7分）：
□ 当前估值接近历史平均
□ 在历史估值区间中部

高估（1-4分）：
□ 当前 P/E > 历史平均 20%+
□ 在历史估值区间上沿
□ 明显偏高

我的评分：___/10
当前 P/E：_____ 历史平均：_____
当前 P/B：_____ 历史平均：_____
```

### 3.2 相对同行估值（10分）

```
与同行业公司比较：

低估（8-10分）：

高估（1-4分）：
□ P/E 高于同行平均 20%+

```

```
□ 知道如何赚钱

□ 理解主要风险
□ 能写出完整投资论文

□ 大致了解业务

不太理解（1-4分）：
□ 不清楚商业模式
□ 不知道竞争对手
□ 不理解财务数据
□ 只是跟风

_________________________________
```

```

高度匹配（8-10分）：
```

## 决策指南

```
总分解读：

90-100分：
优秀机会
→ 可以重仓
→ 但仍要分散

80-89分：
良好机会
→ 可以建仓
→ 标准仓位

70-79分：
可接受机会
→ 可以小仓位
→ 继续观察

60-69分：
边缘机会
→ 暂不投资
→ 放入观察列表

<60分：
不合适
→ 不投资
→ 寻找其他机会
```

## 使用示例

```
公司：苹果 (AAPL)
日期：2024年1月

业务质量：
- 护城河：9/10（生态系统、品牌）
- 行业前景：8/10（智能设备继续增长）
- 管理层：9/10（库克表现优秀）
小计：26/30

财务质量：
- 盈利稳定性：9/10（持续盈利）
- 资产负债表：8/10（现金充裕，有债务）
- 现金流质量：10/10（FCF极强）
小计：27/30

估值：
- 历史估值：6/10（接近历史平均）
- 同行估值：5/10（高于大多数科技股）
小计：11/20

个人适合度：
- 能力圈：9/10（产品用户，理解业务）
- 风险匹配：8/10（大盘股，波动可控）
小计：17/20

总分：81/100

决策：良好机会，可以建仓
```
