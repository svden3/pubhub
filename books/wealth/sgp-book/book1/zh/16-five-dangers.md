# 第十六章：将之五危 — 投资者的致命弱点

---

> "故将有五危：必死，可杀也；必生，可虏也；忿速，可侮也；廉洁，可辱也；爱民，可烦也。"
>
> — 《孙子兵法·九变篇》

---

## 孙子论"将之五危"

孙子说：将帅有五种致命弱点：

| 五危 | 原意 | 结果 |
|-----|------|------|
| **必死** | 只知道拼命 | 会被消灭 |
| **必生** | 贪生怕死 | 会被俘虏 |
| **忿速** | 急躁易怒 | 会被激怒 |
| **廉洁** | 过于爱惜名誉 | 会被羞辱 |
| **爱民** | 过于仁慈 | 会被拖累 |

投资者也有类似的"五危"，每一种都可能导致毁灭。

---

## 投资者的五大致命弱点

### 一、必死 — 过度冒险

```
表现：
- 全仓单只股票
- 使用高杠杆
- 追逐"十倍股"
- 无视风险

心理：
"要么暴富，要么归零"
"搏一搏，单车变摩托"

后果：
- 一次失败就归零
- 没有第二次机会

案例：
2021年 Bill Hwang (Archegos)
- 高杠杆集中持仓
- 一周内亏损 $200亿
- 从亿万富翁变成负债
```

### 二、必生 — 过度保守

```
表现：
- 害怕任何亏损
- 只敢买货币基金
- 稍有波动就卖出
- 永远等"更便宜"

心理：
"股市太危险"
"现金最安全"

后果：
- 错过长期增长
- 被通胀侵蚀
- 永远在等

案例：
从2009年开始"等更低点"
- S&P 500 从 666 涨到 4000+
- 等了 14 年，越等越贵
- 被通胀吃掉 30% 购买力
```

### 三、忿速 — 情绪化交易

```
表现：
- 被新闻吓到卖出
- 被上涨激动买入
- 对亏损愤怒加仓
- 频繁交易

心理：
"涨了！赶快买"
"跌了！全部卖"
"亏了！加仓报复"

后果：
- 追涨杀跌
- 交易成本累积
- 长期亏损

案例：
2020年3月恐慌卖出
- VIX 创历史新高
- 恐惧之下卖光
- 错过 9 个月后创新高
```

### 四、廉洁 — 死不认错

```
表现：
- "不卖就不算亏"
- 死扛到底
- 拒绝止损
- 不愿承认错误

心理：
"我不可能错"
"总会涨回来"
"认赔太丢脸"

后果：
- 小亏变大亏
- 错失其他机会
- 本金大幅缩水

案例：
持有诺基亚从 $60 到 $5
- "智能手机只是时尚"
- "诺基亚会反击"
- 最终承认时已亏 92%
```

### 五、爱民 — 持仓过多

```
表现：
- 舍不得卖任何股票
- "万一涨了呢"
- 持有 100 只股票
- 每只都"有感情"

心理：
"都是我选的宝贝"
"这只有故事"
"卖了就后悔"

后果：
- 无法集中管理
- 收益接近指数
- 但费用更高

案例：
持有 50 只股票
- 无法深入研究每只
- 好的被坏的拖累
- 不如直接买指数
```

## 如何克服五危

### 克服"必死" — 设定风险边界

```
规则：
□ 单只股票 < 5%
□ 杠杆 < 1.5x（或不用）
□ 现金储备 > 10%
□ 止损纪律

心法：
"活着比暴富更重要"
"有限亏损，无限机会"
```

### 克服"必生" — 接受合理风险

```
规则：
□ 理解波动 ≠ 风险
□ 长期投资可以承受波动
□ 设定合理的股票比例
□ 分批建仓减少恐惧

心法：
"最大的风险是不承担任何风险"
"通胀也是风险"
```

### 克服"忿速" — 建立交易纪律

```
规则：
□ 预先制定计划
□ 不因新闻交易
□ 设置冷却期（24小时）
□ 限制交易频率

心法：
"情绪是投资的敌人"
"计划交易，交易计划"
```

### 克服"廉洁" — 学会认错

```
规则：
□ 设定止损位
□ 定期重新评估持仓
□ 问"如果现在没有，会买吗"
□ 接受错误是正常的

心法：
"认错是纠错"
"早认错，损失小"
```

### 克服"爱民" — 定期精简

```
规则：
□ 持股不超过 20-30 只
□ 每季度审视持仓
□ 卖掉不再符合标准的
□ 集中在最好的想法上

心法：
"少即是多"
"持有最好的，卖掉其他"
```

## 自我诊断

### 你的弱点是什么？

```
必死倾向（过度冒险）：
□ 我经常全仓单只股票
□ 我使用杠杆
□ 我追逐高风险高回报
□ 我认为"搏一把"

必生倾向（过度保守）：
□ 我害怕任何亏损
□ 我经常踏空
□ 我总在等"更低"
□ 股票占比 < 20%

忿速倾向（情绪化）：
□ 我因新闻而交易
□ 我频繁买卖
□ 亏损时我很愤怒
□ 盈利时我很亢奋

廉洁倾向（死不认错）：
□ 我从不止损
□ 我相信"会涨回来"
□ 我持有亏损 50% 以上的股票
□ 我不愿承认判断错误

爱民倾向（持仓过多）：
□ 我持有 30 只以上股票
□ 我舍不得卖任何一只
□ 我无法说出每只持仓的投资逻辑
□ 我的持仓像收藏品

每项勾选越多，该弱点越严重。
```

## 案例分析

### 案例：LTCM 的覆灭

```
长期资本管理公司 (LTCM)

背景：
- 1994年成立
- 创始人包括两位诺贝尔奖得主
- "史上最聪明的交易团队"

五危体现：

必死（过度冒险）：
- 杠杆高达 25:1
- "我们的模型不会错"

廉洁（死不认错）：
- 1998年开始亏损
- 不愿减仓
- "市场会回归"

结果：
- 1998年俄罗斯债务危机
- 4个月亏损 $46亿
- 美联储协调救助
- 公司倒闭

教训：
再聪明的人，也逃不过"五危"。
```

## 本章总结

### 将之五危

| 五危 | 投资表现 | 克服方法 |
|-----|---------|---------|
| 必死 | 过度冒险 | 设定风险边界 |
| 必生 | 过度保守 | 接受合理风险 |
| 忿速 | 情绪化交易 | 建立交易纪律 |
| 廉洁 | 死不认错 | 学会认错止损 |
| 爱民 | 持仓过多 | 定期精简 |

### 核心心法

> "凡此五者，将之过也，用兵之灾也。"

这五种弱点，任何一种都可能毁灭你。

了解自己的弱点，是避免灾难的第一步。
