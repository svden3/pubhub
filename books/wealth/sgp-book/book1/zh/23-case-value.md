## 价值投资的经典案例

### 案例一：苹果公司（2016年）

```
背景：
2016年，苹果股价低迷
- iPhone 销量首次下滑
- 市场担忧"后 iPhone 时代"
- P/E 约 10，远低于市场平均

价值分析：
- 1500亿美元现金
- 强大的生态系统
- 忠诚的用户群
- 服务业务在增长

巴菲特的行动：
- 2016年开始买入
- 最终投资约 350亿美元
- 成为伯克希尔最大持仓

结果（2016-2024）：
- 苹果股价涨 6倍+
- 伯克希尔获利超 1000亿美元

教训：
当优质公司因短期问题被低估时，
就是价值投资的机会。
```

### 案例二：美国运通（1963年）

```
背景：
1963年"色拉油丑闻"
- 一家子公司担保了虚假的色拉油库存
- 公司面临巨额赔偿
- 股价暴跌 50%

巴菲特的分析：
- 运通的核心业务没有受损
- 信用卡业务依然强劲
- 旅行支票业务依然独占
- 这是一次性问题，不是结构性问题

巴菲特的行动：
- 投资 1300万美元（占当时资产的 40%！）
- 集中投资

结果：
- 运通股价完全恢复
- 巴菲特获利数倍
- 这笔投资奠定了他的声誉

教训：
区分"暂时问题"和"永久问题"。
暂时问题造成的下跌是机会。
```

### 案例三：可口可乐（1988年）

```
背景：
1987年股灾后
- 可口可乐股价下跌
- 但业务没有任何问题
- P/E 约 15

巴菲特的分析：
- 全球最强的品牌
- 无与伦比的分销网络
- 高利润率
- 长期增长潜力

巴菲特的行动：
- 1988年开始买入
- 投资约 10亿美元
- 持有至今

结果：
- 加上分红，投资增长 20倍+
- 35年后仍是核心持仓
- 每年分红超过当初投资本金

教训：
找到伟大的企业，
在合理价格买入，
然后永远持有。
```

## 普通投资者的案例

### 案例四：指数投资者老王

```
背景：
老王，普通工薪族
- 2010年开始投资
- 每月定投 S&P 500 指数基金
- 月投 500美元

策略：
- 不管市场涨跌
- 坚持每月定投
- 从不卖出
- 分红再投资

经历的危机：
- 2011年：欧债危机，跌 20%
- 2015年：中国股灾，跌 12%
- 2018年：加息恐慌，跌 20%
- 2020年：新冠，跌 34%
- 2022年：通胀，跌 25%

老王的反应：
每次下跌：继续定投，买更多份额

结果（2010-2024）：
- 总投入：约 $84,000
- 账户价值：约 $240,000
- 年化收益：约 12%

教训：
普通人靠纪律和时间也能积累财富。
不需要择时，不需要选股。
```

### 案例五：耐心持有者小李

```
背景：
小李，2015年买入腾讯
- 买入价：150港元
- 买入理由：微信用户爆发

经历：
2015-2018：
- 股价涨到 400+港元
- 朋友说"卖吧，涨太多了"
- 小李：业务还在增长，继续持有

2018-2019：
- 游戏版号问题
- 股价跌到 250港元
- 朋友说"快卖，腾讯完了"
- 小李：核心业务没问题，继续持有

2020-2021：
- 股价涨到 700+港元
- 朋友说"这次该卖了吧"
- 小李：微信生态还在扩展，继续持有

2022-2024：
- 监管风暴，股价跌到 200港元
- 这次真的考验来了...
- 小李：业务依然赚钱，继续持有

结果：
- 9年持有，平均年化约 3%
- 但收到大量分红
- 学会了穿越周期

教训：
长期持有不等于永远正确，
但给优质公司时间是必要的。
```

## 失败的案例

### 案例六：价值陷阱——通用电气

```
背景：
2017年，GE 看起来很便宜
- 百年老店
- P/E 只有 12
- 股息率 4%+
- 很多人认为是价值机会

问题：
- 会计问题掩盖了真实状况
- 金融业务风险被低估
- 管理层过度多元化
- 护城河在消失

结果：
- 2017年：$30
- 2018年：$15
- 2019年：$10
- 削减股息 90%
- 从道琼斯指数除名

教训：
低 P/E 不等于便宜。
要理解为什么便宜。
如果是因为业务恶化，那是价值陷阱。
```

### 案例七：没有安全边际——英伟达追高

```
背景：
2024年，某投资者追高英伟达
- 股价已涨 10倍
- P/E 超过 60
- "AI 是未来，不能错过"

买入：
- 2024年3月买入，$900/股
- 全仓买入，没有分散

结果（假设情景）：
情景A：继续涨
- 运气好，赚钱
- 但风险巨大

情景B：估值回归
- 哪怕业务很好
- 如果 P/E 从 60 降到 30
- 股价会腰斩

教训：
再好的公司，买贵了也会亏。
没有安全边际的投资不是价值投资。
```

## S-G-P 分析框架

### 如何评估案例

```
孙子视角（战略）：
1. 这是有利地形吗？（行业趋势）
2. 敌人（竞争对手）的情况如何？
3. 我们有什么优势？

Graham 视角（估值）：
1. 价格合理吗？
2. 有安全边际吗？
3. 财务稳健吗？

实践视角（执行）：
1. 我能理解吗？
2. 我能承受波动吗？
3. 我能长期持有吗？
```

```
评估任何投资机会：
```

## 价值投资的要素

### 要素一：耐心

```
价值投资需要时间：

- 苹果：巴菲特持有 8年+
- 可口可乐：持有 35年+
- 美国运通：持有 60年+

耐心的来源：
1. 对企业的深入理解
2. 对估值的信心
3. 对复利的信仰

没有耐心，价值投资不可能成功。
```

### 要素二：纪律

```
价值投资需要纪律：

买入纪律：
- 只在安全边际足够时买入
- 不因 FOMO 追高
- 分批建仓

持有纪律：
- 不因短期波动卖出
- 定期重新评估
- 记录投资逻辑

卖出纪律：
- 逻辑改变时卖出
- 估值过高时减仓
- 有更好机会时换仓
```

### 要素三：独立思考

```
价值投资需要逆向：

大众恐惧时：
- 往往是买入机会
- 但需要独立判断

大众贪婪时：
- 往往是卖出信号
- 但不盲目逆向

独立思考的关键：
- 做自己的研究
- 不依赖"专家"
- 相信自己的分析
```

## 本章总结

### 成功案例的共性

| 要素 | 说明 |
|-----|------|
| 优质企业 | 有护城河，业务简单 |
| 合理价格 | 有安全边际 |
| 长期持有 | 给复利时间 |
| 逆向买入 | 在恐惧时贪婪 |

### 失败案例的教训

```
1. 低估值 ≠ 便宜
   要理解为什么便宜

2. 高增长 ≠ 好投资
   价格同样重要

3. 短期 ≠ 验证
   需要时间证明
```

### 核心心法

> "善用兵者，譬如率然。"

好的投资就像常山之蛇：
- 各部分相互配合（分析框架）
- 能屈能伸（灵活应变）
- 首尾相应（逻辑一致）

找到优质企业，
在合理价格买入，
然后让时间发挥魔力。
