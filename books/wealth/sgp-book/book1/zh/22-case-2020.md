## 危机概述

### 最快的熊市

```
2020年新冠暴跌时间线：

2020年2月19日：
- S&P 500 创历史新高 3393

2020年2月下旬：
- 疫情在全球扩散
- 市场开始下跌

2020年3月9日：
- 熔断！S&P 500 跌 7.6%
- 原油暴跌 25%

2020年3月12日：
- 再次熔断！
- 道琼斯跌 10%

2020年3月16日：
- 第三次熔断！
- S&P 500 跌 12%

2020年3月23日：
- S&P 500 触底 2237
- 从高点跌 34%
- 仅用 33 天！

这是历史上最快的熊市。
```

### 与2008年的对比

```
2008 vs 2020：

下跌速度：
- 2008：17个月跌 57%
- 2020：33天跌 34%

原因：
- 2008：金融系统崩溃
- 2020：外部冲击（疫情）

复苏速度：
- 2008：4年才回到高点
- 2020：5个月就创新高

政策反应：
- 2008：缓慢、犹豫
- 2020：迅速、大规模

教训：
不同的危机，需要不同的应对。
```

## 投资者的选择

### 选择一：恐慌卖出

```
2020年3月的恐慌：

典型想法：
- "这次真的不一样"
- "经济会停摆"
- "可能是大萧条"
- "先卖出，等明朗再说"

如果3月23日卖出：
- 锁定 -34% 亏损
- 错过接下来 5 个月 +68% 的反弹
- 完美地卖在底部

VIX（恐慌指数）：
- 正常：15-20
- 2020年3月：82（历史第二高）

当所有人都恐慌时，
往往是最差的卖出时机。
```

### 选择二：什么都不做

```
"躺平"策略：

做法：
- 不看账户
- 不做任何操作
- 继续定投

结果：
如果从 2020年1月持有到年底：
- S&P 500：+18%
- 纳斯达克：+44%
- 一年收益创历史

教训：
什么都不做，有时是最好的选择。
前提是你的投资组合本来就合理。
```

### 选择三：逆向加仓

```
"别人恐惧时贪婪"：

做法：
- 3月份加仓
- 买入优质公司
- 用闲置现金

结果：
如果 3月23日买入 S&P 500：
- 到 2020年底：+68%
- 到 2024年底：+150%

具体案例：
- 亚马逊：3月低点 $1626 → 年底 $3256（+100%）
- 特斯拉：3月低点 $70 → 年底 $705（+900%）
- 苹果：3月低点 $56 → 年底 $132（+136%）

教训：
如果有现金和勇气，危机是最好的机会。
```

## S-G-P 框架分析

### 孙子视角

```
"善战者，立于不败之地"

2020年的应用：

不败之地 = 稳健的资产配置
- 分散投资
- 有现金储备
- 不用杠杆

不失敌之败 = 危机中抓住机会
- 有现金可以加仓
- 有勇气逆向操作
- 有耐心等待复苏

即使没有抄底成功，
只要不卖出，就没有输。
```

### Graham 视角

```
安全边际的验证：

2020年3月提供了巨大的安全边际：
- 苹果 P/E 从 25 跌到 18
- 微软 P/E 从 32 跌到 24
- 很多优质公司被错杀

Graham 的方法：
1. 识别被错杀的优质公司
2. 检查财务稳健性
3. 确保有安全边际
4. 分批买入

问题：需要勇气
- 3月23日时，没人知道是底
- 可能还会跌
- 需要承受不确定性
```

### 实践要点

```
从2020年学到的：

1. 速度惊人
   - 33天暴跌34%
   - 没时间慢慢思考
   - 需要预先有计划

2. 复苏也快
   - 5个月创新高
   - 如果等"确认"再买，已经涨完了
   - 最好的日子在最差的日子之后

3. 分化严重
   - 有些行业涨10倍
   - 有些行业跌一半
   - 分散是保护

4. 现金为王
   - 有现金才能抄底
   - 没现金只能看着
```

## 具体操作复盘

### 最佳操作

```
理想的操作：

2020年2月：
- 市场创新高
- 保持正常配置
- 确保有 10-20% 现金

2020年3月初：
- 下跌 10%
- 不恐慌
- 检查持仓质量

2020年3月中：
- 下跌 25%
- 开始小仓位买入
- 选择优质公司

2020年3月23日：
- 下跌 34%
- 继续买入
- 不知道是底，但知道便宜

2020年4-5月：
- 市场反弹
- 持有
- 不追高

2020年全年：
- 保持持有
- 收获复苏红利
```

### 常见错误

```
避免这些错误：

错误1：3月恐慌卖出
- 卖在底部
- 错过反弹

错误2：4月追高买入
- 在反弹后 FOMO
- 买在高点

错误3：全仓单一行业
- 如果是航空股，亏惨
- 如果是科技股，大赚
- 但无法事先知道

错误4：用杠杆
- 3月被强制平仓
- 没有机会等反弹

错误5：等"确认"再买
- 等到确认时已经涨了 50%
- 最好的机会已过
```

## 建立应对机制

### 危机预案

```
提前制定计划：

下跌 10% 时：
□ 检查持仓质量
□ 确认没有杠杆
□ 继续正常定投

下跌 20% 时：
□ 考虑增加投资（如果有现金）
□ 不卖出
□ 减少看盘

下跌 30% 时：
□ 加大投资（如果有现金）
□ 买入优质资产
□ 保持冷静

下跌 40%+ 时：
□ 继续买入
□ 这是难得的机会
□ 相信长期

写下来，在平静时制定。
危机时执行，不临时改变。
```

### 心理准备

```
提前接受：

1. 接受波动
   - 30-50% 的下跌是正常的
   - 每隔几年就会发生
   - 这是获得长期收益的代价

2. 接受不确定
   - 没人知道底在哪
   - 没人知道会跌多久
   - 接受不知道

3. 接受恐惧
   - 恐惧是正常的
   - 但不要被恐惧控制
   - 有计划就有勇气
```

## 本章总结

### 2020年的教训

| 教训 | 说明 |
|-----|------|
| 速度惊人 | 33天跌34%，没时间反应 |
| 复苏也快 | 5个月创新高，等确认就错过 |
| 分化严重 | 科技涨，航空跌 |
| 现金重要 | 有现金才能抓机会 |

### 正确的做法

```
1. 不恐慌卖出
2. 继续定投
3. 有现金就加仓
4. 选择优质资产
5. 保持长期视角
```

### 核心心法

> "善战者，立于不败之地，而不失敌之败也。"

在危机中不败：
- 分散投资
- 不用杠杆
- 有现金储备

抓住危机的机会：
- 有勇气买入
- 有现金加仓
