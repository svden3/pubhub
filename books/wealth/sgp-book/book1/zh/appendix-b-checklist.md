# 附录B：选股清单

## Graham 选股清单

### 防御型投资者清单

```
Benjamin Graham 的七条标准：

1. 规模充足
   □ 年销售额 > 1亿美元（工业）
   □ 年销售额 > 5000万美元（公用事业）

2. 财务稳健
   □ 流动比率 > 2
   □ 长期债务 < 净流动资产

3. 盈利稳定
   □ 过去 10 年每年盈利
   □ 无年度亏损

4. 分红记录
   □ 过去 20 年连续分红

5. 盈利增长
   □ 10 年 EPS 增长 > 33%
   □ 使用 3 年平均比较

6. 适度市盈率
   □ P/E < 15（使用 3 年平均盈利）

7. 适度市净率
   □ P/B < 1.5
   □ 或 P/E × P/B < 22.5

通过标准：至少 5 条
理想情况：全部 7 条
```

### 进取型投资者清单

```
Graham 对积极投资者的补充标准：

基础标准：
□ 流动比率 > 1.5
□ 有正现金流
□ 盈利记录（可以短于 10 年）
□ 正在分红

价值标准：
□ P/E < 同行业平均
□ P/B < 1.2
□ 或股价 < 净流动资产价值

成长标准：
□ 盈利增长趋势
□ 收入增长趋势
□ 利润率改善
```

## 巴菲特式清单

### 业务清单

```
可理解性：
□ 业务简单，能向小孩解释
□ 知道公司如何赚钱
□ 理解产品/服务
□ 了解客户是谁

护城河：
□ 有持久竞争优势
□ 竞争对手难以模仿
□ 定价权
□ 高客户忠诚度

行业：
□ 行业稳定或增长
□ 不需要持续创新才能生存
□ 竞争理性
□ 不会被技术颠覆
```

### 管理层清单

```
能力：
□ 过去业绩证明能力
□ 资本配置能力强
□ 运营效率高
□ 长期任职

诚信：
□ 股东沟通透明
□ 承认错误
□ 不夸大业绩
□ 没有诚信黑历史

股东导向：
□ 持有公司股票
□ 薪酬与业绩挂钩
□ 回购 vs 分红决策合理
□ 不进行毁灭价值的并购
```

### 财务清单

```
盈利：
□ ROE > 15%（且稳定）
□ 利润率稳定或提升
□ 盈利可预测

资产负债表：
□ 低负债或有能力偿还
□ 充足现金
□ 没有复杂的资产负债表外项目

现金流：
□ 经营现金流强劲
□ 资本支出低（轻资产）
□ 自由现金流增长
□ 可用于分红和回购
```

### 估值清单

```
价格：
□ 股价合理
□ 有安全边际
□ 未来回报可接受

具体指标：
□ P/E 合理（< 15 或 < 行业平均）
□ P/FCF 合理（< 20）
□ EV/EBITDA 合理（< 10）
□ 股息率有吸引力（如适用）
```

## 快速筛选清单

### 5分钟快速筛选

```
第一轮筛选（先排除明显不合格的）：

□ P/E > 0（盈利）
□ P/E < 25（估值不疯狂）
□ 负债率 < 70%（不是高杠杆）
□ 最近 5 年盈利为正
□ 你听说过这家公司

如果有一项不满足，放弃或需要特别理由
```

### 30分钟深入筛选

```
第二轮筛选（值得花时间研究）：

业务（10分钟）：
□ 读公司简介，理解业务
□ 识别主要产品/服务
□ 了解竞争对手
□ 初步判断护城河

财务（10分钟）：
□ 过去 5 年收入趋势
□ 过去 5 年盈利趋势
□ 过去 5 年现金流趋势
□ 当前负债水平
□ ROE 和利润率

估值（10分钟）：
□ 当前 P/E vs 历史
□ 当前 P/E vs 同行
□ P/B 水平
□ 股息率
```

### 完整分析清单

```
第三轮分析（准备投资）：

深度业务分析（2-4小时）：
□ 读最新年报
□ 听最近 2 次财报电话
□ 研究竞争格局
□ 评估护城河强度
□ 识别主要风险
□ 了解行业趋势

深度财务分析（1-2小时）：
□ 10 年财务数据
□ 现金流分析
□ 资产负债表分析
□ 关键指标趋势
□ 与竞争对手对比

估值分析（1小时）：
□ DCF 估值
□ 相对估值
□ 历史估值区间
□ 确定安全边际

投资论文（30分钟）：
□ 写下买入理由
□ 写下主要风险
□ 写下卖出条件
□ 确定目标仓位
```

## 红旗清单

### 业务红旗

```
立即警惕：

□ 业务模式不清晰
□ 收入来源可疑
□ 过度依赖单一客户
□ 行业正在衰落
□ 技术被颠覆
□ 监管风险大
□ 高度周期性且在顶部
```

### 财务红旗

```
立即警惕：

□ 经营现金流 << 净利润
□ 应收账款增长 > 收入增长
□ 存货增长 > 收入增长
□ 负债率持续上升
□ 频繁稀释股东
□ 非经常性项目频繁
□ 审计师意见非标准
```

### 管理层红旗

```
立即警惕：

□ CFO 或审计师离职
□ 高管大量抛售
□ 薪酬与业绩脱节
□ 频繁更换战略
□ 并购上瘾
□ 说的和做的不一致
□ 有诚信黑历史
```

### 估值红旗

```
立即警惕：

□ P/E > 50（除非高成长）
□ P/B > 10（除非轻资产）
□ 估值远超历史高点
□ 估值远超同行
□ 股价涨幅远超基本面改善
□ 被热门概念炒作
```

## 持有期检查清单

### 季度检查

```
每季度问自己：

业务：
□ 投资逻辑还成立吗
□ 护城河有变化吗
□ 竞争格局有变化吗
□ 管理层有变化吗

财务：
□ 季度业绩符合预期吗
□ 关键指标趋势正常吗
□ 有新的红旗信号吗

估值：
□ 估值还合理吗
□ 是否明显高估了
□ 需要调整仓位吗
```

### 年度审视

```
每年深度审视：

□ 重新阅读年报
□ 更新财务分析
□ 更新估值分析
□ 重新评估竞争格局
□ 检查投资论文是否成立
□ 与替代投资比较
□ 决定是否继续持有
```

## 卖出检查清单

### 考虑卖出时

```
卖出前问自己：

基本面原因：
□ 投资逻辑变了吗？
□ 护城河削弱了吗？
□ 管理层出问题了吗？
□ 行业恶化了吗？

估值原因：
□ 估值严重过高了吗？
□ 安全边际消失了吗？
□ 有更好的机会吗？

个人原因：
□ 需要资金吗？
□ 风险承受变了吗？
□ 组合需要再平衡吗？

情绪检查：
□ 是因为恐惧吗？
□ 等了 24 小时吗？
□ 卖完钱放哪里？

如果不是因为以上明确理由，不要卖。
```
