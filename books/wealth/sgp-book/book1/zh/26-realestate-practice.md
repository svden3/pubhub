## 买房流程

### 第一步：准备工作

```
财务准备：

首付资金：
□ 投资房通常需要 25%+
□ 多准备 5-10% 用于其他费用
□ 确认资金来源（银行要查）

贷款预批：
□ 找贷款经纪人或银行
□ 提交收入文件（W2、税单）
□ 获得预批信
□ 确定贷款额度

信用优化：
□ 查询信用报告
□ 还清信用卡
□ 不要新开信用账户
□ 目标：740+ 获得最佳利率
```

### 第二步：找房

```
定义标准：

位置标准：
□ 目标城市/区域
□ 学区要求
□ 通勤距离
□ 发展潜力

物业标准：
□ 类型（单户/多户）
□ 大小（卧室/卫浴）
□ 年代（新建/老房）
□ 状态（翻新/原始）

财务标准：
□ 价格范围
□ 租金预期
□ 现金流要求

找房渠道：
- 房产经纪
- MLS（房源网站）
- 法拍房
- 直接联系业主
```

### 第三步：分析房源

```
快速筛选：

1% 法则（快速筛选）：
月租金 ≥ 房价 × 1%
例：$500K 房子，月租 $5000+

注意：加州很难达到 1%，0.5-0.7% 较常见

详细分析：
用电子表格计算：
- 预期租金（看可比租金）
- 所有费用（贷款、税、保险、维修）
- 月现金流
- 现金回报率

如果数字不合理，放弃，看下一个。
```

### 第四步：出价与谈判

```
出价策略：

市场分析：
□ 房子在市多久
□ 价格调整历史
□ 可比销售价格
□ 卖家动机

出价技巧：
- 热市：接近或高于要价
- 冷市：低于要价 5-15%
- 附带合理条件
- 展示诚意（高定金、快关户）

谈判要点：
- 验房后可以重新谈判
- 修理或价格让步
- 关户成本分担
- 保留物品
```

### 第五步：尽职调查期

```
检查清单：

专业检查：
□ 房屋检查（$400-600）
□ 白蚁检查（$100-200）
□ 屋顶检查（如需要）
□ 下水道检查（老房）

文件审查：
□ 产权报告
□ HOA 文件（如有）
□ 披露文件
□ 许可证记录

财务验证：
□ 最终租金确认
□ 费用核实
□ 保险报价
□ 贷款最终确认

如有重大问题，可以：
- 要求修理
- 要求降价
- 退出交易
```

### 第六步：过户

```
过户准备：

资金准备：
□ 首付电汇
□ 过户费用
□ 预付（税、保险）
□ 储备金

文件签署：
□ 贷款文件
□ 过户文件
□ 产权保险
□ 各种披露

过户后：
□ 钥匙交接
□ 水电气转名
□ 保险生效
□ 物业管理安排
```

## 出租准备

### 房屋准备

```
清洁和修缮：

必做项目：
□ 专业深度清洁
□ 墙面粉刷（如需要）
□ 地毯清洗或更换
□ 所有设备检查

可选升级：
□ 厨房台面
□ 电器更新
□ 浴室升级
□ 景观美化

安全合规：
□ 烟雾报警器
□ 一氧化碳报警器
□ 锁具更换
□ 热水器固定（加州）
```

### 定价策略

```
租金定价：

市场调研：
□ 查看同区域类似房源
□ Zillow、Apartments.com
□ 问物业经纪
□ 考虑季节因素

定价原则：
- 市场价：快速出租
- 高于市场 5%：可能空置久
- 低于市场 5%：吸引更多申请

调整因素：
+ 翻新/设备好
+ 好学区
+ 宠物友好
- 无洗碗机/洗衣机
- 交通不便
- 老旧设备
```

### 发布房源

```
房源发布：

发布平台：
- Zillow / Trulia
- Apartments.com
- Craigslist
- Facebook Marketplace
- 本地报纸（老年租客）

房源描述：
- 卧室/卫浴数量
- 面积
- 主要设施
- 宠物政策
- 租金和押金
- 可入住日期
- 联系方式

照片要求：
- 明亮、整洁
- 每个房间
- 外观和停车
- 专业拍摄更好
```

## 租客筛选

### 筛选流程

```
第一步：电话/网络筛选

问题清单：
□ 为什么搬家？
□ 什么时候需要入住？
□ 有几个人住？
□ 有宠物吗？
□ 收入来源和金额？
□ 曾被驱逐过吗？

红旗信号：
- 急着入住
- 不愿回答问题
- 收入来源模糊
- 被驱逐过
```

### 看房

```
第二步：看房

安排看房：
- 约定具体时间
- 确认身份
- 不要单独去（安全）

观察申请人：
- 准时吗
- 穿着整洁吗
- 如何看待房屋
- 问什么问题

提供申请表：
- 现场或在线
- 收取申请费（$30-50）
- 说明筛选流程
```

### 背景调查

```
第三步：背景调查

检查项目：
□ 信用报告
□ 犯罪记录
□ 驱逐记录
□ 收入验证
□ 前房东联系

信用报告关注：
- 信用分（建议 650+）
- 欠租记录
- 破产记录
- 债务负担

联系前房东：
"他们准时付租吗？"
"房屋保持得好吗？"
"你会再租给他们吗？"

收入验证：
- 工资单（最近 2-3 个月）
- W2（最近 1-2 年）
- 银行对账单
- 雇主确认
```

### 最终决定

```
第四步：做出决定

比较申请人：
选择最合格的申请人
（最高信用、最稳定收入、最好评价）

公平住房：
□ 不能因种族、肤色拒绝
□ 不能因宗教拒绝
□ 不能因家庭状况拒绝
□ 不能因残疾拒绝
□ 要有一致的标准

通知决定：
- 录取：通知签约时间
- 拒绝：提供书面理由（可选）
```

## 签约与入住

### 租约准备

```
租约要点：

基本条款：
□ 租期（通常 1 年）
□ 月租金
□ 押金金额
□ 逾期费用
□ 付款方式

规则条款：
□ 宠物政策
□ 转租政策
□ 修缮责任
□ 噪音规定
□ 停车规定

加州特定：
□ 租客权利通知
□ 铅漆披露（1978前房屋）
□ Megan's Law 披露
□ 床虫披露

建议：使用加州房地产协会标准租约
```

### 入住检查

```
入住流程：

文件签署：
□ 租约签字
□ 附加条款签字
□ 收据签发

费用收取：
□ 第一个月租金
□ 押金（最多 2 个月，无家具）
□ 钥匙押金（如有）

入住检查：
□ 房东和租客一起
□ 记录现有损坏
□ 拍照存档
□ 双方签字确认

钥匙交接：
□ 所有钥匙
□ 车库遥控
□ 门禁卡
□ 邮箱钥匙
```

## 日常管理

### 租金收取

```
收租系统：

收租方式：
- 在线支付（推荐）
- 银行转账
- 支票
- 现金（不推荐）

在线平台：
- Cozy（免费）
- Avail
- Buildium
- AppFolio

逾期处理：
第1天：发送提醒
第5天：收取逾期费
第10天：发送付款通知
第15天：开始法律程序
```

### 维修管理

```
维修流程：

紧急维修（24小时内）：
- 漏水
- 无热水
- 无暖气/空调（极端天气）
- 安全问题

常规维修：
- 租客报修
- 评估问题
- 安排修理
- 确认完成

维修责任：
房东负责：
- 结构问题
- 管道电气
- 电器（如提供）
- 公共区域

租客负责：
- 租客造成的损坏
- 小修小补
- 灯泡、电池等
```

### 租约续签

```
续签流程：

提前准备（到期前 60-90 天）：
□ 评估租客表现
□ 研究市场租金
□ 决定续签条款

续签选项：
1. 原条款续签
2. 涨租续签
3. 不续签（需正当理由）

涨租通知：
- 加州：提前 30 天（< 10% 涨幅）
- 加州：提前 90 天（> 10% 涨幅）
- 书面通知

AB 1482 限制：
年涨幅 ≤ 5% + 当地 CPI
最高 10%
```

## 问题处理

### 租客不付租

```
法律程序：

第1步：3天付款或搬离通知
- 书面送达
- 记录送达方式和时间

第2步：等待 3 天

第3步：如果不付/不搬
- 向法院提起驱逐诉讼（Unlawful Detainer）
- 支付起诉费

第4步：法庭听证
- 通常 20-30 天后
- 提供证据

第5步：如果胜诉
- 获得判决
- 申请执行
- Sheriff 强制执行

整个过程：1-3 个月
费用：$500-2000+

预防胜于治疗：筛选好租客！
```

### 其他问题

```
常见问题处理：

噪音投诉：
- 先口头警告
- 书面警告
- 如持续，可能驱逐

违规养宠：
- 记录证据
- 书面通知改正
- 收取额外押金或驱逐

转租/Airbnb：
- 检查租约条款
- 如违规，书面通知
- 可能驱逐

房屋损坏：
- 记录证据
- 评估责任
- 从押金扣除或要求赔偿
```

## 本章总结

### 房产投资流程

| 阶段 | 关键任务 |
|-----|---------|
| 准备 | 财务、贷款预批、信用 |
| 找房 | 定义标准、分析房源 |
| 买入 | 出价、尽调、过户 |
| 出租 | 准备房屋、发布、筛选 |
| 管理 | 收租、维修、续签 |

### 成功关键

```
1. 买对房 —— 地点、价格、条件
2. 选对人 —— 租客筛选最重要
3. 管好房 —— 及时响应、维护关系
4. 懂法规 —— 加州法规复杂
5. 有系统 —— 流程化、记录化
```

### 核心心法

> "夫地形者，兵之助也。"

房产是工具，帮助你积累财富。
但要用好这个工具，需要：
- 了解市场（地形）
- 选对位置（战略）
- 管好运营（战术）

掌握实操，房产才能真正为你服务。
