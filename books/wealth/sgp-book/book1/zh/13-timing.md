# 第十三章：择时智慧

---

> "善战者，求之于势，不责于人。"
>
> — 《孙子兵法·势篇》

---

## 孙子论"势"

孙子说：善于作战的人，**借势而为，不强求**。

投资中的"势"就是**时机**——知道什么时候该进攻，什么时候该防守。

| 原文 | 直译 | 投资映射 |
|-----|------|---------|
| 求之于势 | 借助形势 | 顺势而为 |
| 不责于人 | 不强求 | 不强行择时 |

---

## 择时的两个流派

### 流派一：不择时

```
代表人物：彼得·林奇、约翰·博格

核心观点：
- 择时是不可能的
- 错过最好的几天，收益大打折扣
- 长期持有优于频繁进出

数据支持：
S&P 500 1990-2020年：
- 全程持有：年化 9.8%
- 错过最好的10天：年化 5.6%
- 错过最好的20天：年化 2.9%

结论：
"时间在市场里"比"择时进出"更重要。
```

### 流派二：择时可行

```
代表人物：Howard Marks、Stanley Druckenmiller

核心观点：
- 极端时刻可以择时
- 不是预测具体点位
- 而是识别极端估值

方法：
- 不在日常波动中择时
- 只在极端时刻调整
- 估值是最好的指标

结论：
择时不是每天做，而是在极端时做。
```

## 何时进攻，何时防守

### 进攻的时机

```
信号：

估值信号：
□ P/E < 历史平均 30%
□ CAPE < 15
□ 股息率 > 债券收益率

情绪信号：
□ VIX > 30
□ 恐惧指数极端悲观
□ 媒体全是坏消息

资金信号：
□ 基金大规模赎回
□ 散户持仓创新低
□ 空头仓位创新高

行动：
- 增加股票仓位
- 用光现金储备
- 买入优质公司
```

### 防守的时机

```
信号：

估值信号：
□ P/E > 历史平均 50%
□ CAPE > 30
□ 股息率 < 1.5%

情绪信号：
□ VIX < 15
□ 贪婪指数极端乐观
□ 媒体全是好消息

资金信号：
□ 基金大规模申购
□ 散户持仓创新高
□ IPO 热潮

行动：
- 减少股票仓位
- 增加现金储备
- 卖出高估资产
```

## 择时的具体方法

### 方法一：基于估值

```
简单规则：

CAPE（席勒市盈率）

< 15：满仓进攻（股票 80%）
15-20：正常配置（股票 60%）
20-25：谨慎配置（股票 50%）
25-30：开始防守（股票 40%）
> 30：大幅防守（股票 30%）

优点：
- 简单明确
- 有历史支持
- 不需要预测

缺点：
- 可能长时间等待
- 可能错过短期机会
```

### 方法二：基于均值回归

```
200日均线法：

当价格 > 200日均线：
- 保持股票仓位
- 趋势向上

当价格 < 200日均线：
- 减少股票仓位
- 趋势向下

优点：
- 简单执行
- 避免大幅下跌

缺点：
- 会有假信号
- 横盘时反复触发
```

### 方法三：基于情绪

```
CNN 恐惧贪婪指数：

极度恐惧（< 25）：考虑买入
恐惧（25-45）：保持观望
中性（45-55）：正常配置
贪婪（55-75）：考虑减仓
极度贪婪（> 75）：大幅减仓

优点：
- 反映市场情绪
- 更新及时

缺点：
- 情绪可以持续极端
- 不够精确
```

## 不要做的事

### 误区一：预测具体点位

```
错误：
- "S&P 会跌到 3000"
- "底部在 3200"
- "会涨到 5000"

正确：
- "当前估值偏高"
- "恐慌时增加仓位"
- "狂热时减少仓位"

区别：
前者是预测，后者是应对。
```

### 误区二：频繁进出

```
错误：
- 每周调仓
- 根据新闻买卖
- 追涨杀跌

正确：
- 每季度最多调一次
- 只在极端时调整
- 保持核心仓位稳定

交易成本会吃掉收益。
```

### 误区三：追求完美

```
错误：
- 要买在最低点
- 要卖在最高点
- 稍微早了就后悔

正确：
- 接受会买早卖早
- 分批操作
- 买在"便宜区"而非最低点

彼得·林奇：
"我从来没有在最低点买过，也从没在最高点卖过。"
```

## 实操建议

### 建议一：分批操作

```
买入时：
- 不要一次性买入
- 分 3-5 批
- 每跌 10% 买一批

卖出时：
- 不要一次性卖出
- 分 3-5 批
- 每涨 10% 卖一批

好处：
- 避免择时失败的灾难
- 平均买入/卖出成本
- 心理压力小
```

### 建议二：保持核心仓位

```
无论怎么择时：

核心仓位（60%）：
- 永远不动
- 长期持有
- 无视短期波动

择时仓位（40%）：
- 可以调整
- 根据估值/情绪
- 控制调整幅度

这样既能择时，又不会错过大趋势。
```

### 建议三：预先计划

```
在平静时制定计划：

如果 S&P 跌 20%：
- 把现金比例从 20% 降到 10%
- 买入 XXX 公司

如果 S&P 跌 30%：
- 把现金比例降到 5%
- 继续买入

如果 S&P 涨到 P/E 25：
- 把股票比例从 60% 降到 50%
- 增加债券

写下来，到时候执行。
```

## 案例分析

### 案例一：2020年3月的择时

```
2020年2月中旬：
- S&P 500 创新高
- CAPE = 31（偏高）
- 疫情开始传播

正确的择时：
- 识别到风险
- 减仓到 50%
- 增加现金

2020年3月20日：
- S&P 500 跌 34%
- VIX = 82（极端恐惧）
- CAPE = 21（正常偏低）

正确的择时：
- 识别到机会
- 分批买入
- 用光现金

2020年底：
- S&P 500 创新高
- 择时成功

关键：
不是预测疫情，而是根据估值和情绪调整。
```

### 案例二：长期不择时

```
某投资者 1990-2020：

策略：
- 每月定投
- 无视市场波动
- 从不择时

结果：
- 年化收益 9%+
- 经历多次大跌
- 但都扛过来了
- 最终财富可观

结论：
不择时也是一种有效策略。
```

## 本章总结

### 择时的核心

| 类型 | 说明 |
|-----|------|
| **不择时** | 长期持有，简单有效 |
| **极端择时** | 只在极端时调整 |
| **频繁择时** | 不推荐，成本高 |

### 实操原则

```
1. 不预测具体点位
2. 只在极端时行动
3. 分批操作
4. 保持核心仓位
5. 预先制定计划
```

### 孙子的智慧

> "善战者，求之于势，不责于人。"

好的择时不是强求完美，而是借势而为：
- 势来了，顺势进攻
- 势去了，顺势防守
- 没有势，按兵不动
