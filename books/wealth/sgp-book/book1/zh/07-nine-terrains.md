# 第七章：九地风险分类

---

> "用兵之法，有散地，有轻地，有争地，有交地，有衢地，有重地，有圮地，有围地，有死地。"
>
> — 《孙子兵法·九地篇》

---

## 孙子的九种地形

孙子把战场地形分为九类，每种需要不同的战略。投资市场也有"九种地形"，每种风险需要不同的应对方式。

| 地形 | 原意 | 投资映射 | 风险等级 |
|-----|------|---------|---------|
| **散地** | 在本国作战 | 安全区（低风险） | ★☆☆☆☆ |
| **轻地** | 刚进入敌境 | 试探区（初步风险） | ★★☆☆☆ |
| **争地** | 双方必争之地 | 竞争区（中等风险） | ★★★☆☆ |
| **交地** | 四通八达 | 流动区（高流动性） | ★★☆☆☆ |
| **衢地** | 与多国接壤 | 关联区（系统风险） | ★★★★☆ |
| **重地** | 深入敌境 | 重仓区（高暴露） | ★★★★☆ |
| **圮地** | 难以通行 | 困境区（流动性风险） | ★★★★☆ |
| **围地** | 进退两难 | 被套区（被动持仓） | ★★★★★ |
| **死地** | 背水一战 | 危机区（生存风险） | ★★★★★ |

---

## 第一类：散地 — 安全区

### 定义

> "诸侯自战其地，为散地。"

在自己的领土作战，士兵可能逃散。投资中，这是**低风险、高流动性**的区域。

### 投资对应

| 资产类型 | 特点 |
|---------|------|
| 货币基金 | 几乎无风险，随时可取 |
| 短期国债 | 政府担保，流动性强 |
| 银行存款 | FDIC 保险覆盖 |
| 高评级公司债 | 违约风险低 |

### 应对策略

- 保持一定比例（10-20%）的"散地"资产
- 作为安全垫和机动资金
- 不追求高回报，追求安全

---

## 第二类：轻地 — 试探区

### 定义

> "入人之地不深者，为轻地。"

刚进入敌境，立足未稳。投资中，这是**小仓位试探**的区域。

### 投资对应

| 情况 | 例子 |
|-----|------|
| 新建仓位 | 刚开始买入的股票 |
| 小额实验 | 尝试新策略 |
| 学习成本 | 为了学习而投入的资金 |

### 应对策略

- 轻仓试探，不重仓
- 设定明确的止损
- 验证假设后再加仓

```
轻地规则：

初次建仓：总仓位的 1-2%
验证后加仓：每次 1-2%
最大仓位：5%

例：$100,000 组合
初次买入：$1,000-2,000
验证后：再加 $1,000-2,000
```

---

## 第三类：争地 — 竞争区

### 定义

> "我得则利，彼得亦利，为争地。"

双方都想占据的战略要地。投资中，这是**热门股、热门行业**。

### 投资对应

| 情况 | 例子 |
|-----|------|
| 热门股 | 2021年的 ARKK 成分股 |
| 热门行业 | 2023年的 AI 相关股 |
| 拥挤交易 | 所有人都在买的资产 |

### 应对策略

- 不要追高
- 如果已经涨很多，考虑减仓
- 反向思考：拥挤 = 风险

```
争地警示信号：

✓ 媒体大肆报道
✓ 出租车司机在谈论
✓ 估值创历史新高
✓ "这次不一样"的声音
✓ IPO 热潮

历史教训：
- 2000年科技股
- 2021年 SPAC
- 2022年加密货币
```

---

## 第四类：交地 — 流动区

### 定义

> "我可以往，彼可以来，为交地。"

四通八达之地。投资中，这是**高流动性**的市场。

### 投资对应

| 市场 | 流动性 |
|-----|--------|
| S&P 500 成分股 | 极高 |
| 主要货币对 | 极高 |
| 大盘 ETF | 极高 |
| 国债市场 | 极高 |

### 应对策略

- 优先投资高流动性资产
- 进出容易，成本低
- 避免"卖不掉"的风险

```
流动性规则：

✓ 日均成交量 > 100万股
✓ 买卖价差 < 0.1%
✓ 市值 > $10亿

低流动性警示：
✗ 日成交量 < 10万股
✗ 买卖价差 > 1%
✗ 连续多日无交易
```

---

## 第五类：衢地 — 关联区

### 定义

> "诸侯之地三属，先至而得天下之众者，为衢地。"

与多国接壤的战略要地。投资中，这是**与系统风险高度关联**的区域。

### 投资对应

| 风险类型 | 例子 |
|---------|------|
| 利率敏感 | 银行股、REITs |
| 经济周期 | 周期股（汽车、航空） |
| 大宗商品 | 能源股、矿业股 |
| 地缘政治 | 跨国公司、新兴市场 |

### 应对策略

- 了解持仓的系统性风险暴露
- 分散到不同的"衢地"
- 对冲极端风险

```
衢地分析：

问自己：
- 利率上升会怎样影响？
- 经济衰退会怎样影响？
- 美元走强会怎样影响？
- 地缘冲突会怎样影响？

如果答案都是"负面"→ 仓位过于集中
```

---

## 第六类：重地 — 重仓区

### 定义

> "入人之地深，背城邑多者，为重地。"

深入敌境，退路被切断。投资中，这是**重仓且被套**的情况。

### 投资对应

| 情况 | 特点 |
|-----|------|
| 单只股票 > 20% | 过度集中 |
| 成本线以下 | 被深度套牢 |
| 无法轻易卖出 | 心理或税务约束 |

### 应对策略

- 避免进入"重地"
- 如果已经在"重地"，制定退出计划
- 不要加仓到"重地"

```
重地规则：

单只股票占比警戒线：
黄灯：10-15%
红灯：> 15%
危险：> 20%

应对：
1. 不再加仓
2. 分批减仓
3. 对冲下行风险
```

---

## 第七类：圮地 — 困境区

### 定义

> "山林、险阻、沮泽，凡难行之道，为圮地。"

难以通行之地。投资中，这是**流动性陷阱**。

### 投资对应

| 情况 | 例子 |
|-----|------|
| 小盘股暴跌 | 买盘消失 |
| 私募投资 | 锁定期无法退出 |
| 房地产 | 卖出需要时间 |
| 垃圾债券 | 恐慌时没有买家 |

### 应对策略

- 限制低流动性资产的比例
- 预留足够的流动性缓冲
- 不要在需要用钱时被迫卖出

```
圮地案例：2008年金融危机

雷曼兄弟持有大量MBS
- 面值：$1000亿
- 市场价：无人出价
- 被迫按 $0.20 卖出

教训：流动性风险会突然爆发
```

---

## 第八类：围地 — 被套区

### 定义

> "所由入者隘，所从归者迂，彼寡可以击吾之众者，为围地。"

进入容易，退出困难。投资中，这是**深度被套**的状态。

### 投资对应

| 情况 | 心理 |
|-----|------|
| 亏损 30%+ | "不卖就不算亏" |
| 补仓越补越深 | 成本越来越高 |
| 基本面恶化 | 但不愿承认错误 |

### 应对策略

- 诚实评估：如果现在没有持仓，会买吗？
- 如果答案是"不"，考虑卖出
- 止损是承认错误，不是失败

```
围地检查清单：

□ 当初买入的理由还成立吗？
□ 基本面是否恶化？
□ 如果现在没有，会买吗？
□ 这笔资金有更好的去处吗？
□ 还在等什么？

如果3个以上答案不利，考虑止损。
```

---

## 第九类：死地 — 危机区

### 定义

> "疾战则存，不疾战则亡者，为死地。"

背水一战，必须拼死求生。投资中，这是**生存危机**。

### 投资对应

| 情况 | 例子 |
|-----|------|
| 公司破产风险 | 雷曼、安然 |
| 爆仓风险 | 杠杆过高 |
| 生活资金告急 | 动用了不该动的钱 |

### 应对策略

**最好的策略是永远不要进入死地。**

```
避免死地的规则：

✗ 不用杠杆（或极小杠杆）
✗ 不投资无法承受亏损的资金
✗ 不在单只股票上下重注
✗ 不相信"永远涨"

如果进入死地：
→ 立即止损
→ 保存本金
→ 活下来最重要
```

## 九地综合风险评估

### 你的持仓在哪个地形？

```
检查每一笔持仓：

散地（安全）：□ 现金、货币基金
轻地（试探）：□ 新建的小仓位
争地（竞争）：□ 热门股、拥挤交易
交地（流动）：□ 大盘蓝筹
衢地（关联）：□ 系统性风险暴露
重地（重仓）：□ 占比过高的持仓
圮地（困境）：□ 流动性差的资产
围地（被套）：□ 深度亏损的持仓
死地（危机）：□ 生存风险
```

### 理想的分布

```
安全区（散地+交地）：60-70%
机会区（轻地+衢地）：20-30%
风险区（其他）：< 10%
危险区（重地+围地+死地）：0%
```

---

## 本章总结

### 九地对应

| 地形 | 投资情况 | 应对 |
|-----|---------|------|
| 散地 | 安全资产 | 保持 10-20% |
| 轻地 | 试探建仓 | 小仓位，有止损 |
| 争地 | 热门股 | 不追高，警惕 |
| 交地 | 高流动性 | 优先配置 |
| 衢地 | 系统风险 | 分散暴露 |
| 重地 | 重仓被套 | 减仓，不加仓 |
| 圮地 | 流动性差 | 限制比例 |
| 围地 | 深度被套 | 诚实评估，止损 |
| 死地 | 生存危机 | 永不进入 |

### 核心原则

1. **识别地形**：知道自己在哪里
2. **调整策略**：不同地形，不同打法
3. **避免危险地带**：重地、围地、死地
4. **保持流动性**：确保随时能撤退

> "善用兵者，避其锐气，击其惰归。"

知道风险在哪里，才能避开风险。
