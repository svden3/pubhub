# 釘十字架 (The Crucifixion)

馬太福音 27:27-56; 約翰福音 19:17-37

## 經文 (Scripture)

### 各各他（約翰福音 19:17-22）

#### 中文 — 和合本修訂版 (RCUV)

> ^17^耶穌背着自己的十字架出來，到了一個地方，名叫「髑髏地」，希伯來話叫「各各他」。^18^他們就在那裏把他釘在十字架上，還有兩個人和他同釘，一邊一個，耶穌在中間。^19^彼拉多又寫了一個牌子，釘在十字架上，上面寫着：「猶太人的王，拿撒勒人耶穌」。^20^有許多猶太人念這牌子，因為耶穌被釘十字架的地方靠近城，而且牌子是用希伯來、拉丁、希臘三種文字寫的。

### 「成了」（約翰福音 19:28-30）

#### 中文 — 和合本修訂版 (RCUV)

> ^28^這事以後，耶穌知道各樣的事已經成了，為使經上的話應驗，就說：「我渴了。」^29^有一個盛滿了醋的罐子放在那裏，他們就拿海綿蘸滿了醋，綁在牛膝草上，送到他口邊。^30^耶穌嘗了那醋，就說：「成了！」便低下頭，將靈魂交付神了。

#### English — ESV

> ^28^After this, Jesus, knowing that all was now finished, said (to fulfill the Scripture), "I thirst." ^29^A jar full of sour wine stood there, so they put a sponge full of the sour wine on a hyssop branch and held it to his mouth. ^30^When Jesus had received the sour wine, he said, "It is finished," and he bowed his head and gave up his spirit.

### 超自然現象（太27:51-54）

> ^51^忽然，殿的幔子從上到下裂為兩半，地也震動，石頭也崩裂，^52^墳墓也開了，已睡聖徒的身體多有起來的。^53^到耶穌復活以後，他們從墳墓裏出來，進了聖城，向許多人顯現。^54^百夫長和看守耶穌的人看見地震和所經歷的事，非常害怕，說：「這真是神的兒子！」

## 背景 (Context)

### 十字架刑罰

十字架刑是羅馬帝國對非公民最殘酷的處決方式：

| 層面 | 描述 |
|------|------|
| 肉體 | 鞭打、釘刺、長時間窒息 |
| 羞辱 | 赤身、公開展示 |
| 社會 | 最卑賤的死法，只用於奴隸和叛亂者 |
| 神學 | 「凡掛在木頭上的都是被咒詛的」（申21:23） |

### 時間線

| 時間 | 事件 |
|------|------|
| 上午9時（第三時） | 被釘十字架（可15:25） |
| 中午12時（第六時） | 遍地黑暗開始（太27:45） |
| 下午3時（第九時） | 耶穌死亡（太27:46-50） |

### 三種文字的牌子

| 文字 | 代表 |
|------|------|
| 希伯來文 | 宗教（猶太教） |
| 拉丁文 | 政治（羅馬帝國） |
| 希臘文 | 文化（希臘文明） |

全世界的語言都宣告：這是猶太人的王——這牌子比彼拉多所知道的更真實。

## 原文研讀 (Word Study)

| 希臘文 | 音譯 | 意義 | 經文 |
|--------|------|------|------|
| σταυρός | stauros | 十字架 | 約19:17 |
| Γολγοθᾶ | Golgotha | 各各他（髑髏地） | 約19:17 |
| τετέλεσται | tetelestai | 成了 | 約19:30 |
| παραδίδωμι | paradidōmi | 交付 | 約19:30 |
| καταπέτασμα | katapetasma | 幔子 | 太27:51 |

### 「成了」(Τετέλεσται)

這是希臘文完成式，有三重含義：

| 場合 | 用法 |
|------|------|
| 商業 | 債務「付清了」 |
| 法律 | 刑罰「執行完畢了」 |
| 祭祀 | 獻祭「完成了」 |

耶穌說「成了」，宣告：
- 罪債付清
- 刑罰執行完畢
- 完美的祭已經獻上

### 幔子裂開

聖殿的幔子（隔開聖所與至聖所）約60呎高、30呎寬、4吋厚——「從上到下」裂開，表明這是神的作為，不是人為。

## 解經洞見 (Commentary)

### John MacArthur 論「我的神，為甚麼離棄我？」

> 「這是十架上最深奧的話語。耶穌不是引用詩篇22篇作詩意的感嘆，而是真實經歷與父的隔絕。從永恆以來與父完美合一的聖子，此刻經歷了父『離棄』的滋味——因為祂承擔了我們的罪。『神使那無罪的替我們成為罪』（林後5:21）。這不是表演；這是真實的。聖潔的神無法與罪同在，而耶穌成為了罪。這隔絕只持續了幾個小時，但對耶穌來說，那是比肉體痛苦更難以承受的地獄。」
> — John MacArthur, *The MacArthur New Testament Commentary: Matthew 24-28*

### G. Campbell Morgan 論幔子裂開

> 「幔子從上到下裂開——這是神的手從天上撕開的。意義是明確的：進入至聖所的路打開了。從前只有大祭司一年一次可以進入的至聖所，現在對所有人敞開。希伯來書說得清楚：『我們既然...得以坦然進入至聖所，是藉着耶穌的血...是藉着他給我們開了一條又新又活的路，從幔子經過。』（來10:19-20）那幔子代表基督的身體；祂的身體被撕裂，路就打開了。」
> — G. Campbell Morgan, *The Gospel According to Matthew*

## 神學要點 (Theological Points)

### 1. 十架七言的神學

| 話語 | 指向 |
|------|------|
| 赦免他們 | 恩典——為仇敵禱告 |
| 今日在樂園 | 救恩——臨終悔改有效 |
| 看你的母親 | 人性——照顧家人 |
| 為甚麼離棄我 | 代贖——承擔罪的刑罰 |
| 我渴了 | 人性——真實的受苦 |
| 成了 | 完成——救贖大功告成 |
| 將靈魂交付 | 順服——主動捨命 |

### 2. 詩篇22篇的應驗

| 詩篇22篇 | 應驗 |
|---------|------|
| 「我的神，為甚麼離棄我？」(1節) | 太27:46 |
| 「凡看見我的都嗤笑我」(7節) | 太27:39 |
| 「他們分我的外衣」(18節) | 約19:24 |
| 「我的骨頭都脫了節」(14節) | 釘十字架的描述 |
| 「他們扎了我的手、我的腳」(16節) | 釘子 |

大衛在主前1000年寫的詩篇，精確描述了主後30年的十字架。

### 3. 幔子裂開的意義

| 舊約 | 新約 |
|------|------|
| 幔子隔開神與人 | 幔子裂開，路打開 |
| 只有大祭司進入 | 所有信徒可進入 |
| 一年一次 | 隨時可進 |
| 靠動物的血 | 靠基督的血 |

### 4. 百夫長的認信

> 「這真是神的兒子！」(太27:54)

這外邦軍官的認信，預表將來萬國歸主——福音從猶太人傳到外邦人。

## 個人反思 (Personal Reflection)

1. **你明白「成了」的意義嗎？** 救恩已經完成，不需要你添加什麼。你是否還在試圖靠行為賺取？

2. **幔子已經裂開**：你是否「坦然無懼」地進入神面前（來4:16），還是仍活在隔絕中？

3. **十架上的強盜**：臨終悔改有效——但你敢等到最後一刻嗎？今天就是悔改的日子。

4. **「為甚麼離棄我？」** 耶穌經歷被離棄，使你永不被離棄。這給你的苦難帶來什麼盼望？
