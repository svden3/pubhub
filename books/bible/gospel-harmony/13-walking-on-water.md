馬太福音 14:22-36; 約翰福音 6:16-21

## 背景 (Context)

### 緊接在五餅二魚之後

這神蹟發生在五餅二魚的當晚。約翰特別記載這次序，因為兩個神蹟一起揭示耶穌的身份：
- 五餅二魚：供應者（創造的權柄）
- 水上行走：主宰者（自然界的權柄）

### 時間與地點

| 細節 | 意義 |
|------|------|
| 「四更天」 | 凌晨3:00-6:00（羅馬時制） |
| 「五六公里」 | 湖的中央 |
| 「風不順」 | 門徒划了整夜也沒有多少進展 |
| 「天已經黑了」 | 約翰用「黑暗」象徵門徒沒有耶穌的處境 |

### 加利利海的特性

加利利海（又稱提比哩亞海）：
- 低於海平面約200米
- 被四周山丘包圍
- 冷空氣從山谷急降，造成突發風暴
- 漁民對此習以為常，但這次不同

## 原文研讀 (Word Study)

| 希臘文 | 音譯 | 意義 | 經文 |
|--------|------|------|------|
| φάντασμα | phantasma | 幽靈/鬼怪 | 太14:26 |
| θαρσέω | tharseō | 放心/振作 | 太14:27 |
| ἐγώ εἰμι | egō eimi | 是我/我是 | 太14:27 |
| ὀλιγόπιστος | oligopistos | 小信的 | 太14:31 |
| διστάζω | distazō | 懷疑/猶豫 | 太14:31 |

### 「是我」(ἐγώ εἰμι) 的神學意義

耶穌說「是我」（egō eimi），在希臘文中與出埃及記3:14神對摩西啟示的聖名相同。這不只是自我認同，更是神性的宣告：

| 舊約 | 新約 |
|------|------|
| 「我是自有永有的」(出3:14) | 「是我，不要怕」(太14:27) |
| 神在西奈山啟示聖名 | 耶穌在風浪中宣告神性 |
| 神差遣摩西 | 耶穌差遣門徒 |

### 「小信」的原文分析

ὀλιγόπιστος (oligopistos) = ὀλίγος (小/少) + πίστις (信心)

耶穌對彼得的責備不是「沒有信」，而是「小信」。彼得有信心踏出船，但信心不足以持續行走。

## 解經洞見 (Commentary)

### John MacArthur 論彼得的信心

> 「彼得是唯一敢從船上下來的人——這需要極大的信心。但他的失敗也是可預見的：他開始時眼目注視耶穌，因此能行走在水上；當他『看見風勢猛烈』，把注意力從耶穌轉到環境，就開始下沉。這是所有基督徒生命的寫照：信心行走的關鍵不在於風浪的大小，而在於你看的是誰。彼得沒有沉下去，因為他立刻呼求『主啊，救我！』這三個字是有史以來最短、最有效的禱告之一。耶穌『趕緊伸手拉住他』——祂永遠不會讓呼求祂的人沉沒。」
> — John MacArthur, *The MacArthur New Testament Commentary: Matthew 8-15*

### G. Campbell Morgan 論夜間的禱告

> 「耶穌『獨自上山去禱告』——在餵飽五千人之後，在水上行走之前。這個次序至關重要。祂剛才經歷了一個偉大的公開神蹟，群眾想強逼祂作王；祂需要獨處，需要與父交通。接下來祂將在最不可能的時刻（凌晨四更天）、最不可能的地方（暴風的海上）出現，向門徒啟示祂是誰。禱告是能力的泉源，也是引導的管道。每一次服事之前之後，我們都需要這樣的獨處與神相交。」
> — G. Campbell Morgan, *The Gospel According to Matthew*

## 神學要點 (Theological Points)

### 1. 舊約的呼應

「在海面上行走」在舊約中是神獨有的權柄：

> **約伯記 9:8**：「他獨自鋪張蒼天，步行在海浪之上。」

> **詩篇 77:19**：「你的道在海中，你的路在大水中；你的腳蹤無人知道。」

當耶穌在海面上行走時，祂正在做只有神才能做的事。

### 2. 神性的顯現（Theophany）

這是一個神性顯現的時刻：

| 元素 | 意義 |
|------|------|
| 在水上行走 | 超越自然律 |
| 「我是」的宣告 | 神聖的名 |
| 平靜風浪 | 掌管自然 |
| 門徒敬拜 | 認出神性 |

### 3. 門徒的回應

> 「在船上的人都拜他，說：『你真是神的兒子！』」(14:33)

這是馬太福音中門徒第一次集體敬拜耶穌並承認祂的神性。

| 之前 | 之後 |
|------|------|
| 看見神蹟就驚奇 | 敬拜耶穌為神子 |
| 不明白五餅二魚 (可6:52) | 認出耶穌是神的兒子 |

### 4. 彼得的經歷——信心的功課

| 階段 | 彼得的狀態 | 屬靈功課 |
|------|-----------|---------|
| 1. 在船上 | 害怕 | 與主分離時，風浪可怕 |
| 2. 踏出去 | 信心 | 憑信心回應主的呼召 |
| 3. 看風浪 | 下沉 | 眼目轉離主，就失敗 |
| 4. 呼求主 | 得救 | 呼求主的人必被拉住 |
| 5. 上船後 | 敬拜 | 經歷後更認識主 |

## 個人反思 (Personal Reflection)

1. **你的船在哪裏？** 當你在人生的風浪中掙扎，耶穌在凌晨四更天向你走來。你認得出祂嗎？

2. **你敢踏出去嗎？** 彼得是唯一踏出船的人。信心需要行動——你願意冒險踏出安全區嗎？

3. **你看的是風浪還是耶穌？** 彼得開始時注視耶穌，能行在水上；當他看風浪，就開始下沉。你的眼目注視的是什麼？

4. **「主啊，救我！」** 當你開始下沉，最重要的禱告只有三個字：「主啊，救我！」你願意這樣呼求嗎？

5. **祂永遠趕緊伸手**：耶穌「趕緊」伸手拉住彼得。祂對你的回應也是如此——不會遲延、不會猶豫。
