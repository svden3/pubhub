## 背景 (Context)

### 山上寶訓的地點與場景

**傳統地點**：加利利海西北岸的山丘，今稱「八福山」（Mount of Beatitudes）
**聽眾**：門徒為主，但群眾也在旁聽（7:28-29）
**姿勢**：耶穌「坐下」教導——拉比正式教學的姿勢

### 山上寶訓與摩西的對比

| 摩西 | 耶穌 |
|------|------|
| 在西奈山領受律法 | 在山上頒布天國法則 |
| 傳達神的話：「耶和華說…」 | 以神的權柄說：「我告訴你們…」 |
| 刻在石版上 | 刻在心版上 |
| 外在行為的規範 | 內在動機的要求 |

### 山上寶訓的結構

| 段落 | 內容 | 經文 |
|------|------|------|
| 八福 | 天國子民的品格 | 5:1-12 |
| 鹽與光 | 天國子民的影響 | 5:13-16 |
| 律法的成全 | 超越律法的義 | 5:17-48 |
| 真實的敬虔 | 施捨、禱告、禁食 | 6:1-18 |
| 天上的財寶 | 單純的眼光 | 6:19-34 |
| 論斷與祈求 | 對人與對神 | 7:1-12 |
| 兩條路 | 抉擇與根基 | 7:13-27 |

## 原文研讀 (Word Study)

| 希臘文 | 音譯 | 意義 | 經文 |
|--------|------|------|------|
| μακάριος | makarios | 有福的/蒙福的 | 5:3-11 |
| πτωχός | ptōchos | 貧窮/極度匱乏 | 5:3 |
| πραΰς | praus | 謙和/溫柔 | 5:5 |
| δικαιοσύνη | dikaiosynē | 義/公義 | 5:6, 10, 20 |
| τέλειος | teleios | 完全/成熟 | 5:48 |

### 「有福」(μακάριος) 的真義

μακάριος 不只是「快樂」，而是一種超越環境的、神所賜的、恆久的福樂狀態。它描述的是一種從神來的祝福，與外在境況無關。

### 「心靈貧窮」的原文分析

πτωχοὶ τῷ πνεύματι（poor in spirit）：
- πτωχός：極度貧窮，一無所有（比 πένης 更強）
- τῷ πνεύματι：在靈裏/關於心靈

這不是指物質貧窮，而是靈性上認識到自己在神面前完全破產、一無所有。

## 解經洞見 (Commentary)

### John MacArthur 論八福的次序

> 「八福不是隨機排列，而是有邏輯次序。它從『心靈貧窮』開始——認識到自己靈性破產、在神面前一無所有。這認識自然導致『哀慟』——為自己的罪和敗壞而憂傷。哀慟產生『謙和』——不再自我辯護，在神和人面前謙卑。然後是『飢渴慕義』——渴望神的義來充滿自己。這四福是向上（對神），接下來四福是向外（對人）：憐憫、清心、締造和平，最後是預備承受因義受逼迫。八福描繪的是每一個真基督徒的心路歷程。」
> — John MacArthur, *The MacArthur New Testament Commentary: Matthew 1-7*

### G. Campbell Morgan 論主禱文的結構

> 「主禱文是完美的禱告模式。它以『我們在天上的父』開始——這是一切禱告的基礎：我們是兒女，祂是父親。接下來三個祈求都是關於神的——願人尊祂的名為聖，願祂的國降臨，願祂的旨意成就。只有在這之後，才是關於我們的需要——日用飲食、赦免、脫離試探。這次序至關重要：先是神的榮耀，然後是我們的需要。許多人的禱告失敗，正因為次序顛倒。」
> — G. Campbell Morgan, *The Gospel According to Matthew*

## 神學要點 (Theological Points)

### 1. 八福的神學結構

八福可分為兩組，每組以「天國是他們的」結束（首尾呼應）：

**第一組（向上的關係）**：
| 福 | 條件 | 應許 |
|---|------|------|
| 1 | 心靈貧窮 | 天國是他們的 |
| 2 | 哀慟 | 必得安慰 |
| 3 | 謙和 | 必承受土地 |
| 4 | 飢渴慕義 | 必得飽足 |

**第二組（向外的關係）**：
| 福 | 條件 | 應許 |
|---|------|------|
| 5 | 憐憫人 | 必蒙憐憫 |
| 6 | 清心 | 必得見神 |
| 7 | 締造和平 | 必稱為神的兒子 |
| 8 | 為義受迫害 | 天國是他們的 |

### 2. 律法的成全

> 「莫想我來要廢掉律法和先知。我來不是要廢掉，乃是要成全。」(5:17)

耶穌「成全」律法的方式：

| 舊約律法 | 耶穌的教導 |
|---------|-----------|
| 不可殺人 | 不可恨人 (5:21-22) |
| 不可姦淫 | 不可動淫念 (5:27-28) |
| 休妻要給休書 | 不可離婚（除淫亂外）(5:31-32) |
| 不可背誓 | 不要起誓，是就說是 (5:33-37) |
| 以眼還眼 | 不要與惡人作對 (5:38-42) |
| 愛鄰舍恨仇敵 | 愛你的仇敵 (5:43-48) |

### 3. 內在的義 vs. 外在的義

> 「你們的義若不勝於文士和法利賽人的義，斷不能進天國。」(5:20)

法利賽人的義是外在的、表演性的、可計算的。耶穌要求的義是內在的、真實的、從心發出的。

| 法利賽人的義 | 天國的義 |
|------------|---------|
| 不殺人 | 不恨人 |
| 不姦淫 | 不動淫念 |
| 做給人看 | 做在暗中 |
| 追求人的稱讚 | 追求神的稱讚 |
| 外在遵守 | 內在更新 |

### 4. 兩條路的抉擇

山上寶訓以嚴肅的抉擇結束：

| 對比 | 寬門大路 | 窄門小路 |
|------|---------|---------|
| 入口 | 容易 | 困難 |
| 人數 | 多 | 少 |
| 終點 | 滅亡 | 生命 |
| 根基 | 沙土 | 磐石 |
| 結局 | 倒塌 | 屹立 |

## 個人反思 (Personal Reflection)

1. **八福中的你**：八福描述的是真門徒的品格。你在哪幾個福分上經歷真實？哪幾個是你需要成長的？

2. **你的禱告次序**：你的禱告是以神的榮耀為先，還是以自己的需要為先？

3. **外在還是內在？** 你的敬虔是做給人看的表演，還是做在暗中的真實？

4. **你走哪條路？** 窄門小路雖然艱難，但通往生命。你願意選擇這條路嗎？

5. **聽而行道**：你是「聽」山上寶訓的人，還是「行」山上寶訓的人？
