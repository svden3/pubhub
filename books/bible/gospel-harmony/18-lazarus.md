# 拉撒路復活 (The Raising of Lazarus)

約翰福音 11:1-57

## 經文 (Scripture)

### 中文 — 和合本修訂版 (RCUV)

## 背景 (Context)

### 約翰福音第七個神蹟

這是約翰福音記載的第七個也是最後一個神蹟——七神蹟的高峰。它直接預表耶穌自己的復活。

| 神蹟序列 | 意義的遞進 |
|---------|-----------|
| 1. 水變酒 | 彰顯榮耀 |
| 2. 醫大臣之子 | 話語的權柄 |
| 3. 醫畢士大癱子 | 安息日的主 |
| 4. 五餅二魚 | 生命的糧 |
| 5. 水上行走 | 超越自然 |
| 6. 醫生來瞎眼 | 世界的光 |
| **7. 復活拉撒路** | **復活與生命** |

### 伯大尼的位置

伯大尼位於橄欖山東坡，距耶路撒冷約3公里（「離耶路撒冷不遠，約有六里路」11:18）。這是耶穌在耶路撒冷附近最常住宿的地方。

### 「四天」的意義

猶太人相信：
- 靈魂在死後三天還在身體附近徘徊
- 第四天開始，屍體開始腐爛，靈魂完全離去
- 因此，四天後復活表明這是真正的死亡，不是假死或昏迷

## 原文研讀 (Word Study)

| 希臘文 | 音譯 | 意義 | 經文 |
|--------|------|------|------|
| ἀνάστασις | anastasis | 復活 | 11:25 |
| ζωή | zōē | 生命 | 11:25 |
| ἐμβριμάομαι | embrimaomai | 悲嘆/激動 | 11:33, 38 |
| δακρύω | dakryō | 流淚 | 11:35 |
| κράζω | krazō | 大聲呼叫 | 11:43 |

### 「我就是復活與生命」

Ἐγώ εἰμι ἡ ἀνάστασις καὶ ἡ ζωή

這是約翰福音七個「我是」宣告的第五個。注意定冠詞「那/這」(ἡ)——耶穌不是說「我給人復活」，而是說「我就是那復活本身」。

### 「耶穌哭了」(11:35)

Ἐδάκρυσεν ὁ Ἰησοῦς

這是新約聖經中最短的一節經文（希臘文只有兩個字）。δακρύω 表示安靜的流淚（不同於 κλαίω 的大聲哭號）。

### 「悲嘆」的深層含義

ἐμβριμάομαι 原意是「在鼻息中震怒」，通常用於表達憤怒。耶穌可能是對死亡本身——這個因罪進入世界的仇敵——感到憤怒。

## 解經洞見 (Commentary)

### John MacArthur 論耶穌的眼淚

> 「『耶穌哭了』——這三個字包含了極深的神學。耶穌不是因為拉撒路死了而哭——祂知道幾分鐘後就要使他復活。祂哭是因為祂看見死亡給祂所愛的人帶來的痛苦，看見罪的咒詛如何蹂躪這個世界。這是神對人類苦難的心碎。同時，祂『心裏悲嘆』(ἐμβριμάομαι) 帶有憤怒的意味——祂對死亡這個仇敵感到義憤。這位即將在十字架上擊敗死亡的主，此刻站在墳墓前，為人類的處境流淚，也為即將施行的拯救而激動。」
> — John MacArthur, *The MacArthur New Testament Commentary: John 1-11*

### G. Campbell Morgan 論馬大的信心

> 「馬大對耶穌說：『即使是現在，我也知道，你無論向神求甚麼，神都會賜給你。』這句話顯示她心中有一絲希望，卻又不敢明說。耶穌說：『你弟弟會復活的。』她的回應是神學正確卻缺乏期待的：『我知道在末日復活的時候，他會復活。』她相信末日的復活，卻沒想到復活就站在她面前。耶穌的回答是福音書中最偉大的宣告之一：『復活在我，生命也在我。』不是『我會使人復活』，而是『我就是復活本身』。在基督裏，末日的復活已經預先臨到。」
> — G. Campbell Morgan, *The Gospel According to John*

## 神學要點 (Theological Points)

### 1. 「我是復活與生命」的意義

| 層面 | 含義 |
|------|------|
| 復活 (ἀνάστασις) | 勝過死亡，從死裏復起 |
| 生命 (ζωή) | 永恆的生命，神自己的生命 |
| 合一 | 復活與生命在基督裏合而為一 |

### 2. 耶穌延遲的目的

> 「這病不至於死，而是為了神的榮耀。」(11:4)

| 人的角度 | 神的角度 |
|---------|---------|
| 耶穌應該立刻來 | 等待是有目的的 |
| 拉撒路不應該死 | 死亡讓復活更榮耀 |
| 神遲延是不關心 | 神遲延是為了更大的益處 |

耶穌聽到消息後，「在所住的地方仍住了兩天」(11:6)——這是刻意的等待，為要彰顯更大的榮耀。

### 3. 三個復活的對比

| 復活事件 | 特點 |
|---------|------|
| 睚魯的女兒 | 剛死 |
| 拿因城寡婦的兒子 | 出殯中 |
| **拉撒路** | **死了四天，已開始腐爛** |

拉撒路的復活是最無可置疑的——沒有人能說他只是昏迷。

### 4. 這神蹟的後果

> ^45^「那些來看馬利亞的猶太人，見了耶穌所做的事，有很多人就信了他。」
> ^53^「從那天起，他們就商議要殺耶穌。」

這神蹟產生了兩極的反應：
- 許多人信了
- 公會決定殺耶穌

拉撒路的復活直接導致了耶穌的受難——該亞法說「一個人替百姓死」(11:50)。

### 5. 預表耶穌自己的復活

| 拉撒路 | 耶穌 |
|--------|------|
| 在墳墓裏四天 | 在墳墓裏三天 |
| 被石頭封住 | 被石頭封住 |
| 被別人呼喚出來 | 自己從死裏復活 |
| 復活後仍會再死 | 復活後永不再死 |

## 個人反思 (Personal Reflection)

1. **你在等待神嗎？** 當馬大和馬利亞等待耶穌時，祂似乎遲延了。你正在經歷神的「遲延」嗎？記得：祂的時間表總是為了更大的榮耀。

2. **你相信復活嗎？** 不只是末日的復活，而是此刻、今天——「復活在我，生命也在我」。這對你今天的處境意味著什麼？

3. **耶穌與你同哭**：當你悲傷時，耶穌也為你流淚。祂不是冷漠的神，而是會為你心碎的主。

4. **什麼是你的「墳墓」？** 有什麼在你生命中已經「死」了——夢想、關係、盼望？聽見主的聲音：「出來！」
