## 背景 (Context)

### 講論的時機

這是耶穌被捕前幾個小時對門徒說的話——最後的教導、最深的安慰、最重要的應許。

| 背景 | 門徒的狀態 |
|------|-----------|
| 耶穌預言被賣 | 困惑 |
| 耶穌說要離開 | 憂愁 |
| 彼得說要跟從 | 自信卻即將跌倒 |

### 講論的主題

| 章節 | 主題 |
|------|------|
| 約14 | 道路、真理、生命；聖靈的應許 |
| 約15 | 真葡萄樹；彼此相愛；世界的恨 |
| 約16 | 聖靈的工作；憂愁變喜樂 |
| 約17 | 大祭司的禱告（下一章詳述） |

### 第六個「我是」：道路、真理、生命

這是約翰福音七個「我是」宣告中最全面的一個：

| 「我是」 | 意義 |
|---------|------|
| 道路 | 到父那裏去的唯一途徑 |
| 真理 | 關於神的完全啟示 |
| 生命 | 永恆生命的源頭 |

### 「保惠師」(Paraklētos)

παράκλητος 可翻譯為：
- 保惠師（Comforter）——安慰的那一位
- 幫助者（Helper）——來幫助的那一位
- 辯護者（Advocate）——在旁辯護的那一位

在約翰壹書2:1，這詞用於基督自己——「我們有一位中保」。

## 解經洞見 (Commentary)

### John MacArthur 論基督的獨一性

> 「『若不藉着我，沒有人能到父那裏去』——這是基督教最排他性也最充滿恩典的宣告。排他性：只有一條路，不是許多路。其他宗教、其他教主、其他努力，都不能使人到父那裏。恩典性：這條路是敞開的，對所有人都有效。耶穌不是說『若不藉着猶太人』或『若不藉着有學問的人』，而是『若不藉着我』。任何人，無論背景，只要來到基督面前，就有路到父那裏。」
> — John MacArthur, *The MacArthur New Testament Commentary: John 12-21*

### G. Campbell Morgan 論葡萄樹的比喻

> 「『我是真葡萄樹』——『真』(ἀληθινή) 意味著祂是舊約以色列（被稱為葡萄樹，賽5:1-7）的實體和成全。以色列失敗了，不結果子；基督是那真正結果子的葡萄樹。門徒是枝子——不是獨立的植物，而是連於樹的枝子。枝子唯一的責任是『常在』(μένω) 葡萄樹裏，果子就自然結出。這不是努力的宗教，而是連結的生命。離了祂，我們不能做甚麼；常在祂裏面，就多結果子。」
> — G. Campbell Morgan, *The Gospel According to John*

## 神學要點 (Theological Points)

### 1. 第七個「我是」：真葡萄樹

| 元素 | 意義 |
|------|------|
| 葡萄樹 | 基督 |
| 栽培的人 | 父神 |
| 枝子 | 信徒 |
| 果子 | 生命改變的證據 |
| 剪掉 | 審判不結果的 |
| 修剪 | 管教使更結果 |

### 2. 聖靈的三重工作

對世人（約16:8-11）：
| 工作 | 說明 |
|------|------|
| 叫人知罪 | 因為不信基督 |
| 叫人知義 | 因為基督到父那裏去 |
| 叫人知審判 | 因為撒但已受審判 |

對信徒（約14:26; 16:13-15）：
| 工作 | 說明 |
|------|------|
| 教導 | 教導並使想起主的話 |
| 引導 | 引導進入一切真理 |
| 榮耀基督 | 把從基督領受的告訴我們 |

### 3. 憂愁變喜樂

> 「你們現在也有憂愁，但我會再見到你們，你們的心就會喜樂，這喜樂沒有人能奪去。」(約16:22)

| 現在 | 將來 |
|------|------|
| 憂愁（門徒失去耶穌） | 喜樂（復活的主回來） |
| 如產婦的陣痛 | 如產後的喜樂 |
| 暫時的 | 永久的，沒人能奪去 |

### 4. 奉主名祈求

> 「你們若奉我的名向父求甚麼，他必賜給你們。」(約16:23)

「奉我的名」不是魔法公式，而是：
- 按照基督的旨意
- 以基督的權柄
- 為基督的榮耀

## 個人反思 (Personal Reflection)

1. **你的心憂愁嗎？** 耶穌說「你們心裏不要憂愁」，接著給出理由：信神、信基督、有天家、祂必再來。這些應許如何安慰你今天的憂慮？

2. **你常在基督裏嗎？** 枝子離了葡萄樹就枯乾。你與基督的連結是真實活潑的，還是表面疏遠的？

3. **你經歷聖靈的工作嗎？** 聖靈教導、引導、榮耀基督。你在日常生活中經歷這些嗎？

4. **你的喜樂來自哪裏？** 世界的喜樂會被奪去，基督給的喜樂沒人能奪去。你追求的是哪一種？
