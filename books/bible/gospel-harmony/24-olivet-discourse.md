# 橄欖山講論 (The Olivet Discourse)

馬太福音 24:1-51; 馬可福音 13:1-37

## 經文 (Scripture)

### 中文 — 和合本修訂版 (RCUV)

#### 聖殿毀滅的預言（太24:1-2）

> ^1^耶穌出了聖殿，正往前走的時候，門徒前來，把聖殿的建築指給他看。^2^耶穌對他們說：<red>「你們不是看見這一切嗎？我實在告訴你們，這裏將沒有一塊石頭留在另一塊石頭上面不被拆毀的。」</red>

#### 末世的預兆（太24:4-14）

> ^4^耶穌回答他們：<red>「你們要謹慎，免得有人迷惑你們。^5^因為將有許多人冒我的名來，說『我是基督』，並且要迷惑許多人。^6^你們將聽見戰爭和戰爭的風聲，不要驚慌！因為這些事必須發生，但這還不是終結。^7^民要攻打民，國要攻打國；多處必有飢荒、地震。^8^這一切不過是災難的起頭。」</red>

> ^14^<red>「這天國的福音要傳遍天下，對萬民作見證，然後終結才來到。」</red>

#### 人子再來（太24:29-31）

> ^29^<red>「那些日子的災難一過去，日頭就變黑了，月亮也不放光，星辰要從天上墜落，天上的萬象都要震動。^30^那時，人子的預兆要顯在天上，地上的萬族都要哀哭。他們要看見人子帶着能力和大榮耀，駕着天上的雲來臨。^31^他要差遣使者，用大聲的號角，把他的選民從四方，從天這邊到天那邊，都招聚了來。」</red>

#### 警醒的命令（太24:42-44）

> ^42^<red>「所以，你們要警醒，因為不知道你們的主哪一天來到。^43^家主若知道幾更天有盜賊來，他就會警醒，不讓人挖穿房屋；這是你們所知道的。^44^所以，你們也要預備，因為在你們想不到的時候，人子就來了。」</red>

## 背景 (Context)

### 地點與時間

這是受難週週二（有些學者認為是週三），耶穌離開聖殿後，在橄欖山上對門徒發表的最長的末世論講論。

### 門徒的三個問題

> 「請告訴我們，甚麼時候會有這些事？你來臨和這世代的終結有甚麼預兆呢？」(太24:3)

| 問題 | 關於 |
|------|------|
| 什麼時候有這些事？ | 聖殿被毀 |
| 你來臨有什麼預兆？ | 基督再來 |
| 世代終結有什麼預兆？ | 末日 |

### 兩個應驗層面

這段講論有「近期」和「遠期」兩個應驗層面：

| 近期（主後70年） | 遠期（末日） |
|----------------|-------------|
| 聖殿被毀 | 基督再來 |
| 耶路撒冷被圍 | 全地受審判 |
| 猶太人分散 | 選民被招聚 |

## 原文研讀 (Word Study)

| 希臘文 | 音譯 | 意義 | 經文 |
|--------|------|------|------|
| παρουσία | parousia | 來臨/降臨 | 太24:3, 27 |
| συντέλεια | synteleia | 終結/完成 | 太24:3 |
| θλῖψις | thlipsis | 災難/患難 | 太24:21, 29 |
| γρηγορέω | grēgoreō | 警醒 | 太24:42, 43 |
| ἑτοιμάζω | hetoimazō | 預備 | 太24:44 |

### 「Parousia」的含義

παρουσία 在希臘羅馬世界中用於描述君王或皇帝的正式「蒞臨」——城市會盛大迎接，是榮耀的到來。用在基督身上，指祂榮耀的再來。

### 「生產之痛」的隱喻

> 「這一切不過是災難(ὠδίνων)的起頭。」(太24:8)

ὠδίν 是「生產之痛」——暗示：
1. 痛苦會逐漸加劇
2. 痛苦是為了新生命（新世代）
3. 時間是有限的，結局必來

## 解經洞見 (Commentary)

### John MacArthur 論主後70年的應驗

> 「耶穌預言的聖殿毀滅在主後70年精確應驗。提多的軍隊圍困耶路撒冷，聖殿被焚燒時，金箔熔化流入石縫。士兵為了取出黃金，真的把每一塊石頭都翻起來——『不會留下一塊石頭在另一塊石頭上』。約瑟夫記載了恐怖的細節：饑荒導致人吃人，超過一百萬人死亡，剩下的被擄。但這只是『預表』，更大的災難將在末日來臨。」
> — John MacArthur, *The MacArthur New Testament Commentary: Matthew 24-28*

### G. Campbell Morgan 論警醒的命令

> 「橄欖山講論的實際應用集中在一個詞：『警醒』。耶穌沒有給我們計算日期的方程式，反而說『那日子、那時辰沒有人知道』(太24:36)。為什麼？因為重點不是預測時間，而是隨時準備好。忠心管家的比喻(太24:45-51)、十個童女的比喻(太25:1-13)、才幹的比喻(太25:14-30)——都指向同一點：主人會在意想不到的時候回來，關鍵是你在做什麼。」
> — G. Campbell Morgan, *The Gospel According to Matthew*

## 神學要點 (Theological Points)

### 1. 末世的預兆

| 預兆 | 經文 |
|------|------|
| 假基督、假先知 | 太24:5, 11, 24 |
| 戰爭、飢荒、地震 | 太24:6-7 |
| 信徒被逼迫 | 太24:9 |
| 許多人跌倒 | 太24:10 |
| 不法的事增多、愛心冷淡 | 太24:12 |
| 福音傳遍天下 | 太24:14 |

### 2. 大災難

> 「因為那時必有大災難，從世界的起頭直到如今，沒有這樣的災難，將來也必沒有。」(太24:21)

這是人類歷史上最黑暗的時期，但神會：
- 為選民縮短那些日子(太24:22)
- 保守祂的百姓

### 3. 基督再來的特徵

| 特徵 | 經文 |
|------|------|
| 如閃電——明顯可見 | 太24:27 |
| 帶著能力和大榮耀 | 太24:30 |
| 駕雲來臨 | 太24:30 |
| 差遣天使招聚選民 | 太24:31 |
| 如賊——出人意外 | 太24:43 |

### 4. 三個比喻的教訓

| 比喻 | 教訓 |
|------|------|
| 忠心與惡僕(太24:45-51) | 忠心服事，不要懈怠 |
| 十童女(太25:1-13) | 隨時預備，不可遲鈍 |
| 才幹(太25:14-30) | 善用恩賜，不可埋藏 |

### 5. 綿羊與山羊的審判

> 「這些事你們既做在我這弟兄中最小的一個身上，就是做在我身上了。」(太25:40)

| 綿羊 | 山羊 |
|------|------|
| 給餓的吃、渴的喝 | 沒有這樣做 |
| 收留客旅 | 沒有這樣做 |
| 給赤身的穿 | 沒有這樣做 |
| 探望病人、囚犯 | 沒有這樣做 |
| 進入永生 | 進入永刑 |

這不是說行為賺取救恩，而是真信心必產生行為。

## 個人反思 (Personal Reflection)

1. **你警醒嗎？** 「你們不知道你們的主哪一天來到。」如果基督今天再來，你預備好了嗎？

2. **你是忠心的僕人嗎？** 忠心僕人在主人不在時仍然盡責。你在沒有人監督時如何生活？

3. **你的燈有油嗎？** 五個愚拙童女有燈卻沒有油（外表有形式卻沒有聖靈的生命）。你的信仰是真實的還是表面的？

4. **你如何使用你的「才幹」？** 神給你的恩賜、資源、時間——你在使用它們服事神的國度嗎？

5. **你如何對待「最小的」？** 愛鄰舍的實際行動是真信心的證據。你怎樣對待貧窮、軟弱、被忽視的人？
