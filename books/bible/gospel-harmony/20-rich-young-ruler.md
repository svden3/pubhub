# 財主與天國 (The Rich Young Ruler)

馬太福音 19:16-30; 馬可福音 10:17-31; 路加福音 18:18-30

## 經文 (Scripture)

### 中文 — 和合本修訂版 (RCUV)

> ^16^有一個人來見耶穌，說：「老師，我該做甚麼善事才能得永生？」^17^耶穌對他說：<red>「你為甚麼以善事問我呢？只有一位是善的。你若要進入永生，就當遵守誡命。」</red>^18^他說：「哪些誡命？」耶穌說：<red>「就是不可殺人；不可姦淫；不可偷盜；不可作假見證；^19^當孝敬父母；又當愛鄰如己。」</red>^20^那年輕人對他說：「這一切我都遵守了，還缺少甚麼呢？」^21^耶穌對他說：<red>「你若願意作完全人，去變賣你所擁有的，分給窮人，就必有財寶在天上；然後來跟從我。」</red>^22^那年輕人聽見這話，就憂憂愁愁地走了，因為他的財產很多。

## 背景 (Context)

### 三福音書的合參

三卷對觀福音都記載這事件，各有補充：

| 福音書 | 特殊記載 |
|--------|---------|
| 馬太 | 稱他為「年輕人」(νεανίσκος) |
| 馬可 | 「跑來跪下」「耶穌看著他，就愛他」 |
| 路加 | 稱他為「官」(ἄρχων) |

綜合三處，這人是：年輕、富有、有權力、有宗教熱忱。

### 當時的財富觀念

猶太人普遍認為：
- 財富是神祝福的記號
- 富人更容易遵守律法（有時間、有資源）
- 因此財主應該更容易進天國

耶穌的話顛覆了這觀念。

### 「駱駝穿過針眼」的解釋

幾種解釋：

| 解釋 | 評估 |
|------|------|
| 耶路撒冷有個小門叫「針眼」| 沒有歷史證據 |
| 「駱駝」是「繩索」(κάμιλος) 的誤傳 | 抄本不支持 |
| 字面意思：完全不可能 | 最可能的解釋 |

耶穌用誇張手法強調：靠人自己的能力進天國是不可能的。

| τέλειος | teleios | 完全 | 太19:21 |

| δύσκολος | dyskolos | 困難 | 可10:23 |

### 「看著他，就愛他」

馬可福音10:21獨有這細節：ἐμβλέψας αὐτῷ ἠγάπησεν αὐτόν

耶穌對這年輕人不是冷漠的拒絕，而是帶著愛的挑戰。祂要求他放棄財富，正因為祂愛他——祂知道財富是他的偶像。

### 「完全」的意義

τέλειος 不是道德上的無罪，而是「完整」、「成熟」、「達到目標」。耶穌不是說他差一步就完美，而是說他需要面對核心問題——他的心歸向誰。

## 解經洞見 (Commentary)

### John MacArthur 論這人的問題

> 「這年輕人問的是『我該做甚麼善事才能得永生？』——問題的框架就已經錯了。永生不是靠做善事賺取的。耶穌引導他看見這點：如果靠遵守誡命，你其實沒有真正遵守。『愛鄰如己』——你愛窮人嗎？你願意把財產分給他們嗎？這個挑戰暴露了他的真實狀態：他聲稱遵守律法，其實他的心還是屬於財富。律法的目的是讓人看見自己的罪，但這人用律法來建立自義。耶穌撕開了他的面具。」
> — John MacArthur, *The MacArthur New Testament Commentary: Matthew 16-23*

### G. Campbell Morgan 論憂愁地離去

> 「這是福音書中最悲傷的離去之一。這人『憂憂愁愁地走了』——他沒有生氣，沒有辯駁，只是憂愁。他知道耶穌說的是對的，但他做不到。他的財富太重要了。注意：他沒有說『我不能』，而是用行動說『我不願』。這就是罪的本質——不是不能，而是不願。許多人就像這年輕人：對信仰有興趣，對天國有渴望，但當被要求放下偶像時，就轉身離開。」
> — G. Campbell Morgan, *The Gospel According to Matthew*

## 神學要點 (Theological Points)

### 1. 這人的優點與缺點

| 優點 | 缺點 |
|------|------|
| 認真尋求永生 | 靠行為自義 |
| 謙卑跪下請教 | 不認識自己的罪 |
| 道德生活良好 | 財富成為偶像 |
| 承認自己還缺少 | 不願付代價 |

### 2. 耶穌的測試

耶穌沒有要求每個人變賣一切，但祂要求每個人面對自己的偶像：

| 對象 | 偶像 | 耶穌的要求 |
|------|------|-----------|
| 財主少年 | 財富 | 變賣分給窮人 |
| 彼得等人 | 漁業 | 捨了網跟從 |
| 馬太 | 稅關 | 離開稅關 |
| 我們 | ？ | 放下你的____ |

### 3. 救恩的不可能與可能

> 「在人這是不能的，在神凡事都能。」(太19:26)

| 人的不能 | 神的能 |
|---------|--------|
| 靠行為稱義 | 因信稱義 |
| 自己改變心 | 聖靈重生 |
| 放下偶像 | 恩典釋放 |
| 賺取救恩 | 白白恩賜 |

### 4. 真正的財寶

> 「你必有財寶在天上。」(太19:21)

| 地上財寶 | 天上財寶 |
|---------|---------|
| 會朽壞 | 永存 |
| 會被偷 | 安全 |
| 暫時 | 永恆 |
| 不能帶走 | 先存在那裏 |

### 5. 跟從的代價與獎賞

> ^28^「你們這些跟從我的人，到復興的時候，人子坐在他榮耀的寶座上，你們也要坐在十二個寶座上，審判以色列十二個支派。^29^凡為我的名撇下房屋，或是弟兄、姐妹、父親、母親、兒女、田地的，必得着百倍，並且承受永生。」(太19:28-29)

| 撇下 | 得著 |
|------|------|
| 房屋、田地 | 百倍（今世的團契） |
| 家人關係 | 屬靈家人 |
| 暫時損失 | 永生 |
| 地上地位 | 天上寶座 |

## 個人反思 (Personal Reflection)

1. **你的偶像是什麼？** 不一定是金錢——可能是地位、關係、舒適、成就。如果耶穌要求你放下它來跟從祂，你會怎樣回應？

2. **你「還缺少甚麼」？** 這年輕人外表敬虔，內心卻把財富放在神之上。你生命中有沒有什麼是你不願意交給神的？

3. **你靠什麼得救？** 這人問「我該做甚麼」——這是錯誤的問題。救恩不是做出來的，是領受的。你是否還在試圖靠行為賺取神的接納？

4. **你會憂愁地離去嗎？** 這人聽見真理卻選擇離開。當福音的要求觸及你的核心，你會選擇跟從還是離開？
