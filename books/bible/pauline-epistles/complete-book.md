# 保羅書信概覽 (Pauline Epistles Overview)

## 書卷簡介

**作者**：使徒保羅 (The Apostle Paul)
**原名**：掃羅 (Saul of Tarsus)
**寫作時期**：約主後 49-67 年
**書信數量**：13 封 (傳統歸屬)
**主題**：因信稱義、基督的身體、基督徒生活

---

## 核心經文 (Key Verse)

> 我已經與基督同釘十字架，現在活著的不再是我，乃是基督在我裏面活著；並且我如今在肉身活著，是因信神的兒子而活；他是愛我，為我捨己。
> — 加拉太書 2:20

> I have been crucified with Christ. It is no longer I who live, but Christ who lives in me. And the life I now live in the flesh I live by faith in the Son of God, who loved me and gave himself for me.
> — Galatians 2:20 (ESV)

## 各書信詳細資料 (Detailed Information for Each Epistle)

### 1. 羅馬書 (Romans)

| 項目 | 內容 |
|------|------|
| **寫作日期** | 主後 57 年 |
| **寫作地點** | 哥林多 (Corinth) |
| **收信人** | 羅馬教會 (未曾親訪) |
| **章數** | 16 章 |
| **主題** | 因信稱義的福音 |
| **使徒行傳** | 徒 20:2-3 (第三次旅程末期，在希臘三個月) |
| **背景** | 保羅計劃訪問羅馬，前往西班牙宣教，先以書信闡明福音要義 |

**地理位置**：羅馬帝國首都，位於義大利半島中部，台伯河畔。

**核心經文**：
> 因為世人都犯了罪，虧缺了神的榮耀；如今卻蒙神的恩典，因基督耶穌的救贖，就白白地稱義。— 羅 3:23-24

---

### 2. 哥林多前書 (1 Corinthians)

| 項目 | 內容 |
|------|------|
| **寫作日期** | 主後 55 年 |
| **寫作地點** | 以弗所 (Ephesus) |
| **收信人** | 哥林多教會 |
| **章數** | 16 章 |
| **主題** | 教會紀律與合一 |
| **使徒行傳** | 徒 18:1-18 (建立教會); 徒 19 (以弗所期間寫信) |
| **背景** | 哥林多教會面臨分裂、淫亂、訴訟、偶像祭物、屬靈恩賜等問題 |

**地理位置**：希臘南部亞該亞省，控制兩個港口（堅革哩和利凱昂），商業繁榮但道德敗壞。

**核心經文**：
> 你們要追求愛，也要切慕屬靈的恩賜，其中更要羡慕的是作先知講道。— 林前 14:1

---

### 3. 哥林多後書 (2 Corinthians)

| 項目 | 內容 |
|------|------|
| **寫作日期** | 主後 55-56 年 |
| **寫作地點** | 馬其頓 (Macedonia，可能腓立比) |
| **收信人** | 哥林多教會 |
| **章數** | 13 章 |
| **主題** | 使徒職分與新約服事 |
| **使徒行傳** | 徒 20:1 (離開以弗所往馬其頓) |
| **背景** | 回應假使徒的攻擊，為使徒職分辯護，鼓勵捐獻 |

**地理位置**：同哥林多前書，寫於馬其頓北部地區。

**核心經文**：
> 他對我說：「我的恩典夠你用的，因為我的能力是在人的軟弱上顯得完全。」— 林後 12:9

---

### 4. 加拉太書 (Galatians)

| 項目 | 內容 |
|------|------|
| **寫作日期** | 主後 49 年 (可能最早的書信) |
| **寫作地點** | 安提阿 (Antioch) 或哥林多 |
| **收信人** | 加拉太眾教會 (南加拉太：彼西底安提阿、以哥念、路司得、特庇) |
| **章數** | 6 章 |
| **主題** | 基督裏的自由，不靠律法稱義 |
| **使徒行傳** | 徒 13-14 (第一次旅程建立教會); 徒 15 (耶路撒冷會議) |
| **背景** | 猶太主義者滲入，要求外邦信徒受割禮守律法 |

**地理位置**：加拉太位於小亞細亞中部高原（今土耳其中部），包含彼西底安提阿、以哥念、路司得、特庇等城。

**核心經文**：
> 基督釋放了我們，叫我們得以自由。所以要站立得穩，不要再被奴僕的軛挾制。— 加 5:1

---

### 5. 以弗所書 (Ephesians)

| 項目 | 內容 |
|------|------|
| **寫作日期** | 主後 60-62 年 |
| **寫作地點** | 羅馬 (Rome) — 第一次囚禁期間 |
| **收信人** | 以弗所教會（可能為傳閱信） |
| **章數** | 6 章 |
| **主題** | 教會是基督的身體 |
| **使徒行傳** | 徒 19-20 (以弗所事工); 徒 28 (羅馬囚禁) |
| **背景** | 保羅在以弗所服事三年，與教會關係深厚，在獄中論述教會奧秘 |

**地理位置**：以弗所是亞細亞省首府，位於愛琴海東岸，著名的亞底米神廟所在地，當時人口約25萬。

**核心經文**：
> 你們得救是本乎恩，也因著信；這並不是出於自己，乃是神所賜的。— 弗 2:8

## 主題分類 (Thematic Categories)

### 救恩論書信 (Soteriological)
- **羅馬書** — 系統闡述因信稱義
- **加拉太書** — 為自由福音辯護

### 教會論書信 (Ecclesiological)
- **哥林多前後書** — 教會問題處理
- **以弗所書** — 教會是基督的身體
- **歌羅西書** — 基督是教會的頭

## 保羅神學核心主題 (Key Pauline Themes)

### 1. 因信稱義 (Justification by Faith)
> 人稱義不是因行律法，乃是因信耶穌基督。— 加 2:16

### 2. 在基督裏 (In Christ)
「在基督裏」(*en Christō*) 在保羅書信中出現超過 160 次，表達信徒與基督的聯合。

### 3. 基督的身體 (Body of Christ)
教會是基督的身體，各肢體互相配搭（林前12；弗4）。

### 4. 聖靈的工作 (Work of the Spirit)
靠聖靈行事、結聖靈果子、被聖靈充滿（加5；弗5）。

### 5. 新舊約對比 (Old/New Covenant)
律法是訓蒙的師傅，恩典在基督裏成全（加3-4；林後3）。

### 6. 末世盼望 (Eschatological Hope)
基督再來、身體復活、與主同在（帖前4；腓3）。
