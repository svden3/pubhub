## 主要主題

### 1. 長老資格 (1:5-9)

與提前3章對比，強調：
- 無可指責
- 一妻之夫
- 兒女信主
- 堅守真道
- 能勸化人、駁倒爭辯的人

### 2. 各群體的教導 (2:1-10)

| 群體 | 教導 |
|------|------|
| 老年人 | 節制、莊重 |
| 老年婦人 | 舉止敬虔、教導婦女 |
| 少年婦人 | 愛丈夫、愛兒女 |
| 少年人 | 謹守 |
| 僕人 | 凡事順服 |

### 3. 救恩的恩典 (2:11-14; 3:4-7)
- 救眾人的恩典顯明
- 重生的洗、聖靈的更新
- 因恩稱義、成為後嗣
