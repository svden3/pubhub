## 地理與歷史背景

### 寫作情境

```
帖撒羅尼迦書信時間線

    帖前 ─────────→ 數月後 ─────────→ 帖後
     │                                   │
    安慰已死者              糾正假教導
    盼望被提                不可懶惰
```

### 寫作原因

1. **假冒信件**：有人假託保羅的名（帖後2:2）
2. **錯誤教導**：主的日子已經到了
3. **懶惰問題**：有人不做工，專管閒事

## 核心經文

> 弟兄們，論到我們主耶穌基督降臨和我們到他那裏聚集，我勸你們：無論有靈、有言語、有冒我名的書信，說主的日子現在到了，不要輕易動心，也不要驚慌。
> — 帖撒羅尼迦後書 2:1-2

> 若有人不肯做工，就不可吃飯。
> — 帖撒羅尼迦後書 3:10

## 主要主題

### 1. 主再來的審判 (1:5-10)
- 神公義的審判
- 主耶穌顯現
- 報應不認識神的人

### 2. 離道反教與大罪人 (2:1-12)

| 要素 | 描述 |
|------|------|
| 離道反教 | 必先發生 |
| 大罪人/沉淪之子 | 抵擋主、自稱是神 |
| 那攔阻的 | 現在攔阻他 |
| 無法無天的人 | 主要消滅他 |

### 3. 殷勤做工 (3:6-15)
- 不可懶惰
- 效法使徒榜樣
- 親手做工吃飯
