## 經文 (Scripture)

### 中文 — 和合本修訂版 (RCUV)

> ^1^耶穌基督的啟示，就是神賜給他，要他把短期內必須發生的事指示他的眾僕人。他差遣使者向他的僕人約翰表明出來。^2^約翰就為神的道和耶穌基督的見證，把他所見的一切都見證出來。^3^**誦讀這書上預言的和那些聽見又遵守其中所記載的，都是有福的，因為時候近了。**

> ^8^**「我是阿拉法，我是俄梅戛，」主神說，「是昔在、今在、以後永在的全能者。」**

> ^12^我轉過身來要看是誰的聲音在跟我說話；轉過來，我看見七個金燈臺。^13^燈臺中間有一位好像人子，身穿長衣，直垂到腳，胸間束著金帶。^14^他的頭與髮皆白，如白羊毛，如雪；眼目如同火焰；^15^腳好像在爐中鍛煉發亮的銅；聲音如同眾水的聲音。^16^他右手拿著七顆星，從他口中出來一把兩刃的利劍；面貌如同正午烈日放光。

## 背景 (Context)

### 拔摩海島

約翰因見證耶穌被放逐到拔摩海島，這是羅馬帝國用來放逐政治犯的地方。

### 主日的異象

「在主日」(1:10) — 這是最早提到「主日」(星期日) 的文獻之一，紀念基督復活。

### 七教會的地理位置

七個教會位於亞細亞省（今土耳其西部），形成一個環形路線：
以弗所 → 士每拿 → 別迦摩 → 推雅推喇 → 撒狄 → 非拉鐵非 → 老底嘉

## 原文研讀 (Word Study)

| 希臘文 | 音譯 | 意義 | 註解 |
|--------|------|------|------|
| *apokalypsis* | apokalypsis | 啟示/揭露 | 揭開幔子 |
| *Alpha kai Omega* | Alpha kai Omega | 阿拉法和俄梅戛 | 希臘字母首末 |
| *pantokratōr* | pantokrator | 全能者 | 掌管萬有 |
| *rhomphaia* | rhomphaia | 兩刃利劍 | 審判的話語 |
| *kleis* | kleis | 鑰匙 | 權柄的象徵 |

### 「阿拉法與俄梅戛」

這是希臘字母表的第一個和最後一個字母：
- **Alpha (Α)** — 開始、起源
- **Omega (Ω)** — 結束、終極
- 基督是萬有的始終

## 神學要點 (Theological Points)

### 1. 啟示的來源

| 層次 | 來源 |
|------|------|
| 1 | 神 |
| 2 | 耶穌基督 |
| 3 | 天使 |
| 4 | 約翰 |
| 5 | 讀者/聽眾 |

### 2. 第一個福 (1:3)

> 「誦讀⋯⋯聽見⋯⋯遵守⋯⋯都是有福的」

這是啟示錄七福的第一個，強調：
- **誦讀** — 公開宣讀
- **聽見** — 留心聆聽
- **遵守** — 實際順服

### 3. 基督的神性宣告

| 宣告 | 意義 |
|------|------|
| 阿拉法與俄梅戛 | 永恆的始終 |
| 昔在今在以後永在 | 永恆的存在 |
| 全能者 | 無限的權能 |
| 首先的、末後的 | 掌管歷史 |

### 4. 榮耀基督的形像

| 描述 | 意義 | 舊約呼應 |
|------|------|---------|
| 身穿長衣 | 祭司的職分 | 出 28:4 |
| 頭與髮皆白 | 永恆的智慧 | 但 7:9 |
| 眼目如火焰 | 洞察一切 | 但 10:6 |
| 腳如銅 | 審判的穩固 | 但 10:6 |
| 聲音如眾水 | 威嚴的權柄 | 結 1:24 |
| 口出利劍 | 話語的能力 | 賽 49:2 |
| 面如烈日 | 神聖的榮光 | 太 17:2 |

### 5. 死亡與陰間的鑰匙

> 「拿著死亡和陰間的鑰匙」(1:18)

- 基督勝過死亡
- 掌管死後的命運
- 信徒無需懼怕

## 歷代注疏 (Historical Commentary)

### 教父時期

**愛任紐 (Irenaeus, 130-202)**:

> 「約翰在拔摩海島看見這異象，直到圖密善統治的末期。」這是我們知道啟示錄寫作日期的重要見證。

### 改革宗時期

**約翰·加爾文 (John Calvin, 1509-1564)**:

雖然加爾文沒有寫啟示錄註釋，他說：「這書滿了奧秘，我不敢輕易解釋。但有一點是清楚的：基督是得勝的君王。」

### 當代釋經

**約翰·麥克乾瑟 (John MacArthur)**:

> 「約翰看見的不是他認識的那個耶穌——那個柔和謙卑的老師。他看見的是榮耀的審判者。難怪他仆倒像死了一樣。」
> — *MacArthur Study Bible*

**乾伯·摩根 (G. Campbell Morgan)**:

> 「『不要懼怕』——這是復活主對他僕人說的第一句話。這與他在世時對門徒說的一樣。基督的能力帶來安慰，不是恐懼。」

## 生命應用 (Application)

### 默想問題

1. **看見基督**：如果你像約翰一樣看見榮耀的基督，你會有什麼反應？

2. **不要懼怕**：基督說「不要懼怕」——你生命中有什麼需要交給這位掌管萬有的主？

3. **死亡的鑰匙**：知道基督拿著「死亡和陰間的鑰匙」，如何影響你對死亡的態度？

4. **誦讀聽見遵守**：你如何實踐這三重回應？

### 禱告回應

> 榮耀的主耶穌基督，
> 你是首先的、末後的，
> 是那存活的，曾死過又活了，
> 直活到永永遠遠。
> 當我看見你的榮耀，我俯伏敬拜。
> 感謝你說「不要懼怕」，
> 因為你掌管一切，包括死亡和陰間。
> 幫助我誦讀這預言，聽見並遵守，
> 在等候你再來的日子，忠心到底。
> 阿們。

## 反思問題 (Reflection Questions)

1. 「耶穌基督的啟示」——這啟示的來源和目的是什麼？

2. 為什麼誦讀、聽見、遵守啟示錄的人是「有福的」？

3. 「阿拉法與俄梅戛」「昔在今在以後永在」這些稱號揭示基督什麼屬性？

4. 榮耀基督的七個描述（1:13-16）各有什麼意義？

5. 約翰仆倒在基督腳前——這告訴我們什麼關於敬拜的真諦？
