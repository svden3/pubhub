## 七教會書信的結構 (Structure) {-}

每封書信都有相似的結構：

| 元素 | 內容 |
|------|------|
| **稱呼** | 教會所在城市的天使 |
| **基督的自稱** | 呼應第一章的異象 |
| **「我知道」** | 對教會的評估 |
| **稱讚或責備** | 正面或負面的評價 |
| **命令** | 悔改或持守的吩咐 |
| **應許** | 給得勝者的獎賞 |
| **呼召** | 「聖靈向眾教會所說的話，凡有耳的，就應當聽」 |

## 以弗所 (Ephesus) — 2:1-7 {-}

### 背景

- 亞細亞省最大城市
- 亞底米神廟所在地
- 保羅曾在此事奉三年（徒 19）
- 提摩太、約翰都曾在此牧會

### 稱讚

| 稱讚 | 經文 |
|------|------|
| 勞碌工作 | 2:2 |
| 不能容忍惡人 | 2:2 |
| 試驗假使徒 | 2:2 |
| 為主名忍耐 | 2:3 |
| 恨惡尼哥拉一黨的行為 | 2:6 |

### 責備

> 「你把起初的愛心離棄了」(2:4)

- 不是失去，而是**離棄**（主動）
- 對基督的愛冷淡了
- 外在行為正確，內心卻遠離

### 命令

1. **回想** — 從哪裏墜落
2. **悔改** — 轉回
3. **行** — 起初所行的

### 得勝者的應許

> 「吃生命樹的果子」(2:7) — 伊甸園失去的，在新耶路撒冷恢復

---

## 士每拿 (Smyrna) — 2:8-11 {-}

### 背景

- 美麗的港口城市，稱為「亞細亞的冠冕」
- 對羅馬皇帝崇拜非常熱心
- 基督徒面臨嚴重逼迫
- 坡旅甲 (Polycarp) 在此殉道

### 稱讚

這是唯一沒有責備的教會之一（另一個是非拉鐵非）。

| 稱讚 | 經文 |
|------|------|
| 在患難中 | 2:9 |
| 在貧窮中（卻是富足的）| 2:9 |
| 忍受毀謗 | 2:9 |

### 「撒但的會堂」

> 「那自稱是猶太人卻不是的⋯⋯撒但的會堂」(2:9)

指那些逼迫基督徒的猶太人，他們失去了真正的屬靈身份。

### 命令

> 「不用怕⋯⋯要至死忠心」(2:10)

### 得勝者的應許

> 「必不受第二次死的害」(2:11) — 永恆的生命保障

---

## 別迦摩 (Pergamum) — 2:12-17 {-}

### 背景

- 亞細亞省的行政首府
- 宙斯祭壇——「撒但的座位」
- 阿斯克勒庇俄斯（蛇形醫神）崇拜中心
- 羅馬皇帝崇拜的中心

### 稱讚

| 稱讚 | 經文 |
|------|------|
| 住在撒但座位之處 | 2:13 |
| 持守主的名 | 2:13 |
| 沒有否認信仰 | 2:13 |
| 即使安提帕殉道 | 2:13 |

### 責備

| 責備 | 說明 |
|------|------|
| 有人持守巴蘭的教訓 | 引誘以色列人拜偶像、行淫 |
| 有人持守尼哥拉一黨的教訓 | 主張基督徒可參與異教習俗 |

### 命令

> 「悔改」(2:16)

### 得勝者的應許

1. **隱藏的嗎哪** — 天上的糧食
2. **白石和新名** — 新的身份和關係

## 神學要點 (Theological Points)

### 1. 愛與服事的平衡

以弗所教會教導我們：
- 正統信仰不能取代愛心
- 服事熱心不能取代與主的關係
- 外在正確不等於內心正確

### 2. 患難中的富足

士每拿教會提醒我們：
- 物質貧窮可以與靈性富足並存
- 忠心比安全更重要
- 生命的冠冕比今生的安逸更寶貴

### 3. 在黑暗中持守

別迦摩教會顯示：
- 環境惡劣不是妥協的理由
- 殉道者的見證激勵後人
- 教會內部的妥協比外部逼迫更危險

## 歷代注疏 (Historical Commentary)

### 殉道者的見證

**坡旅甲的殉道 (Polycarp, 69-155)**:

當羅馬官員要求他咒詛基督時，坡旅甲說：「我服事他八十六年，他從未虧待過我。我怎能褻瀆那拯救我的王？」

他被火燒死時禱告：「為我被算在殉道者中，我讚美你、感謝你。」

### 當代釋經

**約翰·麥克乾瑟 (John MacArthur)**:

> 「以弗所教會什麼都對，只有一件事錯——失去起初的愛。但這『一件事』幾乎就是一切。沒有愛的服事是空洞的宗教。」
> — *MacArthur Study Bible*

## 生命應用 (Application)

### 默想問題

1. **以弗所**：你的服事是否出於愛？還是已成為例行公事？

2. **士每拿**：你願意為信仰付出什麼代價？

3. **別迦摩**：你生活的環境中有什麼是「撒但的座位」？你如何持守信仰？

### 禱告回應

> 主耶穌，
> 你行走在七個金燈臺中間，
> 你察看每一個教會。
> 求你幫助我不離棄起初的愛心，
> 在患難中忠心，在黑暗中持守你的名。
> 使我成為得勝者，得著你應許的獎賞。
> 阿們。

## 反思問題 (Reflection Questions)

1. 以弗所教會有那麼多優點，為什麼「失去起初的愛」是如此嚴重？

2. 士每拿教會受逼迫卻得稱讚——這對今天追求成功的教會文化有什麼挑戰？

3. 「撒但的座位」在今天可能是什麼形式？

4. 巴蘭和尼哥拉一黨的教訓有什麼共同點？今天有類似的妥協嗎？

5. 「得勝者」在這三個教會各得什麼應許？這些應許有什麼共同主題？
